Qu'est-ce que la chimie générale ?

Introduction et vue d'ensemble

La chimie générale constitue le socle fondamental de toute la chimie. Elle étudie les principes universels qui régissent la matière : comment les atomes sont structurés, comment ils se lient entre eux, comment ils réagissent et dans quelles proportions. Sans ces bases, aucune autre branche de la chimie ne peut être abordée.

Pourquoi commencer ici ? La chimie générale vous donne les outils pour comprendre pourquoi le sel se dissout dans l'eau, pourquoi le fer rouille, comment fonctionnent les médicaments et pourquoi certaines réactions libèrent de l'énergie. C'est la grammaire de la chimie.

Ce domaine couvre cinq thèmes majeurs. La structure de l'atome révèle comment les électrons sont organisés et déterminent la réactivité. Le tableau périodique classe les 118 éléments selon leurs propriétés. Les liaisons chimiques expliquent comment les atomes s'assemblent en molécules. Les réactions chimiques décrivent les transformations de la matière. La stœchiométrie quantifie ces transformations.
