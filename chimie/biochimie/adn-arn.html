Introduction

Le langage de la vie

L'ADN (acide désoxyribonucléique) et l'ARN (acide ribonucléique) sont les molécules qui stockent et transmettent l'information génétique. L'ADN contient le « plan » de construction de tous les organismes vivants, tandis que l'ARN sert d'intermédiaire pour fabriquer les protéines.

La découverte de la double hélice de l'ADN par Watson et Crick en 1953 (grâce aux données de Rosalind Franklin) est l'une des plus grandes avancées de la biologie du XXe siècle.

Dogme central de la biologie moléculaire : ADN ⟶ (réplication) ADN ⟶ (transcription) ARN ⟶ (traduction) Protéine. L'information circule de l'ADN vers les protéines, avec l'ARN comme intermédiaire.

Le cours

Acides nucléiques et expression génétique

Structure des nucléotides

Les acides nucléiques sont des polymères de nucléotides. Chaque nucléotide est composé de trois parties :

  • Une base azotée : purines (adénine A, guanine G) ou pyrimidines (cytosine C, thymine T dans l'ADN, uracile U dans l'ARN).
  • Un sucre : désoxyribose (ADN) ou ribose (ARN). La différence est un -OH en position 2'.
  • Un groupement phosphate : relie les nucléotides entre eux par des liaisons phosphodiester (5' → 3').

Structure de l'ADN

L'ADN est formé de deux brins antiparallèles enroulés en double hélice droite (forme B). Les bases sont appariées par des liaisons hydrogène selon les règles de Chargaff :

  • A = T : 2 liaisons hydrogène.
  • G ≡ C : 3 liaisons hydrogène (plus stable).

L'hélice B a un diamètre de 2 nm, un pas de 3,4 nm (10 paires de bases par tour). Les brins sont complémentaires et antiparallèles (l'un va 5'→3', l'autre 3'→5').

Dénaturation de l'ADN : Le chauffage sépare les deux brins (fusion). La température de fusion Tm augmente avec le pourcentage de G-C (3 liaisons H vs. 2 pour A-T).

Structure de l'ARN

L'ARN est généralement simple brin mais peut former des structures secondaires (boucles, tiges). Il existe plusieurs types d'ARN :

  • ARNm (messager) : copie de l'ADN portant le message génétique vers les ribosomes.
  • ARNt (transfert) : adaptateur portant un acide aminé et un anticodon complémentaire du codon de l'ARNm.
  • ARNr (ribosomal) : composant structural et catalytique des ribosomes.
  • Petits ARN régulateurs : miARN, siARN (régulation de l'expression génétique).

La réplication de l'ADN

La réplication est la duplication de l'ADN avant la division cellulaire. Elle est semi-conservative (chaque nouvelle molécule contient un brin ancien et un brin néosynthétisé).

  • Hélicase : déroule la double hélice en séparant les brins.
  • Primase : synthétise une courte amorce d'ARN.
  • ADN polymérase III : synthétise le nouveau brin dans le sens 5'→3'. Brin continu (leading) et brin discontinu (fragments d'Okazaki).
  • ADN ligase : relie les fragments d'Okazaki.
  • Corrections : activité exonucléase 3'→5' de la polymérase (relecture). Taux d'erreur final : ~10⁻&sup9; par nucléotide.

La transcription

La transcription copie un gène (ADN) en ARNm, dans le noyau chez les eucaryotes :

  • Initiation : l'ARN polymérase se fixe sur le promoteur (séquence TATA box chez les eucaryotes) avec les facteurs de transcription.
  • Élongation : synthèse de l'ARNm dans le sens 5'→3', lecture du brin matrice 3'→5'.
  • Terminaison : signal de terminaison (séquences spécifiques ou facteur rho).
  • Maturation (eucaryotes) : ajout d'une coiffe 5' (m7G), polyadénylation 3' (queue poly-A), épissage (excision des introns, jonction des exons).

La traduction

La traduction convertit le message de l'ARNm en protéine, sur les ribosomes :

  • Code génétique : chaque triplet de nucléotides (codon) code un acide aminé. 64 codons pour 20 acides aminés + 3 codons stop (UAA, UAG, UGA). Le code est dégénéré et (quasi) universel.
  • Initiation : le ribosome se fixe sur l'ARNm au codon AUG (méthionine).
  • Élongation : l'ARNt aminoacylé entre dans le site A, liaison peptidique (site P), translocation.
  • Terminaison : un codon stop est reconnu par un facteur de libération, la protéine est relâchée.

Régulation de l'expression génétique

L'expression des gènes est finement régulée à plusieurs niveaux : remodelage de la chromatine (épigénétique), transcription (facteurs de transcription, enhancers, silencers), maturation de l'ARN (épissage alternatif), traduction (miARN) et modifications post-traductionnelles.

Exercices

Mettez en pratique

Exercice 1 — Complémentarité

Le brin matrice d'un ADN est : 3'-TACGGATCCA-5'. Donner la séquence de l'ARNm correspondant.

Voir la solution

L'ARNm est complémentaire du brin matrice, lu dans le sens 3'→5', avec U à la place de T. ARNm : 5'-AUGCCUAGGU-3'.

Exercice 2 — Chargaff

Un échantillon d'ADN contient 22 % d'adénine. Déterminer les pourcentages des trois autres bases.

Voir la solution

Selon les règles de Chargaff : %A = %T = 22 %. Donc %G + %C = 100 - 44 = 56 %. Comme %G = %C : %T = 22 %, %G = 28 %, %C = 28 %.

Exercice 3 — Code génétique

Pourquoi dit-on que le code génétique est « dégénéré » mais pas « ambigu » ?

Voir la solution

Dégénéré : plusieurs codons différents peuvent coder le même acide aminé (ex : UUU et UUC codent tous deux la phénylalanine). Il y a 61 codons sens pour 20 acides aminés. Non ambigu : un codon donné ne code qu'un seul acide aminé. La correspondance est univoque dans le sens codon → acide aminé.

Exercice 4 — Mutations

Expliquer la différence entre une mutation faux-sens, non-sens et silencieuse.

Voir la solution

Faux-sens : le codon modifié code un acide aminé différent (ex : GAG → GUG change Glu en Val dans la drépanocytose). Non-sens : le codon modifié devient un codon stop (ex : UAC → UAA), tronquant la protéine. Silencieuse : le codon modifié code le même acide aminé (grâce à la dégénérescence, souvent en 3e position du codon).

Exercice 5 — Traduction

L'ARNm suivant est traduit : 5'-AUGUUUGCUUAA-3'. Donner la séquence peptidique.

Voir la solution

Découpage en codons : AUG | UUU | GCU | UAA. AUG = Met, UUU = Phe, GCU = Ala, UAA = Stop. Peptide : Met-Phe-Ala (tripeptide).
