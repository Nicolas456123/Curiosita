Introduction

Les transformations de la matière

Une réaction chimique est une transformation au cours de laquelle des substances (les réactifs) se convertissent en de nouvelles substances (les produits). Les liaisons entre atomes se rompent et se reforment, mais les atomes eux-mêmes sont conservés : c'est la loi de Lavoisier. Comprendre les réactions chimiques est essentiel pour maîtriser la chimie, de la synthèse industrielle à la biochimie.

Loi de conservation de la masse (Lavoisier, 1789) : « Rien ne se perd, rien ne se crée, tout se transforme. » Dans une réaction chimique, la masse totale des réactifs est égale à la masse totale des produits.

Le cours

Types de réactions, équilibrage et cinétique

Les grands types de réactions

On distingue plusieurs catégories fondamentales de réactions chimiques :

  • Synthèse (combinaison) : A + B → AB. Deux réactifs forment un seul produit. Exemple : 2H₂ + O₂ → 2H₂O.
  • Décomposition : AB → A + B. Un composé se scinde en substances plus simples. Exemple : 2H₂O₂ → 2H₂O + O₂.
  • Substitution (déplacement simple) : A + BC → AC + B. Un élément remplace un autre dans un composé. Exemple : Zn + CuSO₄ → ZnSO₄ + Cu.
  • Double substitution (métathèse) : AB + CD → AD + CB. Deux composés échangent leurs partenaires. Exemple : NaCl + AgNO₃ → AgCl↓ + NaNO₃.
  • Combustion : Réaction rapide avec le dioxygène, exothermique. Exemple : CH₄ + 2O₂ → CO₂ + 2H₂O.

Équilibrage des équations chimiques

Équilibrer une équation chimique consiste à ajuster les coefficients stœchiométriques pour respecter la conservation des atomes. Méthode :

  • Écrire l'équation non équilibrée avec les bonnes formules chimiques.
  • Compter les atomes de chaque élément de chaque côté.
  • Ajuster les coefficients en commençant par l'élément le plus complexe.
  • Vérifier que chaque élément est équilibré et que les coefficients sont les plus petits entiers possibles.

Exemple : Fe₂O₃ + CO → Fe + CO₂. Bilan : Fe₂O₃ + 3CO → 2Fe + 3CO₂. On vérifie : 2 Fe, 3 C, 6 O de chaque côté.

Réactions d'oxydoréduction

Les réactions redox impliquent un transfert d'électrons entre espèces chimiques :

  • Oxydation : perte d'électrons (l'oxydant gagne des électrons). Le nombre d'oxydation augmente.
  • Réduction : gain d'électrons (le réducteur perd des électrons). Le nombre d'oxydation diminue.

Exemple : Zn + Cu²⁺ → Zn²⁺ + Cu. Le zinc s'oxyde (perd 2 e−), le cuivre se réduit (gagne 2 e−). Pour équilibrer une redox, on utilise la méthode des demi-réactions.

Équilibre chimique

De nombreuses réactions sont réversibles et atteignent un état d'équilibre où les vitesses des réactions directe et inverse sont égales. La constante d'équilibre Kéq quantifie la position de l'équilibre :

Pour aA + bB ⇌ cC + dD : Kéq = [C]c[D]d / [A]a[B]b

  • K >> 1 : équilibre déplacé vers les produits (réaction quasi complète).
  • K << 1 : équilibre déplacé vers les réactifs (réaction très limitée).

Principe de Le Chatelier : Si l'on perturbe un système à l'équilibre (concentration, pression, température), le système évolue de manière à s'opposer à la perturbation.

Cinétique chimique

La cinétique étudie la vitesse des réactions chimiques. La vitesse dépend de plusieurs facteurs :

  • Concentration : plus les réactifs sont concentrés, plus les collisions sont fréquentes.
  • Température : une augmentation de 10 °C double généralement la vitesse (loi d'Arrhenius).
  • Surface de contact : diviser un solide en poudre augmente la surface réactive.
  • Nature des réactifs : les ions en solution réagissent plus vite que les molécules covalentes.

L'énergie d'activation (Ea) est l'énergie minimale nécessaire pour qu'une collision entre réactifs soit efficace et déclenche la réaction.

Catalyse

Un catalyseur accélère une réaction chimique sans être consommé. Il agit en abaissant l'énergie d'activation, offrant un chemin réactionnel alternatif plus favorable :

  • Catalyse homogène : le catalyseur est dans la même phase que les réactifs. Ex : H⁺ catalyse l'estérification en solution.
  • Catalyse hétérogène : le catalyseur est dans une phase différente (solide + gaz). Ex : pot catalytique (Pt, Pd, Rh) dans les voitures.
  • Catalyse enzymatique : les enzymes sont les catalyseurs du vivant, extrêmement spécifiques et efficaces.

Thermochimie : énergie des réactions

Toute réaction s'accompagne d'un échange d'énergie, mesuré par l'enthalpie de réaction (ΔH) :

  • Exothermique (ΔH < 0) : libère de la chaleur. Ex : combustion du méthane (ΔH = -890 kJ/mol).
  • Endothermique (ΔH > 0) : absorbe de la chaleur. Ex : décomposition thermique du CaCO₃ (ΔH = +178 kJ/mol).

La loi de Hess permet de calculer ΔH d'une réaction en combinant les enthalpies de réactions intermédiaires, car l'enthalpie est une fonction d'état.

Exercices

Mettez en pratique

Exercice 1 — Équilibrage

Équilibrer l'équation : C₃H₈ + O₂ → CO₂ + H₂O

Voir la solution

C₃H₈ + 5O₂ → 3CO₂ + 4H₂O. Vérification : 3 C, 8 H, 10 O de chaque côté. La combustion complète du propane produit du CO₂ et de l'eau.

Exercice 2 — Identification du type

Classer chaque réaction : (a) 2KClO₃ → 2KCl + 3O₂ ; (b) Fe + 2HCl → FeCl₂ + H₂ ; (c) NaOH + HCl → NaCl + H₂O

Voir la solution

(a) Décomposition : un seul réactif donne deux produits. (b) Substitution simple : le fer remplace l'hydrogène dans HCl. (c) Double substitution (neutralisation acido-basique) : NaOH et HCl échangent leurs ions pour former NaCl et H₂O.

Exercice 3 — Oxydoréduction

Identifier l'oxydant et le réducteur dans : 2Al + 3Cl₂ → 2AlCl₃.

Voir la solution

Al passe de 0 à +3 : il perd 3 e− (oxydation), c'est le réducteur. Cl passe de 0 à -1 : il gagne 1 e− (réduction), c'est l'oxydant. Bilan : 2Al → 2Al³⁺ + 6e− et 3Cl₂ + 6e− → 6Cl−.

Exercice 4 — Équilibre chimique

Pour la réaction N₂ + 3H₂ ⇌ 2NH₃ (ΔH = -92 kJ/mol), comment déplacer l'équilibre vers la production de NH₃ ?

Voir la solution

D'après Le Chatelier : (1) Augmenter la pression (4 mol de gaz → 2 mol : l'équilibre favorise le côté avec moins de moles gazeuses). (2) Diminuer la température (réaction exothermique, le refroidissement favorise le sens direct). (3) Augmenter [N₂] ou [H₂]. (4) Retirer NH₃ du milieu. C'est le procédé Haber-Bosch (200 atm, 450 °C, catalyseur Fe).

Exercice 5 — Thermochimie

Calculer ΔH de la réaction C + O₂ → CO₂ sachant que : C + ½O₂ → CO (ΔH₁ = -110,5 kJ) et CO + ½O₂ → CO₂ (ΔH₂ = -283 kJ).

Voir la solution

Par la loi de Hess : ΔH = ΔH₁ + ΔH₂ = -110,5 + (-283) = -393,5 kJ/mol. On additionne les deux étapes car le CO est un intermédiaire qui s'annule. C'est l'enthalpie standard de combustion du carbone.
