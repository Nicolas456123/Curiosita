Introduction

La brique élémentaire de la matière

L'atome est l'unité fondamentale de la matière. Son nom vient du grec atomos (« indivisible »), bien que l'on sache aujourd'hui qu'il est composé de particules subatomiques. Comprendre la structure de l'atome est le premier pas pour maîtriser la chimie.

Histoire : De Dalton (1803, atomes indivisibles) à Thomson (1897, électron), Rutherford (1911, noyau), Bohr (1913, orbites quantifiées) et Schrödinger (1926, modèle quantique) — notre vision de l'atome n'a cessé d'évoluer.

02

Le cours

Structure atomique et configuration électronique

Les particules subatomiques

L'atome est composé de trois types de particules :

  • Protons (charge +1, masse 1,673 × 10−²&sup7; kg) : dans le noyau. Le nombre de protons Z (numéro atomique) définit l'élément chimique.
  • Neutrons (charge 0, masse ~1,675 × 10−²&sup7; kg) : dans le noyau. Le nombre de neutrons N distingue les isotopes d'un même élément.
  • Électrons (charge -1, masse 9,109 × 10−³¹ kg) : autour du noyau. Ils déterminent le comportement chimique de l'atome.

Nombre de masse : A = Z + N. Exemple : le carbone-12 a Z = 6 protons et N = 6 neutrons, donc A = 12.

Le modèle quantique

Selon la mécanique quantique, les électrons n'ont pas de trajectoire définie mais occupent des orbitales — des régions de l'espace où la probabilité de trouver l'électron est maximale. Chaque orbitale est définie par quatre nombres quantiques :

  • n (principal) : niveau d'énergie (1, 2, 3...). Détermine la taille de l'orbitale.
  • l (secondaire) : forme de l'orbitale (0 = s, 1 = p, 2 = d, 3 = f).
  • mₗ (magnétique) : orientation dans l'espace (-l à +l).
  • mₛ (spin) : rotation de l'électron (+1/2 ou -1/2).

Configuration électronique

Le remplissage des orbitales suit trois règles fondamentales :

  • Principe d'Aufbau : les électrons remplissent d'abord les orbitales de plus basse énergie (1s, 2s, 2p, 3s, 3p, 4s, 3d, 4p...).
  • Principe d'exclusion de Pauli : chaque orbitale contient au maximum 2 électrons de spins opposés.
  • Règle de Hund : dans des orbitales de même énergie, les électrons se répartissent avec un maximum de spins parallèles.

Exemple : L'oxygène (Z = 8) : 1s² 2s² 2p&sup4;. Il a 6 électrons de valence (couche n = 2), d'où sa tendance à gagner 2 électrons pour atteindre la configuration du néon.

La règle de l'octet

Les atomes tendent à acquérir 8 électrons sur leur couche de valence (2 pour l'hydrogène et l'hélium) pour atteindre la stabilité des gaz nobles. Cette tendance explique la formation des liaisons chimiques :

  • Le sodium (Na, Z = 11 : [Ne] 3s¹) perd 1 électron → Na⁺
  • Le chlore (Cl, Z = 17 : [Ne] 3s² 3p&sup5;) gagne 1 électron → Cl−
  • Ensemble, ils forment NaCl par liaison ionique.

Isotopes et radioactivité

Les isotopes sont des atomes d'un même élément (même Z) mais avec un nombre de neutrons différent. Exemples :

  • Carbone-12 (stable, 6p + 6n) et Carbone-14 (radioactif, 6p + 8n)
  • Hydrogène (1p), Deutérium (1p + 1n), Tritium (1p + 2n)

Les isotopes instables se désintègrent par radioactivité (α, β, γ). La période de demi-vie T½ indique le temps nécessaire pour que la moitié des atomes se désintègre.

Exercices

Mettez en pratique

Exercice 1 — Configuration électronique

Écrire la configuration électronique complète du fer (Z = 26).

Voir la solution

Fe (Z = 26) : 1s² 2s² 2p&sup6; 3s² 3p&sup6; 4s² 3d&sup6;. Notation abrégée : [Ar] 4s² 3d&sup6;. Le fer a 8 électrons de valence (2 dans 4s et 6 dans 3d). Attention : le remplissage suit l'ordre 4s avant 3d, mais lors de l'ionisation, les électrons 4s partent en premier.

Exercice 2 — Isotopes

L'uranium possède deux isotopes principaux : U-235 et U-238. Combien de neutrons contient chacun ? Lequel est utilisé comme combustible nucléaire ?

Voir la solution

L'uranium a Z = 92 protons. U-235 : N = 235 - 92 = 143 neutrons. U-238 : N = 238 - 92 = 146 neutrons. L'U-235 est fissile et utilisé comme combustible nucléaire (0,7 % de l'uranium naturel). L'U-238 est le plus abondant (99,3 %) mais non fissile.

Exercice 3 — Règle de l'octet

Expliquer pourquoi le magnésium (Z = 12) forme l'ion Mg²⁺ et non Mg⁺.

Voir la solution

Mg : [Ne] 3s². En perdant ses 2 électrons de valence (3s²), le magnésium atteint la configuration du néon [He] 2s² 2p&sup6; — couche de valence complète (8 électrons). S'il ne perdait qu'un seul électron (Mg⁺, config [Ne] 3s¹), sa couche externe ne serait pas complète. La stabilité maximale est atteinte avec Mg²⁺.
