Introduction

Au-delà de la droite

De nombreuses relations dans la nature ne sont pas linéaires. La croissance d'une population suit une courbe logistique. La décroissance radioactive est exponentielle. Le rendement agricole en fonction de l'engrais suit une courbe en forme de plateau. La régression non linéaire offre des outils pour modéliser ces relations courbes.

Il faut distinguer deux situations : les modèles linéarisables (polynomiaux, log-linéaires) qui se ramènent à une régression linéaire par transformation, et les modèles intrinsèquement non linéaires (logistique, Michaelis-Menten) qui nécessitent des algorithmes itératifs d'optimisation.

Principe clé : un modèle est linéaire en ses paramètres si Y est une combinaison linéaire des βᵢ (même si les variables sont transformées). Y = β₀ + β₁X² est linéaire en paramètres. Y = β₀ × e^(β₁X) ne l'est pas.

Le cours

Modèles et méthodes

Régression polynomiale

La régression polynomiale ajoute des puissances de X comme prédicteurs : Y = β₀ + β₁X + β₂X² + ... + βₖXᵏ + ε. C'est un modèle linéaire en paramètres, donc estimable par MCO classiques.

  • Degré 2 (quadratique) : modélise une courbe en U ou U inversé (ex : relation performance-stress)
  • Degré 3 (cubique) : modélise des courbes avec un point d'inflexion
  • Attention au surajustement : un polynôme de degré (n−1) passe par tous les n points mais n'a aucun pouvoir prédictif

Transformations linéarisantes

Certaines relations non linéaires deviennent linéaires après transformation :

  • Modèle exponentiel : Y = ae^(bX) → ln(Y) = ln(a) + bX (régression linéaire sur ln Y)
  • Modèle puissance : Y = aX^b → ln(Y) = ln(a) + b ln(X) (régression log-log)
  • Modèle inverse : Y = a + b/X → régression linéaire sur Z = 1/X

Attention : la transformation change la structure des erreurs. Minimiser les moindres carrés sur ln(Y) n'est pas identique à les minimiser sur Y.

Régression logistique

La régression logistique modélise une variable réponse binaire (0/1). Au lieu de prédire Y directement, on modélise la probabilité P(Y=1) via la fonction logistique :

P(Y=1|X) = 1 / (1 + e^(−(β₀ + β₁X)))

Le logit (log-odds) ln(P/(1−P)) = β₀ + β₁X est linéaire en les paramètres. Les coefficients sont estimés par maximum de vraisemblance, non par MCO. β₁ s'interprète comme le changement du log-odds pour une unité de X : e^β₁ est l'odds ratio.

Applications : diagnostic médical (malade/sain), scoring bancaire (défaut/remboursement), marketing (achat/non-achat). La courbe en S (sigmoïde) borne les probabilités entre 0 et 1, contrairement à la régression linéaire qui peut prédire des probabilités négatives ou supérieures à 1.

Modèles de croissance

  • Croissance exponentielle : Y(t) = Y₀ × e^(rt). Croissance illimitée à taux constant r
  • Croissance logistique : Y(t) = K / (1 + ((K−Y₀)/Y₀) × e^(−rt)). Croissance limitée par une capacité K (plateau)
  • Modèle de Gompertz : Y(t) = K × e^(−a × e^(−bt)). Croissance asymétrique avec ralentissement progressif

Estimation non linéaire

Pour les modèles intrinsèquement non linéaires, l'estimation des paramètres utilise des algorithmes itératifs :

  • Gauss-Newton : linéarise le modèle à chaque itération par développement de Taylor
  • Levenberg-Marquardt : compromis entre Gauss-Newton et descente de gradient, plus robuste
  • Convergence : dépend des valeurs initiales. De mauvais points de départ peuvent mener à un minimum local

Critères de choix de modèle

Pour comparer différents modèles non linéaires :

  • AIC/BIC : comparent les modèles en pénalisant la complexité
  • R² généralisé : R² = 1 − SCR/SCT, mais son interprétation est moins claire qu'en régression linéaire
  • Analyse des résidus : les résidus doivent être aléatoires et sans patron
  • Validation croisée : évalue la capacité prédictive sur des données non vues
  • Principe de parcimonie : préférer le modèle le plus simple qui ajuste bien les données

Exercice 1 — Régression polynomiale

Des données montrent une relation en U inversé entre le stress (X) et la performance (Y). Quel modèle proposez-vous ? Que signifie un β₂ négatif ?

Voir la solution

On propose un modèle quadratique : Y = β₀ + β₁X + β₂X² + ε.

Un β₂ négatif signifie que la parabole est ouverte vers le bas (U inversé). La performance augmente d'abord avec le stress puis diminue. Le sommet de la parabole est à X* = −β₁/(2β₂), qui donne le niveau de stress optimal.

Exercice 2 — Linéarisation

Des données de décroissance radioactive suivent Y = 100 × e^(−0,05t). Transformez ce modèle pour l'estimer par régression linéaire.

Voir la solution

ln(Y) = ln(100) + (−0,05) × t = 4,605 − 0,05t.

On pose Y' = ln(Y). La régression linéaire Y' = a + bt donne a ≈ 4,605 (donc Y₀ = e^a ≈ 100) et b ≈ −0,05 (constante de décroissance).

Exercice 3 — Régression logistique

Une régression logistique du risque de diabète sur l'IMC donne : logit(P) = −8,5 + 0,25 × IMC. Calculez la probabilité de diabète pour un IMC de 30 et l'odds ratio.

Voir la solution

logit = −8,5 + 0,25 × 30 = −8,5 + 7,5 = −1,0.

P = 1/(1+e^1) = 1/(1+2,718) = 1/3,718 = 0,269 (26,9 %).

Odds ratio = e^0,25 = 1,284. Chaque point d'IMC supplémentaire multiplie les odds de diabète par 1,28 (augmentation de 28 %).

Exercice 4 — Choix de modèle

Pour modéliser la population mondiale au cours du temps, trois modèles sont ajustés : exponentiel (AIC = 350), logistique (AIC = 310), polynomial de degré 4 (AIC = 305). Quel modèle choisir et pourquoi ?

Voir la solution

L'AIC favorise le polynôme (305), mais le modèle logistique (310) est préférable :

1. La différence d'AIC est faible (5 unités). 2. Le modèle logistique a un sens biologique (capacité de charge). 3. Le polynôme de degré 4 peut osciller dangereusement en extrapolation. 4. Le principe de parcimonie favorise les modèles interprétables.

Exercice 5 — Interprétation d'une sigmoïde

Un modèle de croissance logistique donne K = 10 000 (capacité), r = 0,15 (taux de croissance), Y₀ = 500 (population initiale). À quel moment t la population atteindra-t-elle 5 000 (la moitié de K) ?

Voir la solution

Y(t) = K/(1 + ((K−Y₀)/Y₀) × e^(−rt)) = 10000/(1 + 19 × e^(−0,15t)).

Pour Y = 5000 : 5000 = 10000/(1 + 19e^(−0,15t)).

1 + 19e^(−0,15t) = 2 → 19e^(−0,15t) = 1 → e^(−0,15t) = 1/19.

−0,15t = ln(1/19) = −2,944 → t = 2,944/0,15 = 19,6 unités de temps.

Le point d'inflexion (croissance maximale) se situe toujours à K/2 pour le modèle logistique.
