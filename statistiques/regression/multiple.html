Introduction

Plusieurs variables pour un seul modèle

En pratique, un phénomène est rarement expliqué par une seule variable. Le salaire dépend de l'expérience, du niveau d'études, du secteur d'activité. La régression multiple généralise la régression simple en incluant plusieurs variables explicatives : Y = β₀ + β₁X₁ + β₂X₂ + ... + βₚXₚ + ε.

Cette extension soulève des questions nouvelles : comment interpréter chaque coefficient quand les variables sont corrélées entre elles ? Comment choisir les variables à inclure dans le modèle ? Comment détecter et gérer la multicolinéarité ?

Principe clé : en régression multiple, chaque coefficient βⱼ mesure l'effet de Xⱼ sur Y « toutes choses égales par ailleurs », c'est-à-dire en maintenant les autres variables constantes. C'est l'effet partiel ou ceteris paribus.

02

Le cours

Modèle, estimation et sélection

Formulation matricielle

Le modèle s'écrit Y = Xβ + ε, où Y est le vecteur (n×1) des réponses, X la matrice (n×(p+1)) de design, β le vecteur ((p+1)×1) des coefficients et ε le vecteur (n×1) des erreurs. L'estimateur MCO est :

β̂ = (X'X)⁻¹X'Y

Cet estimateur existe si et seulement si X'X est inversible, ce qui requiert que les colonnes de X soient linéairement indépendantes (pas de multicolinéarité parfaite) et que n > p + 1.

R² et R² ajusté

Le R² augmente mécaniquement chaque fois qu'on ajoute une variable, même non pertinente. Le R² ajusté pénalise pour le nombre de variables :

R²ₐⱼ = 1 − (1 − R²) × (n − 1) / (n − p − 1)

Le R² ajusté peut diminuer si une variable ajoutée n'apporte pas suffisamment d'information. C'est un meilleur critère de comparaison entre modèles de tailles différentes.

Multicolinéarité

La multicolinéarité survient quand les variables explicatives sont fortement corrélées entre elles. Conséquences :

  • Les coefficients estimés deviennent instables (grande variance)
  • Les signes des coefficients peuvent être contre-intuitifs
  • Les tests individuels (t-tests) perdent de la puissance, alors que le test global F peut rester significatif

Le facteur d'inflation de la variance (VIF) détecte la multicolinéarité : VIFⱼ = 1 / (1 − R²ⱼ), où R²ⱼ est le R² de la régression de Xⱼ sur les autres variables. Un VIF > 5 (ou > 10) est problématique.

Solutions : retirer une des variables corrélées, combiner les variables (ACP), utiliser une régression pénalisée (ridge, lasso) qui gère naturellement la multicolinéarité en ajoutant une contrainte sur les coefficients.

Sélection de variables

Choisir les variables à inclure est crucial. Trop de variables → surajustement. Trop peu → biais d'omission.

  • Forward (progressive) : partir du modèle vide, ajouter la variable la plus significative, itérer
  • Backward (rétrograde) : partir du modèle complet, retirer la variable la moins significative, itérer
  • Stepwise : combinaison de forward et backward
  • Critères d'information : AIC = n ln(SCR/n) + 2p, BIC = n ln(SCR/n) + p ln(n). Le BIC pénalise davantage les modèles complexes
  • Validation croisée : évaluer la performance prédictive sur des données non utilisées pour l'ajustement

Variables qualitatives (dummy)

Les variables catégorielles sont intégrées via des variables indicatrices (dummy variables). Une variable à k modalités nécessite (k−1) indicatrices, la modalité omise servant de référence.

Exemple : si la variable « région » a 3 modalités (Nord, Sud, Ouest), on crée D_Sud = 1 si Sud, D_Ouest = 1 si Ouest. Le coefficient de D_Sud mesure la différence de Y entre Sud et Nord (référence).

Interactions

Un terme d'interaction X₁ × X₂ permet de modéliser le fait que l'effet de X₁ dépend du niveau de X₂. Le modèle Y = β₀ + β₁X₁ + β₂X₂ + β₃X₁X₂ + ε signifie que l'effet de X₁ est β₁ + β₃X₂ : il varie selon X₂.

Diagnostic et validation

Comme en régression simple, on vérifie les hypothèses par l'analyse des résidus. En plus :

  • Test F global : teste H₀ : β₁ = β₂ = ... = βₚ = 0 (aucune variable n'est utile)
  • Tests t partiels : testent la significativité de chaque variable conditionnellement aux autres
  • Test F partiel : compare un modèle réduit à un modèle complet pour tester la significativité d'un groupe de variables
  • Mallows' Cp : évalue le biais et la variance du modèle, Cp ≈ p+1 indique un bon ajustement

Exercices

Exercice 1 — Interprétation d'une sortie de régression

Un modèle de régression donne : Note = 45,2 + 2,1 × Heures_étude + 3,8 × Assiduité − 0,5 × Stress (R² = 0,68, R²ₐⱼ = 0,65). Interprétez chaque coefficient et le R² ajusté.

Voir la solution

Heures_étude (2,1) : une heure d'étude supplémentaire est associée à +2,1 points, à assiduité et stress constants.

Assiduité (3,8) : un point d'assiduité supplémentaire est associé à +3,8 points de note.

Stress (−0,5) : un point de stress supplémentaire est associé à −0,5 point de note.

R²ₐⱼ = 0,65 : le modèle explique 65 % de la variabilité des notes après pénalisation pour le nombre de variables.

Exercice 2 — Multicolinéarité

Dans un modèle avec les variables Taille (cm) et Taille (pouces), le VIF des deux variables est de 500. Expliquez pourquoi et proposez une solution.

Voir la solution

Taille (pouces) = Taille (cm) / 2,54. Les deux variables sont liées par une relation linéaire quasi parfaite (R² ≈ 1), d'où VIF = 1/(1−1) → très grand.

Solution : retirer l'une des deux variables. Elles contiennent exactement la même information. Garder les deux rend les coefficients individuels ininterprétables et instables.

Exercice 3 — Sélection de modèle

Trois modèles sont comparés. M1 : R²ₐⱼ = 0,72, AIC = 245. M2 : R²ₐⱼ = 0,74, AIC = 240. M3 : R²ₐⱼ = 0,74, AIC = 242. Quel modèle choisir ?

Voir la solution

M2 est le meilleur choix : il a le R² ajusté le plus élevé (ex aequo avec M3) et l'AIC le plus bas (240 < 242 < 245). Un AIC plus bas indique un meilleur compromis entre ajustement et parcimonie.

M3 a le même R² ajusté que M2 mais un AIC plus élevé, ce qui signifie qu'il est légèrement moins efficient (probablement une variable de plus pour le même pouvoir explicatif).

Exercice 4 — Variables indicatrices

On modélise le loyer en fonction de la surface et du quartier (Centre, Nord, Sud, Est). Écrivez le modèle avec les variables indicatrices et interprétez le coefficient de D_Centre si le quartier de référence est Sud.

Voir la solution

Loyer = β₀ + β₁ × Surface + β₂ × D_Centre + β₃ × D_Nord + β₄ × D_Est + ε.

D_Centre = 1 si Centre, 0 sinon. Idem pour D_Nord et D_Est. Le Sud est la référence (toutes les indicatrices = 0).

Si β₂ = 150, cela signifie que, à surface égale, un logement au Centre coûte en moyenne 150 € de plus par mois qu'un logement au Sud.

Exercice 5 — Test F partiel

Modèle complet (4 variables) : SCR = 200, n = 50. Modèle réduit (2 variables) : SCR = 280. Testez si les 2 variables supplémentaires sont conjointement significatives. (F_{0,05;2,45} ≈ 3,20)

Voir la solution

F = [(SCR_réduit − SCR_complet) / q] / [SCR_complet / (n − p − 1)]

= [(280 − 200) / 2] / [200 / (50 − 4 − 1)]

= [80/2] / [200/45] = 40 / 4,44 = 9,0.

F = 9,0 > 3,20 → on rejette H₀. Les deux variables supplémentaires apportent une contribution significative au modèle.
