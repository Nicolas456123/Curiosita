01

Introduction

Planifier pour détecter

L'analyse de puissance est une étape cruciale de la planification d'une étude. Avant de collecter des données, le chercheur doit s'assurer que son design expérimental aura suffisamment de puissance pour détecter l'effet recherché. Une étude sous-puissante gaspille des ressources et risque de produire un faux négatif.

L'analyse de puissance relie quatre quantités : la taille de l'effet, la taille de l'échantillon, le seuil de significativité α et la puissance (1 − β). Connaissant trois de ces quantités, on peut calculer la quatrième.

Principe clé : l'analyse de puissance a priori (avant l'étude) détermine la taille d'échantillon nécessaire. L'analyse de puissance a posteriori (après l'étude) est controversée car elle n'apporte pas d'information utile au-delà de la p-value elle-même.

02

Le cours

Calcul et application

Les quatre paramètres liés

L'analyse de puissance repose sur la relation entre quatre quantités interdépendantes :

  • Taille d'effet (δ) : l'ampleur du phénomène à détecter (d de Cohen, r, f²...)
  • Taille d'échantillon (n) : le nombre d'observations par groupe
  • Seuil de significativité (α) : le risque de faux positif accepté (souvent 0,05)
  • Puissance (1 − β) : la probabilité de détecter l'effet (souvent 0,80 ou 0,90)

En fixant trois de ces paramètres, on détermine le quatrième. Le cas le plus courant est de fixer α, la puissance souhaitée et la taille d'effet attendue pour calculer n.

Calcul de puissance pour un test t

Pour un test t à un échantillon (H₀ : μ = μ₀, H₁ : μ = μ₁), la puissance se calcule ainsi :

Sous H₁, la statistique T suit une loi de Student non centrée de paramètre de non-centralité λ = δ√n, où δ = |μ₁ − μ₀|/σ est le d de Cohen. La puissance est P(|T| > t_{α/2, n−1} | λ).

En approximation normale, la taille d'échantillon nécessaire est :

n ≈ ((z_{α/2} + z_β) / δ)²

Pour un test t à deux échantillons : n par groupe ≈ 2 × ((z_{α/2} + z_β) / δ)².

Exemples de calcul de taille d'échantillon

Pour un test t bilatéral à α = 0,05 et puissance = 0,80 (z_{0,025} = 1,96, z_{0,20} = 0,84) :

  • Effet grand (d = 0,8) : n = ((1,96 + 0,84) / 0,8)² = (2,80/0,8)² = 12,25 → n = 13 par groupe
  • Effet moyen (d = 0,5) : n = ((1,96 + 0,84) / 0,5)² = (2,80/0,5)² = 31,36 → n = 32 par groupe
  • Effet petit (d = 0,2) : n = ((1,96 + 0,84) / 0,2)² = (2,80/0,2)² = 196 → n = 196 par groupe

Observation importante : détecter un petit effet nécessite environ 15 fois plus de participants que détecter un grand effet. La taille d'échantillon est inversement proportionnelle au carré de la taille d'effet : n ∝ 1/δ².

Puissance pour d'autres tests

Chaque test a sa propre formule de puissance :

  • Test du Chi-deux : la puissance dépend de w = √(Σ(p₁ᵢ − p₀ᵢ)²/p₀ᵢ), l'indice d'effet de Cohen. n ≈ (z_{α/2} + z_β)² / w²
  • ANOVA à un facteur : la taille d'effet est f = σ_entre / σ_intra. La puissance se calcule via la loi F non centrée
  • Corrélation : pour tester H₀ : ρ = 0, n ≈ ((z_{α/2} + z_β) / arctanh(r))² + 3 (transformation de Fisher)
  • Régression : la taille d'effet est f² = R² / (1 − R²), et n ≈ (z_{α/2} + z_β)² / f² + p + 1, où p est le nombre de prédicteurs

Stratégies pour augmenter la puissance

Quand augmenter n n'est pas possible (coût, éthique, rareté), d'autres stratégies existent :

  • Design apparié : mesurer chaque sujet dans les deux conditions (avant/après) réduit la variabilité inter-individuelle
  • Covariables (ANCOVA) : inclure des covariables réduit la variance résiduelle
  • Mesures répétées : plusieurs mesures par sujet augmentent la précision
  • Réduction du bruit : protocoles standardisés, instruments plus précis
  • Test unilatéral : si la direction de l'effet est connue a priori, un test unilatéral est plus puissant
  • Augmenter α : passer à α = 0,10 augmente la puissance mais au prix de plus de faux positifs

Analyse de puissance a priori vs a posteriori

L'analyse de puissance a priori (avant la collecte) est la pratique recommandée. Elle permet de dimensionner correctement l'étude et est souvent exigée par les comités d'éthique et les agences de financement.

L'analyse de puissance a posteriori (après la collecte, en utilisant la taille d'effet observée) est fortement déconseillée car :

  • La puissance calculée est une transformation monotone de la p-value — elle n'apporte aucune information nouvelle
  • Un résultat non significatif donne toujours une puissance a posteriori faible, créant un raisonnement circulaire
  • La taille d'effet observée est une estimation bruitée de la vraie taille d'effet

Analyse de sensibilité

L'analyse de sensibilité est une alternative utile : pour un n fixé et α donné, quel est le plus petit effet détectable avec une puissance de 80 % ? Cela permet de juger si l'étude est capable de détecter des effets d'intérêt pratique.

On peut aussi représenter la courbe de puissance en fonction de la taille d'effet pour un n et α donnés, ce qui montre la capacité discriminante du test sur toute la gamme d'effets possibles.

Question 1 / 6

Quel est le niveau de puissance conventionnel le plus courant ?

La convention est de viser une puissance de 80 % (β = 0,20). Certaines études visent 90 %, mais 80 % est le minimum généralement accepté.

Question 2 / 6

Combien de participants par groupe faut-il pour détecter un effet moyen (d = 0,5) à α = 0,05 et puissance 80 % ?

Pour un test t bilatéral avec d = 0,5, α = 0,05 et puissance 80 %, il faut environ 32 participants par groupe, soit 64 au total.

Question 3 / 6

Comment évolue n quand la taille d'effet δ est divisée par 2 ?

n est inversement proportionnel au carré de la taille d'effet : n ∝ 1/δ². Si δ est divisé par 2, n est multiplié par 4.

Question 4 / 6

Pourquoi l'analyse de puissance a posteriori est-elle déconseillée ?

La puissance a posteriori est une fonction monotone de la p-value : un résultat non significatif donne toujours une faible puissance a posteriori, ce qui est un raisonnement circulaire.

Question 5 / 6

Quel design expérimental augmente la puissance sans augmenter n ?

Le design apparié réduit la variabilité en éliminant les différences inter-individuelles. Chaque sujet sert de son propre contrôle, ce qui augmente la puissance.

Question 6 / 6

Que détermine une analyse de sensibilité ?

L'analyse de sensibilité détermine le plus petit effet que l'étude peut détecter avec la puissance souhaitée. C'est utile quand n est fixé par des contraintes pratiques.

Exercices

Testez vos connaissances

Exercice 1 — Calcul de taille d'échantillon

Un chercheur veut détecter un effet de d = 0,3 (petit-moyen) avec un test t bilatéral à α = 0,05 et puissance 90 %. Calculez n par groupe. (z_{0,025} = 1,96, z_{0,10} = 1,28)

Voir la solution

n = ((z_{α/2} + z_β) / d)² = ((1,96 + 1,28) / 0,3)² = (3,24 / 0,3)² = (10,8)² = 117 par groupe.

Il faut environ 117 participants par groupe, soit 234 au total. Pour une puissance de 80 % au lieu de 90 %, n = ((1,96 + 0,84)/0,3)² = (9,33)² ≈ 88 par groupe.

Exercice 2 — Analyse de sensibilité

Un chercheur dispose de 50 participants par groupe pour un test t bilatéral à α = 0,05. Quel est le plus petit effet détectable avec une puissance de 80 % ?

Voir la solution

On inverse la formule : d = (z_{α/2} + z_β) / √n = (1,96 + 0,84) / √50 = 2,80 / 7,07 = 0,396.

L'étude peut détecter des effets de taille d ≥ 0,40 (moyen) avec 80 % de puissance. Les effets petits (d = 0,2) resteront probablement indétectables.

Exercice 3 — Gain du design apparié

Un test t indépendant nécessite 64 participants par groupe (d = 0,5, α = 0,05, puissance 80 %). Si l'on utilise un design apparié et que la corrélation entre les mesures appariées est ρ = 0,6, combien de paires faut-il ?

Voir la solution

Pour un design apparié, la variance de la différence est réduite d'un facteur 2(1 − ρ) par rapport au design indépendant.

n_apparié = n_indépendant × (1 − ρ) = 64 × (1 − 0,6) = 64 × 0,4 = 26 paires.

Le design apparié nécessite 26 participants au lieu de 128 (2 × 64), soit une réduction de 80 % ! C'est l'avantage majeur des designs appariés quand la corrélation est élevée.

Exercice 4 — Puissance d'un test du Chi-deux

On teste si un dé est truqué (H₀ : toutes les faces équiprobables). Avec 120 lancers et α = 0,05, quelle est la taille d'effet w minimale détectable à 80 % de puissance ? (df = 5)

Voir la solution

Pour un test du Chi-deux : n ≈ (z_{α/2} + z_β)² / w² (approximation).

Plus exactement, on utilise la formule : n = χ²_{α, df} / w² pour obtenir la puissance souhaitée via la distribution non centrée.

En approximation : w = (z_{α/2} + z_β) / √n = 2,80 / √120 = 2,80 / 10,95 = 0,256.

Selon Cohen, w = 0,1 est petit, 0,3 est moyen, 0,5 est grand. L'étude peut détecter des effets proches de moyen.

Exercice 5 — Comparaison de scénarios

Un essai clinique compare un nouveau traitement au placebo. Budget : 200 patients maximum. Comparez la puissance pour α = 0,05 et d = 0,4 dans trois scénarios : (a) 100 patients par groupe, test indépendant ; (b) 100 paires, test apparié avec ρ = 0,5 ; (c) 200 patients par groupe en réduisant α à 0,01.

Voir la solution

(a) 100/groupe, test indépendant : λ = d√(n/2) = 0,4 × √50 = 2,83. Puissance ≈ Φ(2,83 − 1,96) = Φ(0,87) ≈ 81 %.

(b) 100 paires, test apparié (ρ = 0,5) : d_apparié = d / √(2(1−ρ)) = 0,4/√1 = 0,4. λ = 0,4 × √100 = 4,0. Puissance ≈ Φ(4,0 − 1,96) = Φ(2,04) ≈ 98 %.

(c) Scénario impossible : on n'a que 200 patients au total, donc 100/groupe est le maximum. Avec α = 0,01 : Puissance ≈ Φ(2,83 − 2,58) = Φ(0,25) ≈ 60 %.

Le design apparié (b) est nettement supérieur. Réduire α sans augmenter n fait chuter la puissance.
