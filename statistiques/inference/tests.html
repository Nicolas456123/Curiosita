01

Introduction

Décider sur la base des données

Les tests d'hypothèses constituent le cadre formel pour prendre des décisions à partir de données. Un nouveau médicament est-il plus efficace que le placebo ? La proportion de pièces défectueuses dépasse-t-elle le seuil de tolérance ? La moyenne des notes a-t-elle changé entre deux promotions ?

Le principe est de formuler deux hypothèses concurrentes — l'hypothèse nulle H₀ (statu quo) et l'hypothèse alternative H₁ — puis de déterminer si les données fournissent suffisamment de preuves pour rejeter H₀. Cette démarche repose sur le calcul d'une statistique de test et d'une p-value.

Principe clé : on ne « prouve » jamais H₀ ni H₁. On décide seulement s'il y a suffisamment de preuves contre H₀. Ne pas rejeter H₀ ne signifie pas que H₀ est vraie — cela signifie que les données ne sont pas assez concluantes pour la rejeter.

02

Le cours

Logique et pratique des tests

Formulation des hypothèses

Tout test commence par la formulation de deux hypothèses mutuellement exclusives :

  • H₀ (hypothèse nulle) — hypothèse de référence, souvent « pas d'effet » ou « pas de différence ». Exemple : μ = μ₀
  • H₁ (hypothèse alternative) — ce que l'on cherche à montrer. Peut être bilatérale (μ ≠ μ₀), unilatérale à droite (μ > μ₀) ou unilatérale à gauche (μ < μ₀)

Le choix entre test bilatéral et unilatéral doit être fait avant de regarder les données, en fonction de la question de recherche.

Statistique de test et p-value

La statistique de test est une fonction des données qui mesure l'écart entre ce qui est observé et ce qui est attendu sous H₀. Sous H₀, cette statistique suit une loi connue (normale, Student, Chi-deux, Fisher...).

La p-value (valeur p) est la probabilité d'observer un résultat au moins aussi extrême que celui obtenu, si H₀ est vraie. Plus la p-value est petite, plus les données sont incompatibles avec H₀.

  • Si p-value ≤ α → on rejette H₀ (résultat « statistiquement significatif »)
  • Si p-value > α → on ne rejette pas H₀

Attention : la p-value n'est PAS la probabilité que H₀ soit vraie. C'est P(données aussi extrêmes | H₀ vraie), et non P(H₀ vraie | données). Confondre les deux est l'une des erreurs les plus fréquentes en statistique.

Test de Student (t-test)

Le test de Student est utilisé pour comparer des moyennes :

  • Test à un échantillon : H₀ : μ = μ₀. Statistique : t = (X̄ − μ₀) / (S*/√n), suit une loi t(n−1)
  • Test à deux échantillons indépendants : H₀ : μ₁ = μ₂. Statistique : t = (X̄₁ − X̄₂) / √(S²p(1/n₁ + 1/n₂)), où S²p est la variance poolée
  • Test apparié : pour des mesures avant/après sur les mêmes individus. On teste H₀ : μ_D = 0 sur les différences D = X₁ − X₂

Le test de Welch est une variante du test à deux échantillons qui ne suppose pas l'égalité des variances.

Test du Chi-deux

Le test du Chi-deux (χ²) s'applique aux données catégorielles :

  • Test d'adéquation : les données observées suivent-elles une distribution théorique ? χ² = Σ (O − E)² / E, où O = effectifs observés, E = effectifs attendus
  • Test d'indépendance : deux variables catégorielles sont-elles indépendantes dans un tableau de contingence ? χ² = Σ (Oᵢⱼ − Eᵢⱼ)² / Eᵢⱼ, avec (r−1)(c−1) degrés de liberté
  • Test d'homogénéité : les proportions sont-elles les mêmes dans plusieurs populations ?

Condition : tous les effectifs théoriques doivent être ≥ 5. Sinon, on utilise le test exact de Fisher.

Démarche complète d'un test

La procédure standard comprend cinq étapes :

  • 1. Formuler H₀ et H₁
  • 2. Choisir le seuil de significativité α (souvent 0,05)
  • 3. Calculer la statistique de test
  • 4. Déterminer la p-value ou la région critique
  • 5. Conclure : rejeter ou non H₀

Tests non paramétriques

Quand les hypothèses de normalité ne sont pas satisfaites ou que les données sont ordinales, on utilise des tests non paramétriques :

  • Test de Wilcoxon signé : alternative au test t apparié, basé sur les rangs des différences
  • Test de Mann-Whitney (Wilcoxon rang-somme) : alternative au test t à deux échantillons indépendants
  • Test de Kruskal-Wallis : alternative non paramétrique à l'ANOVA à un facteur
  • Test de Kolmogorov-Smirnov : teste si un échantillon suit une distribution spécifiée

Ces tests sont moins puissants que leurs équivalents paramétriques quand les hypothèses sont satisfaites, mais plus robustes en cas de violation de ces hypothèses.

Corrections pour tests multiples

Quand on réalise plusieurs tests simultanément, le risque global de faux positif augmente. Si l'on effectue m tests indépendants au seuil α, la probabilité d'au moins un faux positif est 1 − (1−α)^m. Pour m = 20 tests à α = 0,05, ce risque atteint 64 %.

  • Correction de Bonferroni : on utilise le seuil α/m pour chaque test. Simple mais conservatrice
  • Correction de Holm : version séquentielle moins conservatrice de Bonferroni
  • Taux de fausse découverte (FDR) : méthode de Benjamini-Hochberg, contrôle la proportion attendue de faux positifs parmi les rejets

Question 1 / 6

Que représente la p-value ?

La p-value est P(résultat ≥ observé | H₀). C'est une probabilité sur les données, pas sur les hypothèses.

Question 2 / 6

Quand rejette-t-on H₀ au seuil α = 0,05 ?

On rejette H₀ quand la p-value est inférieure ou égale au seuil α. Cela signifie que les données sont suffisamment improbables sous H₀.

Question 3 / 6

Quel test utilise-t-on pour vérifier l'indépendance de deux variables catégorielles ?

Le test du Chi-deux d'indépendance s'applique aux tableaux de contingence et vérifie si deux variables catégorielles sont liées.

Question 4 / 6

Le test de Welch est une variante du test t qui ne suppose pas :

Le test de Welch ajuste les degrés de liberté pour tenir compte de variances potentiellement différentes entre les deux groupes.

Question 5 / 6

Pourquoi doit-on corriger pour les tests multiples ?

Sans correction, le risque de trouver au moins un faux positif augmente rapidement avec le nombre de tests. La correction de Bonferroni ou le contrôle du FDR limitent ce risque.

Question 6 / 6

Que signifie « ne pas rejeter H₀ » ?

Ne pas rejeter H₀ signifie que les données sont compatibles avec H₀, mais cela ne prouve pas que H₀ est vraie. L'absence de preuve n'est pas la preuve de l'absence.

Exercices

Testez vos connaissances

Exercice 1 — Test t à un échantillon

Un fabricant affirme que ses barres de chocolat pèsent 100 g en moyenne. Un contrôle qualité sur 16 barres donne X̄ = 98,5 g et S* = 2,4 g. Au seuil α = 0,05, peut-on rejeter l'affirmation du fabricant ? (t_{0,025;15} = 2,131)

Voir la solution

H₀ : μ = 100, H₁ : μ ≠ 100 (test bilatéral).

t = (98,5 − 100) / (2,4/√16) = −1,5 / 0,6 = −2,5.

|t| = 2,5 > t_{0,025;15} = 2,131. On rejette H₀.

Les données fournissent suffisamment de preuves pour conclure que la moyenne diffère de 100 g.

Exercice 2 — Test du Chi-deux d'indépendance

On étudie le lien entre le sexe et la préférence pour un produit (A ou B) sur 200 personnes. Hommes : 60 préfèrent A, 40 préfèrent B. Femmes : 45 préfèrent A, 55 préfèrent B. Testez l'indépendance au seuil 0,05. (χ²_{0,05;1} = 3,841)

Voir la solution

Tableau de contingence : total A = 105, total B = 95, total H = 100, total F = 100.

Effectifs théoriques : E(H,A) = 100×105/200 = 52,5 ; E(H,B) = 47,5 ; E(F,A) = 52,5 ; E(F,B) = 47,5.

χ² = (60−52,5)²/52,5 + (40−47,5)²/47,5 + (45−52,5)²/52,5 + (55−47,5)²/47,5

= 1,071 + 1,184 + 1,071 + 1,184 = 4,51.

4,51 > 3,841 → on rejette H₀. Il y a un lien significatif entre le sexe et la préférence.

Exercice 3 — Test t à deux échantillons

On compare les scores de deux groupes d'étudiants. Groupe 1 : n₁ = 12, X̄₁ = 75, S₁ = 8. Groupe 2 : n₂ = 15, X̄₂ = 70, S₂ = 10. Au seuil 0,05, y a-t-il une différence significative ? (t_{0,025;25} ≈ 2,060)

Voir la solution

H₀ : μ₁ = μ₂, H₁ : μ₁ ≠ μ₂.

Variance poolée : S²p = ((11×64) + (14×100)) / (12+15−2) = (704+1400)/25 = 84,16. Sp = 9,17.

t = (75−70) / (9,17 × √(1/12+1/15)) = 5 / (9,17 × 0,3928) = 5 / 3,602 = 1,388.

|t| = 1,388 < 2,060 → on ne rejette pas H₀. La différence n'est pas significative à 5 %.

Exercice 4 — Test unilatéral

Un nouveau procédé est censé réduire le temps de production. Avec l'ancien procédé, μ₀ = 30 min. Sur 25 essais du nouveau procédé : X̄ = 28,2 min, S* = 4 min. Testez H₁ : μ < 30 au seuil 0,05. (t_{0,05;24} = 1,711)

Voir la solution

H₀ : μ = 30, H₁ : μ < 30 (unilatéral à gauche).

t = (28,2 − 30) / (4/√25) = −1,8 / 0,8 = −2,25.

|t| = 2,25 > 1,711 → on rejette H₀. Le nouveau procédé réduit significativement le temps de production.

Exercice 5 — Correction de Bonferroni

Un chercheur effectue 10 tests d'hypothèses indépendants au seuil α = 0,05. Les p-values obtenues sont : 0,003, 0,012, 0,024, 0,038, 0,041, 0,067, 0,089, 0,112, 0,234, 0,567. Quels tests restent significatifs après correction de Bonferroni ?

Voir la solution

Le seuil corrigé de Bonferroni est α/m = 0,05/10 = 0,005.

Seul le test avec p = 0,003 est significatif après correction (0,003 < 0,005).

Les 4 autres tests initialement significatifs (0,012, 0,024, 0,038, 0,041) ne le sont plus. La correction de Bonferroni est conservatrice : elle réduit fortement le nombre de rejets pour contrôler le risque global de faux positifs.
