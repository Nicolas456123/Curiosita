Introduction

Interagir avec le monde extérieur

Un programme doit pouvoir communiquer : afficher des résultats à l'écran et recevoir des données de l'utilisateur. En Python, print() affiche du texte et input() attend une saisie clavier. Les f-strings permettent de formater l'affichage avec élégance.

Important : input() retourne toujours une chaîne de caractères (str). Si vous attendez un nombre, il faut le convertir avec int() ou float().

02

Le cours

print(), input() et f-strings

La fonction print()

print() affiche du texte dans la console. Elle accepte plusieurs arguments et offre des options de formatage.

Python
# print() — affichage de base
print("Bonjour le monde")

# Plusieurs arguments — séparés par un espace par défaut
print("Python", "est", "génial")
# Python est génial

# Changer le séparateur avec sep
print("2026", "02", "28", sep="-")
# 2026-02-28

# Changer la fin de ligne avec end
print("Chargement", end="...")
print("Terminé !")
# Chargement...Terminé !

Les f-strings

Les f-strings (Python 3.6+) sont la méthode la plus lisible pour insérer des variables dans du texte.

Python
# f-strings — le formatage moderne
nom = "Alice"
age = 25
print(f"Bonjour {nom}, tu as {age} ans.")

# Expressions dans les accolades
prix = 19.99
quantite = 3
print(f"Total : {prix * quantite:.2f} euros")
# Total : 59.97 euros

# Formatage des nombres
pi = 3.14159265
print(f"Pi ≈ {pi:.2f}")        # Pi ≈ 3.14
print(f"Grand : {1000000:,}")  # Grand : 1,000,000
print(f"Pourcentage : {0.85:.0%}")  # Pourcentage : 85%

La fonction input()

input() affiche un message et attend que l'utilisateur tape du texte. Elle retourne toujours un str.

Python
# input() — saisie utilisateur
nom = input("Quel est votre nom ? ")
print(f"Bonjour {nom} !")

# Conversion nécessaire pour les nombres
age_str = input("Quel âge avez-vous ? ")
age = int(age_str)  # str → int

# Version compacte en une ligne
taille = float(input("Votre taille en m ? "))

# Utilisation avec une condition
if age >= 18:
    print("Vous êtes majeur")
else:
    print(f"Encore {18 - age} ans à attendre")

Méthodes de chaînes utiles

Les chaînes de caractères possèdent de nombreuses méthodes intégrées pour les transformer.

Python
# Méthodes de chaînes courantes
texte = "  Bonjour le Monde  "

print(texte.strip())       # "Bonjour le Monde" (sans espaces)
print(texte.lower())       # "  bonjour le monde  "
print(texte.upper())       # "  BONJOUR LE MONDE  "
print(texte.strip().split())  # ["Bonjour", "le", "Monde"]

mots = ["Python", "est", "cool"]
print(" ".join(mots))  # "Python est cool"

print("hello".replace("l", "r"))  # "herro"
print("Python".startswith("Py"))  # True

Formatage avancé avec f-strings

Les f-strings offrent des options de formatage puissantes pour aligner du texte, formater des nombres et afficher des données de manière structurée.

Python
# Alignement et largeur de champ
nom = "Alice"
print(f"|{nom:<15}|")   # |Alice          | (aligné à gauche)
print(f"|{nom:>15}|")   # |          Alice| (aligné à droite)
print(f"|{nom:^15}|")   # |     Alice     | (centré)
print(f"|{nom:*^15}|")  # |*****Alice*****| (remplissage)

# Formatage de nombres
n = 1234567
print(f"{n:,}")          # 1,234,567 (séparateur de milliers)
print(f"{n:_}")          # 1_234_567 (avec underscore)

# Bases numériques
x = 255
print(f"Décimal: {x:d}")   # 255
print(f"Binaire: {x:b}")   # 11111111
print(f"Octal: {x:o}")     # 377
print(f"Hexa: {x:x}")      # ff

# Débogage rapide (Python 3.8+)
prix = 19.99
quantite = 3
print(f"{prix=}, {quantite=}")
# prix=19.99, quantite=3

f"{variable=}" : cette syntaxe (Python 3.8+) affiche automatiquement le nom et la valeur de la variable. Très pratique pour le débogage rapide.

Validation des entrées utilisateur

L'utilisateur peut saisir n'importe quoi. Il faut vérifier et gérer les erreurs de saisie avec try/except.

Python
# Gestion d'erreur avec try/except
while True:
    saisie = input("Entrez un nombre entier : ")
    try:
        nombre = int(saisie)
        break  # Sortir de la boucle si conversion réussie
    except ValueError:
        print(f"'{saisie}' n'est pas un entier valide.")

print(f"Vous avez saisi : {nombre}")

# Valider un choix parmi des options
choix_valides = ["oui", "non"]
while True:
    choix = input("Continuer ? (oui/non) : ").lower()
    if choix in choix_valides:
        break
    print("Répondez par 'oui' ou 'non'.")

Pièges courants

Python
# PIÈGE 1 : input() retourne TOUJOURS un str
age = input("Âge : ")
# age > 18  → TypeError ! (on compare str et int)
age = int(input("Âge : "))  # ✓ convertir d'abord

# PIÈGE 2 : Oublier le \n dans print()
# Chaque print() ajoute un retour à la ligne par défaut
# Pour éviter ça :
print("A", end="")
print("B")  # Affiche : AB

# PIÈGE 3 : Accolades littérales dans f-strings
# Pour afficher { } dans une f-string, doubler les accolades
print(f"Un dict : {{'a': 1}}")  # Un dict : {'a': 1}

# PIÈGE 4 : .strip() oublié après input()
# L'utilisateur peut ajouter des espaces involontaires
nom = input("Nom : ").strip()  # ✓ enlever les espaces

Bonnes pratiques

  • Toujours valider : ne faites jamais confiance aux données saisies par l'utilisateur. Utilisez try/except pour les conversions.
  • f-strings : préférez f"Bonjour {nom}" à "Bonjour " + nom ou "Bonjour %s" % nom.
  • .strip() : ajoutez toujours .strip() après input() pour supprimer les espaces superflus.
  • Messages clairs : indiquez dans le prompt le format attendu : "Âge (nombre entier) : ".
  • Espace final : mettez un espace à la fin du prompt de input() pour la lisibilité : input("Nom : ").

Question 1 / 6

Quel type retourne input() ?

input() retourne toujours une chaîne de caractères (str), même si l'utilisateur tape un nombre. Il faut convertir explicitement avec int() ou float().

Question 2 / 6

Que produit print("A", "B", "C", sep="-") ?

Le paramètre sep définit le séparateur entre les arguments de print(). Ici, "-" remplace l'espace par défaut.

Question 3 / 6

Que retourne f"{3.14159:.2f}" ?

Le format :.2f arrondit le nombre à 2 décimales. f signifie « fixed-point » (nombre à virgule fixe). Résultat : "3.14".

Question 4 / 6

Que produit f"{255:x}" ?

Le format :x convertit un nombre en notation hexadécimale (en minuscules). 255 en hexadécimal est ff. Utilisez :X pour des majuscules.

Question 5 / 6

Quelle méthode de chaîne supprime les espaces en début et fin ?

.strip() enlève les espaces (et retours à la ligne) en début et fin de chaîne. .lstrip() n'enlève qu'à gauche, .rstrip() qu'à droite.

Question 6 / 6

Que produit f"{'Python':*^15}" ?

Le format :*^15 centre le texte dans 15 caractères et remplit avec *. « Python » fait 6 caractères, il reste 9 : 4 à gauche et 5 à droite.

Exercice 1 — Carte de visite

Demandez le prénom, le nom, l'âge et la profession de l'utilisateur. Affichez une carte de visite formatée avec alignement. Exemple :

╔══════════════════════════╗
║   Prénom : Alice         ║
║   Nom    : Dupont        ║
║   Âge    : 25 ans        ║
║   Métier : Développeuse  ║
╚══════════════════════════╝
Voir la solution 
prenom = input("Prénom : ").strip()
nom = input("Nom : ").strip()
age = input("Âge : ").strip()
metier = input("Profession : ").strip()

largeur = 26
print("+" + "=" * largeur + "+")
print(f"|   {'Prénom':<8}: {prenom:<{largeur - 14}}|")
print(f"|   {'Nom':<8}: {nom:<{largeur - 14}}|")
print(f"|   {'Âge':<8}: {age + ' ans':<{largeur - 14}}|")
print(f"|   {'Métier':<8}: {metier:<{largeur - 14}}|")
print("+" + "=" * largeur + "+")

Exercice 2 — Saisie sécurisée

Créez un programme qui demande un nombre entre 1 et 100. Si l'utilisateur entre une valeur invalide (pas un nombre, ou hors limites), affichez un message d'erreur et redemandez. Comptez le nombre de tentatives.

Voir la solution 
tentatives = 0

while True:
    saisie = input("Nombre entre 1 et 100 : ").strip()
    tentatives += 1
    try:
        nombre = int(saisie)
        if 1 <= nombre <= 100:
            break
        else:
            print("Le nombre doit être entre 1 et 100.")
    except ValueError:
        print(f"'{saisie}' n'est pas un nombre valide.")

print(f"Merci ! Vous avez saisi {nombre} en {tentatives} tentative(s).")

Exercice 3 — Tableau formaté

Affichez un tableau de prix avec alignement. Les noms des produits doivent être alignés à gauche, les prix à droite avec 2 décimales, et les quantités centrées.

Voir la solution 
produits = [
    ("Pommes", 2.50, 3),
    ("Pain", 1.20, 1),
    ("Lait", 0.95, 2),
    ("Fromage", 4.80, 1),
]

print(f"{'Produit':<12} {'Qté':^5} {'Prix':>8} {'Total':>8}")
print("-" * 37)

total_general = 0
for nom, prix, qte in produits:
    total = prix * qte
    total_general += total
    print(f"{nom:<12} {qte:^5} {prix:>7.2f}e {total:>7.2f}e")

print("-" * 37)
print(f"{'TOTAL':>25} {total_general:>7.2f}e")

Exercice 4 — Mot de passe

Demandez à l'utilisateur de créer un mot de passe. Vérifiez qu'il contient au moins 8 caractères, au moins une majuscule, une minuscule et un chiffre. Affichez des messages d'erreur précis pour chaque critère non rempli.

Voir la solution 
while True:
    mdp = input("Créez un mot de passe : ")
    erreurs = []

    if len(mdp) < 8:
        erreurs.append("au moins 8 caractères")
    if not any(c.isupper() for c in mdp):
        erreurs.append("au moins une majuscule")
    if not any(c.islower() for c in mdp):
        erreurs.append("au moins une minuscule")
    if not any(c.isdigit() for c in mdp):
        erreurs.append("au moins un chiffre")

    if not erreurs:
        print("Mot de passe accepté !")
        break
    else:
        print("Mot de passe invalide. Il faut :")
        for e in erreurs:
            print(f"  - {e}")
