Introduction

Pourquoi les variables sont essentielles

Une variable est un nom qui fait référence à une valeur stockée en mémoire. En Python, créer une variable est extrêmement simple : il suffit d'écrire un nom, le signe = et une valeur. Pas besoin de déclarer le type — Python le détermine automatiquement.

Typage dynamique : contrairement à Java ou C, Python détermine le type d'une variable au moment de l'exécution. Une même variable peut changer de type au cours du programme (même si c'est rarement recommandé).

Les quatre types fondamentaux sont : int (entiers), float (décimaux), str (chaînes de caractères) et bool (booléens True/False). Maîtriser ces types est la première étape pour écrire n'importe quel programme Python.

Le cours

Variables, types et conversions

Créer une variable

En Python, on crée une variable en lui assignant une valeur avec l'opérateur =. Le nom de la variable doit commencer par une lettre ou un underscore, et ne peut contenir que des lettres, chiffres et underscores.

Python
# Créer des variables
nom = "Alice"
age = 25
taille = 1.68
est_etudiant = True

# Python détecte automatiquement le type
print(type(nom))           # <class 'str'>
print(type(age))           # <class 'int'>
print(type(taille))       # <class 'float'>
print(type(est_etudiant)) # <class 'bool'>

Les types fondamentaux

Python possède quatre types de base que tout débutant doit connaître :

  • int — Nombres entiers : 42, -7, 0. Pas de limite de taille en Python.
  • float — Nombres décimaux : 3.14, -0.5, 2.0. Attention aux erreurs d'arrondi.
  • str — Chaînes de caractères : "hello", 'monde'. Délimitées par des guillemets simples ou doubles.
  • bool — Booléens : True ou False. Utilisés pour les conditions et la logique.
Python
# Entiers (int) — pas de limite de taille
petit = 42
grand = 10 ** 100  # un googol, 1 suivi de 100 zéros

# Flottants (float) — nombres décimaux
pi = 3.14159
negatif = -0.5

# Chaînes (str) — guillemets simples ou doubles
prenom = "Marie"
message = 'Bonjour le monde'
multiligne = """Ceci est
un texte sur
plusieurs lignes"""

# Booléens (bool)
actif = True
termine = False

Conversions de types

On peut convertir une valeur d'un type à un autre avec les fonctions int(), float(), str() et bool(). C'est le casting (transtypage).

Python
# Conversions de types
x = "42"
y = int(x)        # str → int : 42
z = float(x)      # str → float : 42.0

age = 25
texte = str(age)   # int → str : "25"

# bool() — tout ce qui est "vide" ou zéro est False
print(bool(0))       # False
print(bool(""))      # False (chaîne vide)
print(bool(42))      # True
print(bool("hello")) # True

Conventions de nommage

En Python, on utilise le snake_case pour les noms de variables et fonctions : des mots en minuscules séparés par des underscores.

Python
# Bonnes pratiques de nommage
nom_complet = "Jean Dupont"      # snake_case ✓
nombre_de_cours = 5              # descriptif ✓
TAUX_TVA = 0.20                  # CONSTANTE en majuscules ✓

# À éviter
# x = "Jean Dupont"    → pas descriptif
# NomComplet = ...     → CamelCase réservé aux classes
# 2eme_nom = ...       → ne peut pas commencer par un chiffre

Affectation multiple et échange de variables

Python permet d'affecter plusieurs variables en une seule ligne, et d'échanger leurs valeurs sans variable temporaire.

Python
# Affectation multiple
x, y, z = 1, 2, 3
print(x, y, z)  # 1 2 3

# Même valeur pour plusieurs variables
a = b = c = 0

# Échange de variables (swap) — élégant !
a, b = 10, 20
a, b = b, a
print(a, b)  # 20 10

# Unpacking depuis une liste ou un tuple
coordonnees = (48.8566, 2.3522)
latitude, longitude = coordonnees
print(f"Paris : {latitude}°N, {longitude}°E")

None — l'absence de valeur

None est une valeur spéciale en Python qui représente l'absence de valeur. C'est le seul objet de type NoneType.

Python
# None représente "rien" ou "pas de valeur"
resultat = None
print(type(resultat))  # <class 'NoneType'>

# Tester si une variable est None
if resultat is None:
    print("Pas encore de résultat")

# Attention : utiliser 'is' et non '==' pour None
print(resultat is None)    # True ✓ (bonne pratique)
print(resultat == None)   # True mais déconseillé

# None est "falsy"
print(bool(None))  # False

is vs == : is vérifie l'identité (même objet en mémoire), tandis que == vérifie l'égalité de valeur. Pour None, utilisez toujours is None ou is not None.

Pièges courants

Voici les erreurs les plus fréquentes chez les débutants avec les variables et les types.

Python
# PIÈGE 1 : Confondre = (affectation) et == (comparaison)
x = 5       # affectation : x vaut 5
x == 5      # comparaison : retourne True (ne modifie pas x)

# PIÈGE 2 : Concaténer str et int sans conversion
age = 25
# print("J'ai " + age + " ans")  → TypeError !
print("J'ai " + str(age) + " ans")  # ✓
print(f"J'ai {age} ans")             # ✓ plus élégant

# PIÈGE 3 : Erreurs d'arrondi avec les float
print(0.1 + 0.2)          # 0.30000000000000004 !
print(0.1 + 0.2 == 0.3)   # False !
print(round(0.1 + 0.2, 1) == 0.3)  # True ✓

# PIÈGE 4 : int() ne gère pas les décimaux en str
# int("3.14")  → ValueError !
int(float("3.14"))  # 3 ✓ — convertir en float d'abord

Bonnes pratiques

  • Noms descriptifs : préférez nombre_etudiants à n. Le code est lu beaucoup plus souvent qu'il n'est écrit.
  • snake_case : mon_variable pour les variables et fonctions, MA_CONSTANTE pour les constantes, MaClasse pour les classes.
  • f-strings : préférez f"Bonjour {nom}" à la concaténation "Bonjour " + nom.
  • type() pour déboguer : quand une erreur survient, utilisez print(type(variable)) pour vérifier le type réel.
  • Évitez de changer le type : même si Python le permet, ne réassignez pas une variable avec un type différent (x = 5 puis x = "hello").

Astuce : utilisez isinstance() plutôt que type() pour vérifier un type, car elle gère aussi l'héritage : isinstance(42, int) retourne True.

Question 1 / 6

Quel est le type de la valeur 3.14 en Python ?

3.14 est un nombre décimal, donc de type float en Python. Les entiers sans point décimal sont de type int.

Question 2 / 6

Que retourne bool("") ?

Une chaîne vide "" est considérée comme « fausse » en Python. De même, 0, [], {} et None sont tous évalués à False.

Question 3 / 6

Que retourne type(int("42")) ?

int("42") convertit la chaîne "42" en entier 42. type(42) retourne <class 'int'>.

Question 4 / 6

Que retourne 0.1 + 0.2 == 0.3 ?

En raison de la représentation en virgule flottante, 0.1 + 0.2 donne 0.30000000000000004, qui n'est pas strictement égal à 0.3. Utilisez round() pour comparer des flottants.

Question 5 / 6

Après a, b = 5, 10 puis a, b = b, a, que vaut a ?

L'instruction a, b = b, a échange les valeurs de a et b. Après l'échange, a vaut 10 et b vaut 5.

Question 6 / 6

Que se passe-t-il avec int("3.14") ?

int() ne peut pas convertir directement une chaîne contenant un point décimal. Il faut d'abord passer par float() : int(float("3.14")) donne 3.

Exercices

Mettez en pratique

Exercice 1 — Fiche d'identité

Créez des variables pour stocker le nom, l'âge, la taille (en mètres) et la nationalité d'une personne. Affichez ensuite une phrase formatée comme : "Jean Dupont, 30 ans, 1.75 m, nationalité française".

Voir la solution 
nom = "Jean Dupont"
age = 30
taille = 1.75
nationalite = "française"

print(f"{nom}, {age} ans, {taille} m, nationalité {nationalite}")

Exercice 2 — Convertisseur de température

Demandez à l'utilisateur une température en Celsius (avec input()), convertissez-la en Fahrenheit avec la formule F = C * 9/5 + 32, et affichez le résultat arrondi à une décimale.

Voir la solution 
celsius = float(input("Température en °C : "))
fahrenheit = celsius * 9 / 5 + 32
print(f"{celsius}°C = {fahrenheit:.1f}°F")

Exercice 3 — Vérificateur de types

Créez une variable pour chacun des 4 types fondamentaux (int, float, str, bool). Affichez chaque variable avec son type en utilisant type() et les f-strings. Exemple de sortie : "42 est de type <class 'int'>".

Voir la solution 
entier = 42
decimal = 3.14
chaine = "Bonjour"
booleen = True

for var in [entier, decimal, chaine, booleen]:
    print(f"{var} est de type {type(var)}")

Exercice 4 — Échange et affichage

Créez deux variables a = 100 et b = 200. Échangez leurs valeurs sans utiliser de variable temporaire, puis affichez le résultat pour prouver que l'échange a fonctionné.

Voir la solution 
a = 100
b = 200
print(f"Avant : a = {a}, b = {b}")

a, b = b, a
print(f"Après : a = {a}, b = {b}")
# Avant : a = 100, b = 200
# Après : a = 200, b = 100
