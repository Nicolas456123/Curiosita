Introduction

Pourquoi la prononciation anglaise est un défi

L'anglais est célèbre pour l'irrégularité de sa prononciation. Contrairement au français ou à l'espagnol, l'orthographe anglaise ne reflète souvent pas la prononciation. Le mot enough se prononce /ɪˈnʌf/, où le gh se prononce /f/ et le ou se prononce /ʌ/.

Le saviez-vous ? Les combinaisons de lettres ough peuvent se prononcer de 7 manières différentes : through (/uː/), though (/oʊ/), thought (/ɔː/), tough (/ʌf/), cough (/ɒf/), bough (/aʊ/), hiccough (/ʌp/).

Ce cours vous apprend les sons fondamentaux de l'anglais, le fonctionnement de l'accent tonique (stress) et les règles d'intonation qui donnent au discours anglais sa musicalité caractéristique.

02

Le cours

Sons, accentuation et intonation

Les sons difficiles pour les francophones

Certains sons anglais n'existent pas en français. Les maîtriser est essentiel pour être compris :

Le « th » (/θ/ et /ð/) — Le son le plus emblématique de l'anglais :

  • /θ/ (sourd) : think, three, bath — langue entre les dents, sans vibration
  • /ð/ (sonore) : this, the, mother — langue entre les dents, avec vibration

Le « h » aspiré — En français, le h est muet. En anglais, il se prononce comme un souffle :

  • house /haʊs/, happy /ˈhæpi/, behind /bɪˈhaɪnd/

Le schwa /ə/ — La voyelle la plus fréquente en anglais, absente du français standard :

  • about /əˈbaʊt/, banana /bəˈnɑːnə/, the /ðə/

L'accent tonique (word stress)

Chaque mot anglais de plus d'une syllabe a une syllabe accentuée (prononcée plus fort et plus longuement). L'accent tonique peut changer le sens du mot :

Exemples :
REcord (nom = un disque) vs reCORD (verbe = enregistrer)
PREsent (nom = un cadeau) vs preSENT (verbe = présenter)
OBject (nom = un objet) vs obJECT (verbe = s'opposer)

Règles générales :

  • Les mots de 2 syllabes : accent sur la 1re syllabe pour les noms, sur la 2e pour les verbes
  • Les suffixes -tion, -sion, -ic : accent sur la syllabe précédente
  • Les suffixes -ity, -ify, -ogy : accent 2 syllabes avant

La prononciation du « -ed » final

La terminaison du passé -ed se prononce de trois manières différentes :

  • /t/ après un son sourd : walked, stopped, watched
  • /d/ après un son sonore : played, opened, called
  • /ɪd/ après /t/ ou /d/ : wanted, needed, decided

L'intonation

L'intonation anglaise suit des schémas prévisibles :

  • Descendante ↘ : affirmations et questions en WH- (Where do you live? ↘)
  • Montante ↗ : questions fermées (yes/no) (Do you like coffee? ↗)
  • Montante puis descendante ↗↘ : énumérations (I bought apples↗, oranges↗, and bananas↘)

Les lettres muettes

L'anglais regorge de lettres muettes, héritage de son histoire complexe. Connaître les principales évite de nombreuses erreurs :

  • K muet devant N : knife /naɪf/, knee /niː/, know /noʊ/, knight /naɪt/
  • W muet devant R : write /raɪt/, wrong /rɒŋ/, wrist /rɪst/
  • B muet après M : comb /koʊm/, bomb /bɒm/, climb /klaɪm/
  • GH muet : night /naɪt/, light /laɪt/, daughter /ˈdɔːtər/
  • L muet : half /hæf/, talk /tɔːk/, salmon /ˈsæmən/

Règle pratique : Il n'existe pas de règle universelle pour les lettres muettes en anglais. La seule solution est de les apprendre mot par mot. L'écoute régulière de l'anglais authentique est le meilleur moyen de les intégrer naturellement.

La prononciation du « -s » final

Comme le « -ed », la terminaison -s (pluriel, 3e personne) se prononce de trois manières :

  • /s/ après un son sourd : cats /kæts/, books /bʊks/, stops /stɒps/
  • /z/ après un son sonore : dogs /dɒɡz/, plays /pleɪz/, runs /rʌnz/
  • /ɪz/ après /s/, /z/, /ʃ/, /ʒ/, /tʃ/, /dʒ/ : buses /ˈbʌsɪz/, watches /ˈwɒtʃɪz/

Pièges courants de prononciation

Les mots anglais dont la prononciation surprend le plus les francophones :

  • Comfortable → /ˈkʌmftəbəl/ (3 syllabes, pas 4 !)
  • Wednesday → /ˈwɛnzdeɪ/ (le « d » du milieu est muet)
  • Clothes → /kloʊðz/ (une syllabe, pas deux)
  • Colonel → /ˈkɜːrnəl/ (prononcé comme « kernel » !)
  • Queue → /kjuː/ (les quatre dernières lettres sont muettes)
  • Receipt → /rɪˈsiːt/ (le « p » est muet)

Quiz

Testez votre compréhension

🔊 Prononciation — Quiz Score : 0 / 6

Question 1 / 6

Comment se prononce le « -ed » dans « walked » ?

Le /k/ de « walk » est un son sourd, donc le « -ed » se prononce /t/. On dit /wɔːkt/.

Question 2 / 6

Où se place l'accent tonique dans le mot « preSENTation » ?

Le suffixe « -tion » attire l'accent sur la syllabe qui le précède immédiatement : presen-TA-tion. C'est une règle générale pour tous les mots en -tion/-sion.

Question 3 / 6

Quelle est la voyelle la plus fréquente en anglais ?

Le schwa /ə/ est la voyelle la plus fréquente en anglais. C'est une voyelle neutre, non accentuée, présente dans presque tous les mots longs (« about », « banana », « comfortable »).

Question 4 / 6

Quelle lettre est muette dans « knight » ?

Le « k » est muet devant « n » : knight se prononce /naɪt/. Cette règle s'applique aussi à know, knife, knee, knock.

Question 5 / 6

Comment se prononce le « -s » dans « dogs » ?

Le /g/ de « dog » est un son sonore, donc le « -s » se prononce /z/. On dit /dɒɡz/. Le « -s » ne se prononce /s/ qu'après les sons sourds.

Question 6 / 6

Combien de syllabes a le mot « comfortable » en anglais courant ?

En anglais courant, « comfortable » se prononce en 3 syllabes : /ˈkʌmf-tə-bəl/. La syllabe « -or- » est avalée. Les francophones ont tendance à prononcer les 4 syllabes, ce qui sonne non naturel.

Exercices

Mettez en pratique

Exercice 1 — Prononciation du « -ed »

Classez ces verbes au passé selon la prononciation du « -ed » : /t/, /d/ ou /ɪd/ :

played, stopped, wanted, opened, watched, decided, called, worked

Voir la solution

/t/ (après son sourd) : stopped, watched, worked
/d/ (après son sonore) : played, opened, called
/ɪd/ (après /t/ ou /d/) : wanted, decided

Exercice 2 — Trouvez les lettres muettes

Dans chaque mot, identifiez la ou les lettres muettes :

1. knife — 2. Wednesday — 3. climb — 4. receipt — 5. salmon — 6. write

Voir la solution

1. knife → le K est muet devant N
2. Wednesday → le D est muet
3. climb → le B est muet après M
4. receipt → le P est muet
5. salmon → le L est muet (prononcé /ˈsæmən/)
6. write → le W est muet devant R

Exercice 3 — Accent tonique

Indiquez la syllabe accentuée dans chaque mot (majuscules) :

1. photograph — 2. photography — 3. import (nom) — 4. import (verbe) — 5. education

Voir la solution

1. PHO-to-graph → accent sur la 1re syllabe
2. pho-TOG-ra-phy → accent sur la 2e syllabe (le suffixe -y attire l'accent)
3. IM-port (nom) → accent sur la 1re syllabe
4. im-PORT (verbe) → accent sur la 2e syllabe
5. e-du-CA-tion → accent sur la syllabe avant -tion
