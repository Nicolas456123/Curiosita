Introduction

Le vocabulaire espagnol pour les francophones

Le vocabulaire espagnol est très accessible pour un francophone. Les deux langues partagent la même racine latine, ce qui signifie que des milliers de mots sont similaires ou identiques : familia, universidad, importante, diferente.

Le piège des faux amis : Certains mots semblent identiques au français mais ont un sens différent. Embarazada ne signifie pas « embarrassée » mais « enceinte ». Ces confusions peuvent mener à des situations cocasses !

Ce cours présente le vocabulaire essentiel du quotidien, les faux amis les plus dangereux, et les familles de mots qui vous permettront d'enrichir rapidement votre expression en espagnol.

02

Le cours

Vocabulaire essentiel et faux amis

Les faux amis franco-espagnols

Ces mots ressemblent au français mais ont un sens différent :

  • Embarazada = enceinte (≠ embarrassée → avergonzada)
  • Constipado = enrhumé (≠ constipé → estreñido)
  • Largo = long (≠ large → ancho)
  • Éxito = succès (≠ exit/sortie → salida)
  • Conducir = conduire (une voiture) (≠ conduire/mener → llevar)
  • Contestar = répondre (≠ contester → disputar)
  • Guardar = garder, conserver (≠ regarder → mirar)
  • Quitar = enlever (≠ quitter → salir de, dejar)

Vocabulaire du quotidien

La famille (la familia) :

  • Los padres → les parents
  • El hermano / la hermana → le frère / la sœur
  • El hijo / la hija → le fils / la fille
  • El abuelo / la abuela → le grand-père / la grand-mère
  • El marido / la mujer → le mari / la femme (épouse)

La nourriture (la comida) :

  • El desayuno → le petit-déjeuner
  • El almuerzo / la comida → le déjeuner
  • La cena → le dîner
  • El pan → le pain
  • La carne → la viande
  • El pescado → le poisson

Correspondances régulières :
Français -tion → Espagnol -ción : nation → nación, situation → situación
Français -té → Espagnol -dad : liberté → libertad, université → universidad
Français -ment → Espagnol -mente : rapidement → rápidamente

Les nombres et le temps

Les nombres :

  • 1-10 : uno, dos, tres, cuatro, cinco, seis, siete, ocho, nueve, diez
  • 20 : veinte / 100 : cien / 1000 : mil

Les jours de la semaine :

  • lunes, martes, miércoles, jueves, viernes, sábado, domingo
  • En espagnol, les jours ne prennent pas de majuscule.

Les couleurs et les descriptions

Les couleurs (los colores) :

  • Rojo/a → rouge | Azul → bleu | Verde → vert
  • Amarillo/a → jaune | Blanco/a → blanc | Negro/a → noir
  • Marrón → marron | Gris → gris | Rosa → rose

Les descriptions physiques :

  • Alto/a → grand(e) | Bajo/a → petit(e)
  • Delgado/a → mince | Gordo/a → gros(se)
  • Rubio/a → blond(e) | Moreno/a → brun(e)
  • Guapo/a → beau/belle | Joven → jeune | Mayor → âgé(e)

Pièges courants du vocabulaire espagnol

  • Ser caliente vs Estar caliente : Ser caliente a une connotation sexuelle ! Pour dire « être chaud » (température), utilisez estar caliente. Pour le temps, dites Hace calor (il fait chaud).
  • Coger : En Espagne, coger signifie « prendre ». En Amérique latine, ce mot est vulgaire ! Utilisez tomar ou agarrar à la place.
  • Esperar : Ce verbe signifie à la fois « attendre » ET « espérer ». Le contexte détermine le sens.
  • Realizar vs Darse cuenta : Realizar signifie « réaliser un projet ». Pour « se rendre compte », on dit darse cuenta de.

Astuce : Les mots espagnols terminés en -ción sont quasi toujours féminins et correspondent au français -tion : la nación, la situación, la comunicación. C'est un raccourci précieux pour deviner le genre.

Quiz

Testez votre compréhension

Question 1 / 6

Que signifie « embarazada » ?

« Embarazada » signifie « enceinte ». C'est l'un des faux amis les plus célèbres entre le français et l'espagnol. Pour dire « embarrassée », on dit « avergonzada ».

Question 2 / 6

Comment dit-on « le dîner » en espagnol ?

« La cena » = le dîner. « El desayuno » = le petit-déjeuner. « El almuerzo / la comida » = le déjeuner.

Question 3 / 6

Le français « liberté » se traduit en espagnol par :

Le suffixe français « -té » correspond au suffixe espagnol « -dad » : liberté → libertad, université → universidad, société → sociedad.

Question 4 / 6

Que signifie le faux ami « constipado » ?

« Constipado » signifie « enrhumé », pas « constipé ». Pour dire « constipé » en espagnol, on utilise « estreñido ». C'est un faux ami classique qui peut provoquer des malentendus embarrassants !

Question 5 / 6

Comment dit-on « blond » en espagnol ?

« Rubio/rubia » signifie blond(e). « Moreno/morena » signifie brun(e). Ce sont des mots très courants pour les descriptions physiques en espagnol.

Question 6 / 6

Le verbe « esperar » signifie :

Le verbe « esperar » a deux sens : « attendre » et « espérer ». C'est le contexte qui détermine le sens. « Espero el autobús » = j'attends le bus. « Espero que vengas » = j'espère que tu viendras.

Exercices

Mettez en pratique

Exercice 1 — Faux amis franco-espagnols

Traduisez en espagnol en évitant les faux amis :

1. Elle est embarrassée. → ___
2. Ce manteau est très large. → ___
3. C'est un grand succès. → ___
4. Il est enrhumé aujourd'hui. → ___

Voir la solution

1. Ella está avergonzada. → Pas « embarazada » (= enceinte) !
2. Este abrigo es muy ancho. → Pas « largo » (= long) !
3. Es un gran éxito. → « Éxito » = succès (pas sortie).
4. Hoy está constipado. → « Constipado » = enrhumé en espagnol.

Exercice 2 — Correspondances de suffixes

Trouvez le mot espagnol correspondant en appliquant la règle des suffixes :

1. Situation (fr. -tion → esp. -ción) → ___
2. Université (fr. -té → esp. -dad) → ___
3. Rapidement (fr. -ment → esp. -mente) → ___
4. Possible (fr. -ble → esp. -ble) → ___

Voir la solution

1. Situation → situación
2. Université → universidad
3. Rapidement → rápidamente
4. Possible → posible

Exercice 3 — Décrivez la personne

Traduisez cette description en espagnol :

« Mon frère est grand, blond et mince. Il est jeune et très beau. »

Voir la solution

Mi hermano es alto, rubio y delgado. Es joven y muy guapo.

Notez l'utilisation de ser pour toutes ces caractéristiques permanentes.
