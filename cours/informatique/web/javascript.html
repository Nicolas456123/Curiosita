Introduction

Le langage universel du web

JavaScript est le seul langage de programmation exécuté nativement par tous les navigateurs web. Initialement conçu en 1995 pour ajouter de l'interactivité aux pages HTML, il est devenu un langage complet utilisé aussi bien côté client (navigateur) que côté serveur (Node.js). C'est un langage interprété, dynamiquement typé, multi-paradigme (impératif, fonctionnel, orienté objet par prototypes).

ECMAScript : JavaScript suit la spécification ECMAScript. Depuis ES6 (2015), le langage évolue chaque année avec de nouvelles fonctionnalités : let/const, fonctions fléchées, déstructuration, modules, async/await, etc.

Comprendre JavaScript est indispensable pour le développement web moderne. Il permet de manipuler le DOM, de réagir aux actions de l'utilisateur, de communiquer avec des API distantes et de construire des applications complètes dans le navigateur.

Le cours

Fondamentaux et concepts avancés

Variables et types de données

JavaScript dispose de trois façons de déclarer des variables : var (portée fonction, évité aujourd'hui), let (portée bloc, réassignable) et const (portée bloc, non réassignable). Les types primitifs sont : string, number, boolean, null, undefined, symbol et bigint.

JavaScript
const nom = "Alice";        // string, non réassignable
let age = 25;               // number, réassignable
let estEtudiant = true;     // boolean
const notes = [18, 15, 20]; // tableau (objet)
const user = {              // objet littéral
  nom: "Alice",
  age: 25
};

// typeof renvoie le type d'une valeur
typeof nom;        // "string"
typeof notes;      // "object" (les tableaux sont des objets)
typeof null;       // "object" (bug historique du langage)

Fonctions et fonctions fléchées

Les fonctions sont des citoyens de première classe en JavaScript : elles peuvent être assignées à des variables, passées en argument et retournées. ES6 introduit les fonctions fléchées (=>) qui offrent une syntaxe concise et ne possèdent pas leur propre this.

JavaScript
// Déclaration classique
function addition(a, b) {
  return a + b;
}

// Expression de fonction
const soustraction = function(a, b) {
  return a - b;
};

// Fonction fléchée (concise)
const multiplier = (a, b) => a * b;

// Avec déstructuration et valeurs par défaut
const saluer = ({ nom, titre = "M." }) =>
  `Bonjour ${titre} ${nom}`;

saluer({ nom: "Dupont" }); // "Bonjour M. Dupont"

Manipulation du DOM

Le DOM (Document Object Model) est la représentation en arbre du document HTML dans le navigateur. JavaScript peut sélectionner, créer, modifier et supprimer des éléments du DOM de manière dynamique.

JavaScript
// Sélectionner des éléments
const titre = document.querySelector('h1');
const items = document.querySelectorAll('.item');

// Modifier le contenu et le style
titre.textContent = 'Nouveau titre';
titre.style.color = '#2980b9';

// Créer et ajouter un élément
const p = document.createElement('p');
p.textContent = 'Paragraphe ajouté dynamiquement';
p.classList.add('highlight');
document.querySelector('main').appendChild(p);

// Supprimer un élément
titre.remove();

Événements

Les événements permettent à JavaScript de réagir aux actions de l'utilisateur (clic, saisie, défilement, etc.). On attache un écouteur d'événement avec addEventListener, qui est préférable aux attributs HTML (onclick).

JavaScript
const bouton = document.querySelector('#valider');

// Ajouter un écouteur d'événement
bouton.addEventListener('click', (event) => {
  event.preventDefault();   // empêcher le comportement par défaut
  console.log('Bouton cliqué !');
});

// Délégation d'événements (un seul écouteur sur le parent)
document.querySelector('.liste').addEventListener('click', (e) => {
  if (e.target.matches('li')) {
    console.log('Élément :', e.target.textContent);
  }
});

// Événements clavier
document.addEventListener('keydown', (e) => {
  if (e.key === 'Escape') fermerModal();
});

Programmation asynchrone

JavaScript est mono-thread mais gère l'asynchrone grâce à la boucle d'événements (event loop). Les Promises représentent une valeur future, et async/await offre une syntaxe lisible pour les utiliser.

JavaScript
// Fetch API avec Promises
fetch('https://api.example.com/users')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));

// Même chose avec async/await
async function chargerUtilisateurs() {
  try {
    const response = await fetch('https://api.example.com/users');
    const data = await response.json();
    console.log(data);
  } catch (err) {
    console.error('Erreur :', err);
  }
}

// Promise.all pour les requêtes parallèles
const [users, posts] = await Promise.all([
  fetch('/api/users').then(r => r.json()),
  fetch('/api/posts').then(r => r.json())
]);

Déstructuration et opérateur spread

La déstructuration permet d'extraire des valeurs d'objets et de tableaux de manière concise. L'opérateur spread (...) étend un itérable en ses éléments, et le rest collecte les arguments restants.

JavaScript
// Déstructuration d'objet
const { nom, age, ville = "Paris" } = utilisateur;

// Déstructuration de tableau
const [premier, deuxieme, ...reste] = [1, 2, 3, 4, 5];
// premier = 1, deuxieme = 2, reste = [3, 4, 5]

// Spread pour copier / fusionner
const copie = { ...utilisateur, age: 26 };
const tous = [...tableau1, ...tableau2];

// Rest dans les paramètres
function somme(...nombres) {
  return nombres.reduce((acc, n) => acc + n, 0);
}

Modules ES6

Les modules permettent de découper le code en fichiers séparés avec des importations et exportations explicites. Chaque module a sa propre portée, évitant la pollution du scope global.

JavaScript
// utils.js — exportations nommées
export function formater(prix) {
  return `${prix.toFixed(2)} €`;
}
export const TVA = 0.20;

// app.js — importation nommée
import { formater, TVA } from './utils.js';

// Exportation par défaut
export default class Panier { /* ... */ }
import Panier from './Panier.js';

// Import dynamique (chargement à la demande)
const module = await import('./heavy-module.js');

Quiz

Testez votre compréhension

📊 JavaScript — Quiz Score : 0 / 6

Question 1 / 6

Quelle est la différence principale entre let et const ?

const empêche la réassignation de la variable (mais pas la mutation de l'objet référencé). let permet la réassignation. Les deux ont une portée bloc.

Question 2 / 6

Que retourne typeof [] en JavaScript ?

En JavaScript, les tableaux sont des objets. typeof retourne donc "object". Pour vérifier si une valeur est un tableau, on utilise Array.isArray().

Question 3 / 6

Quelle méthode permet de sélectionner un élément par sélecteur CSS dans le DOM ?

querySelector() accepte n'importe quel sélecteur CSS et retourne le premier élément correspondant. querySelectorAll() retourne tous les éléments correspondants sous forme de NodeList.

Question 4 / 6

Que fait le mot-clé await ?

await suspend l'exécution de la fonction async en attendant que la Promise soit résolue ou rejetée. Le reste du programme continue de s'exécuter (la boucle d'événements n'est pas bloquée).

Question 5 / 6

Quelle syntaxe représente une fonction fléchée ?

La syntaxe (x) => x * 2 est une fonction fléchée. Avec un seul paramètre, les parenthèses sont optionnelles : x => x * 2. Avec un corps sur plusieurs lignes, on utilise des accolades et return.

Question 6 / 6

Comment importer une exportation nommée depuis un module ES6 ?

Les exportations nommées s'importent avec des accolades : import { nom } from './module.js'. L'exportation par défaut s'importe sans accolades : import nom from './module.js'.

Exercice 1 — Liste dynamique

Créez une page avec un champ de saisie et un bouton. Quand l'utilisateur clique sur le bouton, le texte saisi est ajouté comme nouvel élément <li> à une liste <ul>. Le champ est vidé après l'ajout.

Voir la solution 
const input = document.querySelector('#saisie');
const btn = document.querySelector('#ajouter');
const liste = document.querySelector('#maListe');

btn.addEventListener('click', () => {
  const texte = input.value.trim();
  if (texte) {
    const li = document.createElement('li');
    li.textContent = texte;
    liste.appendChild(li);
    input.value = '';
    input.focus();
  }
});

Exercice 2 — Compteur avec état

Créez un compteur avec trois boutons : « -1 », « Reset » et « +1 ». Le compteur affiche la valeur actuelle. Utilisez let pour l'état et mettez à jour le DOM à chaque clic.

Voir la solution 
let compteur = 0;
const affichage = document.querySelector('#valeur');

function mettreAJour() {
  affichage.textContent = compteur;
}

document.querySelector('#moins').addEventListener('click', () => {
  compteur--;
  mettreAJour();
});

document.querySelector('#reset').addEventListener('click', () => {
  compteur = 0;
  mettreAJour();
});

document.querySelector('#plus').addEventListener('click', () => {
  compteur++;
  mettreAJour();
});

Exercice 3 — Fetch et affichage

Utilisez fetch pour récupérer les données depuis https://jsonplaceholder.typicode.com/users et affichez le nom et l'email de chaque utilisateur dans une liste HTML. Gérez les erreurs avec try/catch.

Voir la solution 
async function afficherUtilisateurs() {
  const liste = document.querySelector('#users');
  try {
    const response = await fetch('https://jsonplaceholder.typicode.com/users');
    if (!response.ok) throw new Error(`Erreur HTTP ${response.status}`);
    const users = await response.json();
    users.forEach(user => {
      const li = document.createElement('li');
      li.textContent = `${user.name} — ${user.email}`;
      liste.appendChild(li);
    });
  } catch (err) {
    liste.innerHTML = '<li class="error">Impossible de charger les données.</li>';
    console.error(err);
  }
}
afficherUtilisateurs();

Exercice 4 — Filtre de recherche

Créez un champ de recherche qui filtre en temps réel une liste d'éléments. Utilisez l'événement input et la méthode filter() sur un tableau de données.

Voir la solution 
const fruits = ['Pomme', 'Banane', 'Cerise', 'Fraise', 'Mangue', 'Orange'];
const recherche = document.querySelector('#recherche');
const container = document.querySelector('#resultats');

function afficher(items) {
  container.innerHTML = items.map(f => `<li>${f}</li>`).join('');
}

recherche.addEventListener('input', (e) => {
  const filtre = e.target.value.toLowerCase();
  const resultats = fruits.filter(f => f.toLowerCase().includes(filtre));
  afficher(resultats);
});

afficher(fruits); // affichage initial

Exercice 5 — Module de panier

Créez un module panier.js avec les fonctions ajouter(produit), supprimer(id) et total(). Exportez-les avec des exports nommés. Dans app.js, importez le module et testez les fonctions.

Voir la solution 
// panier.js
const articles = [];

export function ajouter(produit) {
  articles.push(produit);
}

export function supprimer(id) {
  const index = articles.findIndex(a => a.id === id);
  if (index !== -1) articles.splice(index, 1);
}

export function total() {
  return articles.reduce((sum, a) => sum + a.prix * a.quantite, 0);
}

export function contenu() {
  return [...articles]; // copie pour éviter la mutation
}

// app.js
import { ajouter, supprimer, total, contenu } from './panier.js';

ajouter({ id: 1, nom: 'Clavier', prix: 49.99, quantite: 1 });
ajouter({ id: 2, nom: 'Souris', prix: 29.99, quantite: 2 });
console.log(total());   // 109.97
supprimer(2);
console.log(contenu()); // [{ id: 1, nom: 'Clavier', ... }]
