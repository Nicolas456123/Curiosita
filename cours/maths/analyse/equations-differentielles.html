01

Introduction

Quand le changement décrit le système

Une équation différentielle est une équation qui relie une fonction inconnue y(x) à ses dérivées y'(x), y''(x), etc. Contrairement aux équations algébriques dont la solution est un nombre, la solution d'une équation différentielle est une fonction.

Idée centrale : les équations différentielles sont le langage naturel des lois de la physique. La deuxième loi de Newton F = ma est une équation différentielle : l'accélération a = y'' est la dérivée seconde de la position y(t). De même, la croissance radioactive, les circuits électriques, la propagation de la chaleur et la dynamique des populations sont tous décrits par des équations différentielles.

Ce cours couvre les types fondamentaux d'équations différentielles ordinaires (EDO), c'est-à-dire celles qui ne font intervenir qu'une seule variable indépendante. Les équations aux dérivées partielles (EDP), qui impliquent plusieurs variables, constituent un sujet plus avancé.

02

Le cours

Types, méthodes de résolution et applications

Vocabulaire et classification

Une équation différentielle ordinaire (EDO) se caractérise par :

  • L'ordre : le plus haut degré de dérivation qui apparaît. y' + y = 0 est d'ordre 1 ; y'' + y = 0 est d'ordre 2
  • Linéarité : l'équation est linéaire si y et ses dérivées n'apparaissent qu'au premier degré (pas de y², y·y', etc.)
  • Coefficients constants : les coefficients devant y, y', y'', ... sont des constantes (pas des fonctions de x)

La solution générale contient des constantes arbitraires (autant que l'ordre de l'EDO). Une condition initiale fixe ces constantes pour donner une solution particulière unique.

Problème de Cauchy : une EDO d'ordre n accompagnée de n conditions initiales y(x₀) = y₀, y'(x₀) = y₁, ..., y(n-1)(x₀) = yn-1. Le théorème de Cauchy-Lipschitz garantit, sous des conditions raisonnables, l'existence et l'unicité de la solution.

Équations du premier ordre à variables séparables

L'EDO y' = f(x)·g(y) se résout en « séparant les variables » :

dy/g(y) = f(x) dx puis on intègre chaque côté

Exemple : résoudre y' = 2xy. On sépare : dy/y = 2x dx. En intégrant : ln|y| = x² + C, d'où y = Kex² (avec K = ±eC). On peut vérifier : y' = 2xKex² = 2xy.

Équations linéaires du premier ordre

L'EDO y' + a(x)y = b(x) se résout en deux étapes :

  • 1. Équation homogène (b(x) = 0) : y' + a(x)y = 0 a pour solution yH = Ce−A(x), où A(x) = ∫a(x) dx
  • 2. Solution particulière par variation de la constante : on cherche yP = C(x)·e−A(x) et on substitue dans l'équation complète

Cas à coefficients constants : y' + ay = b (a, b constants). Solution homogène : yH = Ce−ax. Solution particulière constante : yP = b/a. Solution générale : y = Ce−ax + b/a.

Exemple : résoudre y' + 3y = 6 avec y(0) = 1. Solution générale : y = Ce−3x + 2. Condition initiale : 1 = C + 2, donc C = −1. Solution : y = −e−3x + 2.

Équations linéaires du second ordre à coefficients constants

L'EDO ay'' + by' + cy = 0 se résout grâce à l'équation caractéristique :

ar² + br + c = 0

Le discriminant Δ = b² − 4ac détermine la nature des solutions :

  • Δ > 0 : deux racines réelles distinctes r₁, r₂. Solution : y = C₁er₁x + C₂er₂x
  • Δ = 0 : une racine double r. Solution : y = (C₁ + C₂x)erx
  • Δ < 0 : deux racines complexes conjuguées α ± iβ. Solution : y = eαx(C₁cos(βx) + C₂sin(βx))

Exemple : résoudre y'' − 5y' + 6y = 0. Équation caractéristique : r² − 5r + 6 = 0, Δ = 25 − 24 = 1 > 0. Racines : r₁ = 2, r₂ = 3. Solution : y = C₁e2x + C₂e3x.

Exemple avec racines complexes : y'' + 4y = 0. Équation caractéristique : r² + 4 = 0, r = ±2i (α = 0, β = 2). Solution : y = C₁cos(2x) + C₂sin(2x). C'est l'équation de l'oscillateur harmonique.

Second membre (équation complète)

Pour résoudre ay'' + by' + cy = f(x), on combine :

  • La solution homogène yH (comme ci-dessus, avec f(x) = 0)
  • Une solution particulière yP trouvée par la méthode des coefficients indéterminés ou la variation des constantes

Méthode des coefficients indéterminés : on devine la forme de yP d'après celle de f(x) :

  • Si f(x) = polynôme de degré n → chercher yP = polynôme de degré n
  • Si f(x) = ekx → chercher yP = Aekx (sauf si k est racine de l'équation caractéristique)
  • Si f(x) = cos(ωx) ou sin(ωx) → chercher yP = A·cos(ωx) + B·sin(ωx)

Exemple : résoudre y'' + y = x. Solution homogène : yH = C₁cos(x) + C₂sin(x). On cherche yP = Ax + B. On a yP'' = 0, donc 0 + Ax + B = x, d'où A = 1 et B = 0. Solution générale : y = C₁cos(x) + C₂sin(x) + x.

Applications physiques

Les équations différentielles modélisent de nombreux phénomènes naturels :

  • Croissance/décroissance exponentielle : y' = ky donne y = y₀ekt. Modélise la radioactivité (k < 0), la croissance bactérienne (k > 0), les intérêts composés
  • Oscillateur harmonique : y'' + ω²y = 0 donne y = A·cos(ωt + φ). Modélise le pendule, les ressorts, les circuits LC
  • Oscillateur amorti : y'' + 2λy' + ω²y = 0 ajoute un terme de frottement. Trois régimes : sur-amorti (Δ > 0), critique (Δ = 0), sous-amorti (Δ < 0, oscillations décroissantes)
  • Équation logistique : y' = ry(1 − y/K), modèle de croissance avec capacité limite K (populations, épidémies)
  • Loi de refroidissement de Newton : T' = −k(T − Text), la température converge exponentiellement vers la température ambiante

Pièges courants

  • Oublier la solution homogène : la solution générale est toujours yH + yP. Donner seulement yP est incomplet
  • Oublier les constantes d'intégration : une EDO d'ordre n a n constantes arbitraires dans sa solution générale
  • Résonance : si le second membre f(x) = ekx et k est racine de l'équation caractéristique, il faut multiplier yP par x (ou x² si racine double). Chercher Aekx ne fonctionnera pas
  • Ne pas vérifier la solution : on peut toujours substituer la solution trouvée dans l'EDO pour vérifier qu'elle la satisfait
  • Confondre EDO linéaire et non linéaire : y' = y² n'est pas linéaire et ne se résout pas par les méthodes linéaires. Il faut la traiter comme une équation à variables séparables

Question 1 / 6

Quelle est la solution générale de y' = 3y ?

y' = 3y est une EDO à variables séparables. En séparant : dy/y = 3 dx, d'où ln|y| = 3x + C, donc y = Ce3x (la constante C englobe ±eC). Vérification : (Ce3x)' = 3Ce3x = 3y.

Question 2 / 6

Quelle est l'équation caractéristique de y'' − 4y' + 3y = 0 ?

On remplace y par erx : y'' = r²erx, y' = rerx, y = erx. L'EDO donne (r² − 4r + 3)erx = 0. Comme erx ≠ 0, l'équation caractéristique est r² − 4r + 3 = 0, de racines r = 1 et r = 3.

Question 3 / 6

Que modélise l'équation y'' + ω²y = 0 ?

y'' + ω²y = 0 a pour solutions y = A·cos(ωx) + B·sin(ωx) : ce sont des fonctions sinusoïdales. C'est l'oscillateur harmonique, qui modélise le mouvement d'un pendule, d'un ressort, ou les oscillations dans un circuit LC.

Question 4 / 6

Combien de constantes arbitraires contient la solution générale d'une EDO d'ordre 2 ?

La solution générale d'une EDO d'ordre n contient exactement n constantes arbitraires. Pour une EDO d'ordre 2, il faut 2 conditions initiales (y(x₀) et y'(x₀)) pour déterminer la solution unique.

Question 5 / 6

Si l'équation caractéristique a un discriminant négatif (Δ < 0), la solution contient :

Quand Δ < 0, les racines sont complexes conjuguées α ± iβ. La solution réelle est y = eαx(C₁cos(βx) + C₂sin(βx)). Le facteur exponentiel eαx contrôle l'amplitude (amortissement si α < 0, amplification si α > 0), tandis que cos et sin produisent les oscillations.

Question 6 / 6

Quelle méthode utilise-t-on pour résoudre y' = x/y ?

y' = x/y s'écrit y dy = x dx. En intégrant : y²/2 = x²/2 + C, d'où y² = x² + K. C'est une équation à variables séparables (non linéaire à cause de y au dénominateur). Les méthodes linéaires ne s'appliquent pas ici.

Exercices

Mettez en pratique

Exercice 1 — EDO à variables séparables

Résoudre y' = −2xy avec la condition initiale y(0) = 3.

Voir la solution

On sépare les variables : dy/y = −2x dx. En intégrant : ln|y| = −x² + C, d'où y = Ke−x². Condition initiale : y(0) = K = 3. Solution : y = 3e−x². C'est une courbe en cloche (gaussienne).

Exercice 2 — EDO linéaire du premier ordre

Résoudre y' − 2y = e3x.

Voir la solution

Solution homogène : y' − 2y = 0 donne yH = Ce2x. Solution particulière : on cherche yP = Ae3x. On a yP' = 3Ae3x, donc 3Ae3x − 2Ae3x = e3x, d'où A = 1. Solution générale : y = Ce2x + e3x.

Exercice 3 — EDO du second ordre (racines réelles)

Résoudre y'' + y' − 6y = 0 avec y(0) = 1 et y'(0) = 0.

Voir la solution

Équation caractéristique : r² + r − 6 = 0, Δ = 1 + 24 = 25. Racines : r = (−1 ± 5)/2, donc r₁ = 2 et r₂ = −3. Solution générale : y = C₁e2x + C₂e−3x. Conditions : y(0) = C₁ + C₂ = 1 et y'(0) = 2C₁ − 3C₂ = 0. Du système : C₁ = 3/5, C₂ = 2/5. Solution : y = (3/5)e2x + (2/5)e−3x.

Exercice 4 — EDO du second ordre (racines complexes)

Résoudre y'' + 2y' + 5y = 0.

Voir la solution

Équation caractéristique : r² + 2r + 5 = 0, Δ = 4 − 20 = −16 < 0. Racines complexes : r = (−2 ± 4i)/2 = −1 ± 2i. Donc α = −1 et β = 2. Solution : y = e−x(C₁cos(2x) + C₂sin(2x)). C'est un oscillateur amorti : les oscillations (cos, sin) sont multipliées par e−x qui décroît vers 0.

Exercice 5 — Application : décroissance radioactive

Un échantillon radioactif perd la moitié de sa masse en 5 730 ans (demi-vie du carbone-14). Si la masse initiale est de 100 g, quelle masse reste-t-il après 10 000 ans ?

Voir la solution

Le modèle est m' = −km, de solution m(t) = m₀e−kt. La demi-vie donne m₀/2 = m₀e−5730k, donc k = ln(2)/5730 ≈ 1.21 × 10−4 an−1. Après 10 000 ans : m = 100 × e−10000 × ln(2)/5730 = 100 × 2−10000/5730 = 100 × 2−1.745 ≈ 100 × 0.298 ≈ 29.8 g.
