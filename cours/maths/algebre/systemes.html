01

Introduction

Pourquoi les systèmes d'équations

Un système d'équations est un ensemble de plusieurs équations portant sur les mêmes inconnues. Résoudre un système, c'est trouver les valeurs qui satisfont toutes les équations simultanément. Les systèmes apparaissent naturellement dès qu'un problème fait intervenir plusieurs grandeurs liées entre elles : mélange de solutions chimiques, circuits électriques, équilibres économiques, ou encore trajectoires de mobiles.

Interprétation géométrique : un système de deux équations à deux inconnues correspond à deux droites dans le plan. La solution est leur point d'intersection. Deux droites peuvent se couper (une solution), être parallèles (aucune solution) ou confondues (infinité de solutions).

Ce cours présente les trois méthodes fondamentales de résolution : la substitution, la combinaison linéaire (ou méthode par addition) et le pivot de Gauss. Cette dernière méthode, systématique et puissante, permet de résoudre des systèmes de toute taille et constitue la base de l'algèbre linéaire et du calcul scientifique.

02

Le cours

Méthodes de résolution des systèmes

Méthode par substitution

Le principe est simple : on isole une inconnue dans une équation, puis on remplace cette expression dans l'autre équation. On obtient alors une équation à une seule inconnue.

Exemple : Résoudre le système :

  • x + 2y = 7
  • 3x - y = 7

De la première équation : x = 7 - 2y. On substitue dans la seconde :

  • 3(7 - 2y) - y = 7
  • 21 - 6y - y = 7
  • -7y = -14, donc y = 2
  • x = 7 - 2(2) = 3

Solution : S = {(3, 2)}. Vérification : 3 + 2(2) = 7 et 3(3) - 2 = 7.

Méthode par combinaison linéaire

On multiplie les équations par des coefficients choisis pour éliminer une inconnue lorsqu'on les additionne ou soustrait. Cette méthode est souvent plus rapide que la substitution.

Exemple : Résoudre le système :

  • 2x + 3y = 12 ... (1)
  • 4x - 3y = 6 ... (2)

En additionnant (1) et (2) : 6x = 18, donc x = 3. En reportant : 2(3) + 3y = 12, soit y = 2. Solution : (3, 2).

Le pivot de Gauss

La méthode du pivot de Gauss (ou élimination gaussienne) est la méthode systématique par excellence. Elle fonctionne pour tout système, quelle que soit sa taille. Le principe est de transformer le système en un système triangulaire équivalent, par des opérations élémentaires sur les lignes :

  • L_i ← L_i + λ·L_j : ajouter un multiple d'une ligne à une autre
  • L_i ← λ·L_i : multiplier une ligne par un scalaire non nul
  • L_i ↔ L_j : échanger deux lignes

Exemple : Résoudre le système à 3 inconnues :

  • x + y + z = 6
  • 2x + 3y + z = 14
  • x - y + 2z = 2

Étape 1 : L₂ ← L₂ - 2L₁ et L₃ ← L₃ - L₁. On obtient y - z = 2 et -2y + z = -4.

Étape 2 : L₃ ← L₃ + 2L₂. On obtient -z = 0, donc z = 0.

Remontée : y = 2, puis x = 6 - 2 - 0 = 4. Solution : (4, 2, 0).

Systèmes sans solution ou avec infinité de solutions

Un système est dit incompatible s'il n'admet aucune solution. Cela se produit quand le pivot de Gauss mène à une ligne contradictoire du type « 0 = 5 ». Un système est dit indéterminé s'il admet une infinité de solutions — on obtient alors une ou plusieurs variables libres (paramètres).

Règle générale : un système de n équations à n inconnues admet une solution unique si et seulement si le déterminant de la matrice des coefficients est non nul.

Méthode de Cramer

Pour un système 2×2 : ax + by = e et cx + dy = f, si le déterminant D = ad - bc ≠ 0, alors :

  • x = (ed - bf) / D
  • y = (af - ec) / D

Cette méthode est élégante pour les petits systèmes mais peu efficace pour les grands, où le pivot de Gauss est préféré.

Systèmes non linéaires

Un système non linéaire contient au moins une équation qui n'est pas du premier degré. La méthode de substitution reste souvent applicable :

Exemple : Résoudre le système : x + y = 5 et x² + y² = 13.

  • De la première équation : y = 5 - x
  • On substitue : x² + (5 - x)² = 13
  • x² + 25 - 10x + x² = 13, soit 2x² - 10x + 12 = 0, soit x² - 5x + 6 = 0
  • Δ = 25 - 24 = 1. x₁ = 2, x₂ = 3
  • Solutions : (2, 3) et (3, 2)

Interprétation matricielle

Tout système linéaire s'écrit sous forme matricielle AX = B, où A est la matrice des coefficients, X le vecteur des inconnues, et B le vecteur des seconds membres. Cette écriture est fondamentale car elle permet d'utiliser les outils de l'algèbre linéaire (déterminant, inverse, rang) pour étudier le système.

Théorème de Rouché-Capelli : un système AX = B admet des solutions si et seulement si le rang de la matrice augmentée [A|B] est égal au rang de A. Si ce rang égale le nombre d'inconnues, la solution est unique.

Pièges courants

Les erreurs fréquentes à éviter lors de la résolution de systèmes :

  • Oublier de substituer dans TOUTES les occurrences : en méthode de substitution, chaque occurrence de la variable doit être remplacée.
  • Erreur de signe dans la combinaison : quand on soustrait une équation, il faut changer TOUS les signes de la ligne soustraite, pas seulement le premier terme.
  • Conclure trop vite à l'absence de solution : obtenir 0 = 0 signifie une infinité de solutions (système indéterminé), pas une erreur.
  • Oublier la vérification : toujours substituer la solution dans les équations d'origine pour vérifier.
  • Confondre PGCD et solution : la méthode de Cramer nécessite un déterminant non nul. Si D = 0, le système est soit incompatible, soit indéterminé.

Applications

Les systèmes d'équations modélisent de nombreuses situations concrètes :

  • Mélange : un chimiste mélange deux solutions à 30% et 70% pour obtenir 100 mL à 45%. Si x et y sont les volumes : x + y = 100 et 0,3x + 0,7y = 45. Solution : x = 62,5 mL et y = 37,5 mL.
  • Circuits électriques : les lois de Kirchhoff sur les courants et tensions donnent des systèmes linéaires dont les inconnues sont les intensités dans chaque branche.
  • Économie : l'équilibre offre-demande se modélise par un système : prix et quantité d'équilibre sont les solutions du système formé par la courbe d'offre et la courbe de demande.

Question 1 / 6

Quelle est la solution du système : x + y = 5 et x - y = 1 ?

En additionnant : 2x = 6, donc x = 3. Puis y = 5 - 3 = 2. Vérification : 3 + 2 = 5 et 3 - 2 = 1.

Question 2 / 6

Le système 2x + 4y = 6 et x + 2y = 5 est...

La 1re équation donne x + 2y = 3 (en divisant par 2), mais la 2e dit x + 2y = 5. On obtient 3 = 5, contradiction. Le système est incompatible.

Question 3 / 6

Quelle opération élémentaire n'est PAS utilisée dans le pivot de Gauss ?

Les opérations élémentaires sont : addition d'un multiple, échange de lignes, et multiplication par un scalaire non nul. Élever au carré n'est pas une opération élémentaire car elle ne préserve pas l'équivalence du système.

Question 4 / 6

Par la méthode de Cramer, résoudre : 3x + 2y = 8 et x - y = 1. Que vaut x ?

D = 3×(-1) - 2×1 = -5. Dx = 8×(-1) - 2×1 = -10. x = Dx/D = -10/-5 = 2. Vérification : 3(2) + 2(1) = 8 et 2 - 1 = 1.

Question 5 / 6

Le système x + y = 3 et 2x + 2y = 6 est...

La 2e équation est le double de la 1re (2x + 2y = 2(x+y) = 6). Les deux équations disent la même chose : y = 3 - x. Il y a une infinité de solutions paramétrées par x.

Question 6 / 6

On mélange du café à 6 euros/kg et du café à 10 euros/kg pour obtenir 5 kg à 7,20 euros/kg. Quelle quantité de café à 6 euros ?

Soit x le café à 6 euros et y à 10 euros. x + y = 5 et 6x + 10y = 36. De la 1re : y = 5 - x. Substitution : 6x + 10(5-x) = 36, soit -4x = -14, x = 3,5 kg.

Exercices

Mettez en pratique

Exercice 1 — Substitution

Résoudre par substitution : 3x - 2y = 7 et x + 4y = -1.

Voir la solution

De la 2e équation : x = -1 - 4y. On substitue dans la 1re : 3(-1 - 4y) - 2y = 7, soit -3 - 12y - 2y = 7, donc -14y = 10, y = -5/7. Puis x = -1 - 4(-5/7) = -1 + 20/7 = 13/7. S = {(13/7, -5/7)}.

Exercice 2 — Combinaison linéaire

Résoudre par combinaison : 5x + 3y = 11 et 2x - 5y = -12.

Voir la solution

On multiplie la 1re par 5 et la 2e par 3 : 25x + 15y = 55 et 6x - 15y = -36. En additionnant : 31x = 19, donc x = 19/31. On reporte : 5(19/31) + 3y = 11, soit 95/31 + 3y = 341/31, donc 3y = 246/31, y = 82/31.

Exercice 3 — Système 3×3

Résoudre par le pivot de Gauss : x + 2y - z = 3, 2x - y + z = 5, 3x + y + 2z = 12.

Voir la solution

L₂ ← L₂ - 2L₁ : -5y + 3z = -1. L₃ ← L₃ - 3L₁ : -5y + 5z = 3. L₃ ← L₃ - L₂ : 2z = 4, donc z = 2. De L₂ : -5y + 6 = -1, donc y = 7/5. De L₁ : x + 14/5 - 2 = 3, donc x = 3 - 4/5 = 11/5. Solution : (11/5, 7/5, 2).

Exercice 4 — Problème concret

Un théâtre propose deux tarifs : adulte et enfant. Une famille de 2 adultes et 3 enfants paie 44 euros. Une autre famille de 3 adultes et 2 enfants paie 51 euros. Trouver le prix de chaque billet.

Voir la solution

Soit a le prix adulte et e le prix enfant. 2a + 3e = 44 et 3a + 2e = 51. Combinaison : 3×(1re) - 2×(2e) donne 5e = 132 - 102 = 30, soit e = 6 euros. Puis 2a + 18 = 44, soit a = 13 euros. Vérification : 3(13) + 2(6) = 39 + 12 = 51.

Exercice 5 — Système non linéaire

Résoudre le système : x × y = 12 et x + y = 7.

Voir la solution

De x + y = 7 : y = 7 - x. Substitution : x(7 - x) = 12, soit -x² + 7x - 12 = 0, soit x² - 7x + 12 = 0. Δ = 49 - 48 = 1. x₁ = (7-1)/2 = 3, x₂ = (7+1)/2 = 4. Les solutions sont (3, 4) et (4, 3). Note : x et y sont les racines de t² - 7t + 12 = 0 (lien avec Viète).
