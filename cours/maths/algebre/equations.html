Introduction

Pourquoi les équations sont fondamentales

Une équation est une égalité contenant une ou plusieurs inconnues. Résoudre une équation, c'est déterminer toutes les valeurs de l'inconnue qui rendent cette égalité vraie. Les équations sont présentes partout : calcul de distances, prévisions financières, modélisation physique.

Notation : On note généralement l'inconnue x. L'ensemble des solutions est noté S. Par exemple, S = {2, 3} signifie que x = 2 et x = 3 sont les deux solutions.

Ce cours couvre les deux types d'équations les plus fondamentaux : les équations du premier degré (ax + b = 0) et du second degré (ax² + bx + c = 0). Maîtriser ces deux types est indispensable avant d'aborder tout sujet avancé en mathématiques.

Le cours

Équations du premier et second degré

Équations du premier degré

Une équation du premier degré a la forme ax + b = 0, où a ≠ 0. Elle a toujours une solution unique :

Solution : x = -b/a

Le principe de résolution repose sur les opérations élémentaires qui préservent l'égalité : on peut ajouter, soustraire, multiplier ou diviser les deux membres par un même nombre (non nul pour la division).

Exemple : Résoudre 3x - 7 = 2x + 5

  • On regroupe les termes en x à gauche : 3x - 2x = 5 + 7
  • On simplifie : x = 12
  • Vérification : 3(12) - 7 = 29 et 2(12) + 5 = 29 ✓

Équations du second degré

Une équation du second degré a la forme ax² + bx + c = 0, où a ≠ 0. La méthode de résolution repose sur le discriminant :

Discriminant : Δ = b² - 4ac

Le signe du discriminant détermine le nombre de solutions réelles :

  • Δ > 0 → deux solutions distinctes : x₁ = (-b - √Δ) / 2a et x₂ = (-b + √Δ) / 2a
  • Δ = 0 → une solution double : x₀ = -b / 2a
  • Δ < 0 → aucune solution réelle (les solutions sont complexes)

Exemple : Résoudre x² - 5x + 6 = 0

  • a = 1, b = -5, c = 6
  • Δ = (-5)² - 4(1)(6) = 25 - 24 = 1 > 0
  • x₁ = (5 - 1) / 2 = 2 et x₂ = (5 + 1) / 2 = 3
  • S = {2, 3}

Factorisation et identités remarquables

Quand c'est possible, la factorisation est souvent plus rapide que le discriminant. L'équation x² - 5x + 6 = 0 se factorise en (x - 2)(x - 3) = 0, d'où x = 2 ou x = 3 immédiatement.

Les trois identités remarquables sont essentielles :

  • (a + b)² = a² + 2ab + b²
  • (a - b)² = a² - 2ab + b²
  • a² - b² = (a + b)(a - b)

Relations entre coefficients et racines

Si x₁ et x₂ sont les racines de ax² + bx + c = 0, alors :

Formules de Viète : x₁ + x₂ = -b/a et x₁ · x₂ = c/a

Ces relations permettent de vérifier rapidement ses résultats ou de construire une équation à partir de ses racines.

Forme canonique et sommet de la parabole

Tout trinôme ax² + bx + c peut s'écrire sous forme canonique :

ax² + bx + c = a(x - α)² + β avec α = -b/(2a) et β = c - b²/(4a) = -Δ/(4a).

Le point (α, β) est le sommet de la parabole. Si a > 0, le sommet est un minimum ; si a < 0, c'est un maximum. La droite x = α est l'axe de symétrie de la parabole.

Exemple : Mettre f(x) = 2x² - 12x + 22 sous forme canonique.

  • α = -(-12)/(2×2) = 12/4 = 3
  • β = 22 - (-12)²/(4×2) = 22 - 144/8 = 22 - 18 = 4
  • f(x) = 2(x - 3)² + 4
  • Le sommet est (3, 4) et c'est un minimum car a = 2 > 0.

Équations se ramenant au second degré

Certaines équations ne sont pas directement du second degré mais s'y ramènent par un changement de variable :

Exemple 1 — Équation bicarrée : Résoudre x⁴ - 5x² + 4 = 0.

  • On pose X = x², d'où X² - 5X + 4 = 0
  • Δ = 25 - 16 = 9, X₁ = (5-3)/2 = 1, X₂ = (5+3)/2 = 4
  • x² = 1 donne x = ±1 ; x² = 4 donne x = ±2
  • S = {-2, -1, 1, 2}

Exemple 2 — Équation avec radicaux : Résoudre √(2x + 3) = x - 1.

  • Condition : 2x + 3 ≥ 0 et x - 1 ≥ 0, donc x ≥ 1
  • On élève au carré : 2x + 3 = (x - 1)² = x² - 2x + 1
  • x² - 4x - 2 = 0, Δ = 16 + 8 = 24, x = (4 ± 2√6)/2 = 2 ± √6
  • x = 2 + √6 ≈ 4,45 (valide car ≥ 1) ; x = 2 - √6 ≈ -0,45 (rejeté car < 1)
  • S = {2 + √6}

Signe du trinôme

Le signe de f(x) = ax² + bx + c dépend du signe de a et du discriminant :

  • Δ < 0 : f(x) est du signe de a pour tout x (pas de racine réelle)
  • Δ = 0 : f(x) est du signe de a pour tout x, sauf en x₀ = -b/(2a) où f(x₀) = 0
  • Δ > 0 : f(x) est du signe de a à l'extérieur des racines, du signe opposé entre les racines

Règle mnémotechnique : « du signe de a à l'extérieur des racines ». Pour résoudre ax² + bx + c ≥ 0 avec a > 0 et deux racines x₁ < x₂ : la solution est x ≤ x₁ ou x ≥ x₂.

Pièges courants

Les erreurs les plus fréquentes à éviter :

  • Oublier le cas a = 0 : si a = 0 dans ax² + bx + c = 0, ce n'est plus une équation du second degré mais du premier degré.
  • Erreur de signe dans le discriminant : dans Δ = b² - 4ac, attention au signe de b. Si b = -5, alors b² = 25 (pas -25).
  • Diviser par x : dans x² = 5x, ne jamais diviser par x (on perdrait la solution x = 0). Il faut factoriser : x(x - 5) = 0.
  • Confusion racine carrée : √Δ est toujours positif. Les deux solutions viennent du ± devant √Δ.
  • Oublier de vérifier : surtout quand on a élevé au carré (risque de solutions parasites) ou quand il y a des dénominateurs (risque de division par zéro).

Applications

Les équations du premier et du second degré sont omniprésentes :

  • Physique : la trajectoire d'un projectile suit y = -g/(2v₀²)·x² + tan(α)·x + h₀ — c'est un trinôme en x. Trouver la portée revient à résoudre cette équation du second degré.
  • Économie : le profit P(x) = -2x² + 100x - 800 d'une entreprise selon le prix x se maximise au sommet de la parabole : x = 100/4 = 25 euros.
  • Géométrie : trouver les dimensions d'un rectangle de périmètre 20 et d'aire 24 mène au système 2(l+L) = 20 et l×L = 24, soit l² - 10l + 24 = 0, d'où l = 4 ou l = 6.

Question 1 / 6

Quelle est la solution de 4x + 8 = 0 ?

x = -b/a = -8/4 = -2. Vérification : 4(-2) + 8 = -8 + 8 = 0 ✓

Question 2 / 6

Quel est le discriminant de 2x² - 4x + 2 = 0 ?

Δ = (-4)² - 4(2)(2) = 16 - 16 = 0. L'équation a une solution double : x₀ = 4/4 = 1.

Question 3 / 6

Si x₁ et x₂ sont les racines de x² - 7x + 12 = 0, combien vaut x₁ · x₂ ?

Par les formules de Viète : x₁ · x₂ = c/a = 12/1 = 12. Les racines sont 3 et 4, et bien 3 × 4 = 12.

Question 4 / 6

Quelle est la forme canonique de f(x) = x² - 6x + 11 ?

α = 6/(2×1) = 3 et β = 11 - 36/4 = 11 - 9 = 2. Donc f(x) = (x - 3)² + 2. Le sommet est (3, 2).

Question 5 / 6

L'équation x² + 4x + 5 = 0 admet...

Δ = 16 - 20 = -4 < 0. Le discriminant est négatif, donc l'équation n'admet aucune solution réelle (mais deux solutions complexes conjuguées).

Question 6 / 6

Résoudre x² = 3x. Les solutions sont :

On factorise : x² - 3x = 0, soit x(x - 3) = 0. Donc x = 0 ou x = 3. Attention : diviser par x ferait perdre la solution x = 0.

Exercices

Mettez en pratique

Exercice 1 — Équation du premier degré

Résoudre l'équation : 5(2x - 3) - 3(x + 1) = 2x + 4.

Voir la solution

On développe : 10x - 15 - 3x - 3 = 2x + 4, soit 7x - 18 = 2x + 4. On regroupe : 5x = 22, donc x = 22/5. Vérification : 5(44/5 - 3) - 3(22/5 + 1) = 5(29/5) - 3(27/5) = 29 - 81/5 = (145-81)/5 = 64/5. Et 2(22/5) + 4 = 44/5 + 20/5 = 64/5.

Exercice 2 — Discriminant et solutions

Résoudre l'équation : 3x² - 10x + 3 = 0.

Voir la solution

a = 3, b = -10, c = 3. Δ = 100 - 36 = 64 > 0. √Δ = 8. x₁ = (10 - 8)/6 = 2/6 = 1/3. x₂ = (10 + 8)/6 = 18/6 = 3. Vérification par Viète : x₁ + x₂ = 1/3 + 3 = 10/3 = -b/a et x₁·x₂ = 1/3 × 3 = 1 = c/a.

Exercice 3 — Forme canonique

Écrire f(x) = -x² + 4x - 1 sous forme canonique et en déduire le maximum de f.

Voir la solution

α = -4/(2×(-1)) = 2. β = -1 - 16/(4×(-1)) = -1 + 4 = 3. Donc f(x) = -(x - 2)² + 3. Comme a = -1 < 0, la parabole est tournée vers le bas et le sommet (2, 3) est un maximum. Le maximum de f est 3, atteint en x = 2.

Exercice 4 — Problème concret

Un jardinier dispose de 30 mètres de clôture pour entourer un potager rectangulaire le long d'un mur (le mur forme un des grands côtés, il n'a besoin que de 3 côtés). Quelles dimensions maximisent l'aire ?

Voir la solution

Soit x la largeur (deux côtés) et L la longueur (un côté). Contrainte : 2x + L = 30, donc L = 30 - 2x. Aire : A(x) = x × L = x(30 - 2x) = -2x² + 30x. C'est un trinôme avec a = -2 < 0 (maximum). Sommet : x = -30/(2×(-2)) = 7,5 m. L = 30 - 15 = 15 m. Aire maximale : 7,5 × 15 = 112,5 m².

Exercice 5 — Équation bicarrée

Résoudre x⁴ - 13x² + 36 = 0.

Voir la solution

On pose X = x². L'équation devient X² - 13X + 36 = 0. Δ = 169 - 144 = 25. X₁ = (13 - 5)/2 = 4, X₂ = (13 + 5)/2 = 9. Donc x² = 4 donne x = ±2, et x² = 9 donne x = ±3. S = {-3, -2, 2, 3}.
