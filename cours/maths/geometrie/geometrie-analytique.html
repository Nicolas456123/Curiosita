Introduction

L'algèbre au service de la géométrie

La géométrie analytique est l'étude des figures géométriques à l'aide de coordonnées numériques. Inventée par René Descartes au XVIIe siècle, elle permet de transformer tout problème de géométrie en un problème d'algèbre : un point devient un couple de nombres, une droite devient une équation, un cercle une équation du second degré. Cette idée révolutionnaire a unifié deux branches des mathématiques jusque-là séparées.

Grâce au repère cartésien, on peut calculer des distances, des milieux, des pentes, démontrer que des droites sont parallèles ou perpendiculaires, trouver des intersections, et bien plus encore. C'est le fondement de l'infographie, des systèmes de navigation, et de toute la géométrie computationnelle moderne.

L'idée de Descartes : associer à chaque point du plan un couple de nombres (x, y), ses coordonnées dans un repère. Ce passage du géométrique au numérique permet d'utiliser toute la puissance du calcul algébrique pour résoudre des problèmes de forme et de position.

Ce cours couvre les notions de repère et de coordonnées, les vecteurs du plan, les équations de droites, les distances et milieux, le produit scalaire, et les équations de cercles. Ces outils forment le vocabulaire de base de la géométrie analytique plane.

Le cours

Coordonnées, vecteurs et équations

Le repère cartésien

Un repère cartésien du plan est défini par un point O (l'origine) et deux axes perpendiculaires gradués : l'axe des abscisses (horizontal, Ox) et l'axe des ordonnées (vertical, Oy). Tout point M du plan est repéré par ses coordonnées (x, y).

On distingue :

  • Repère orthonormé (O, i⃗, j⃗) : les vecteurs de base ont même norme (= 1) et sont perpendiculaires. C'est le repère standard.
  • Repère orthogonal : les axes sont perpendiculaires mais les unités peuvent différer.
  • Repère quelconque : les axes ne sont pas nécessairement perpendiculaires.

Distance entre deux points

La distance entre deux points A(x₁, y₁) et B(x₂, y₂) se calcule par le théorème de Pythagore :

AB = √((x₂ - x₁)² + (y₂ - y₁)²)

Exemple : A(1, 2) et B(4, 6). AB = √((4-1)² + (6-2)²) = √(9 + 16) = √25 = 5.

Milieu d'un segment

Le milieu M du segment [AB] a pour coordonnées la moyenne des coordonnées de A et B :

M = ((x₁ + x₂)/2, (y₁ + y₂)/2)

Exemple : milieu de A(3, -1) et B(7, 5) : M = ((3+7)/2, (-1+5)/2) = (5, 2).

Vecteurs du plan

Un vecteur AB⃗ est caractérisé par une direction, un sens et une norme (longueur). Ses coordonnées sont :

AB⃗ = (x₂ - x₁, y₂ - y₁)

Norme : ‖AB⃗‖ = √((x₂ - x₁)² + (y₂ - y₁)²) = AB

Opérations sur les vecteurs :

  • Addition : (a, b) + (c, d) = (a + c, b + d)
  • Multiplication par un scalaire : k × (a, b) = (ka, kb)
  • Colinéarité : u⃗(a, b) et v⃗(c, d) sont colinéaires si et seulement si ad - bc = 0 (le déterminant est nul)

Deux vecteurs colinéaires signifie qu'ils sont parallèles (même direction). Trois points A, B, C sont alignés si et seulement si AB⃗ et AC⃗ sont colinéaires.

Équation de droite

Dans le plan, une droite admet plusieurs formes d'équation :

Forme réduite : y = mx + p (m = pente, p = ordonnée à l'origine)

Forme générale : ax + by + c = 0 (a et b non tous nuls)

Forme paramétrique : (x, y) = (x₀, y₀) + t(a, b), t ∈ ℝ

La pente (coefficient directeur) d'une droite passant par A(x₁, y₁) et B(x₂, y₂) est :

m = (y₂ - y₁) / (x₂ - x₁) (à condition que x₁ ≠ x₂)

Si x₁ = x₂, la droite est verticale, d'équation x = x₁ (pas de pente définie).

Parallélisme et perpendicularité

Deux droites de pentes m₁ et m₂ sont :

Parallèles : m₁ = m₂ (même pente)

Perpendiculaires : m₁ × m₂ = -1 (les pentes sont inverses et opposées)

Exemple : la droite y = 3x + 1 est perpendiculaire à y = -x/3 + 5, car 3 × (-1/3) = -1.

En forme générale, les droites a₁x + b₁y + c₁ = 0 et a₂x + b₂y + c₂ = 0 sont :

  • Parallèles si a₁b₂ - a₂b₁ = 0
  • Perpendiculaires si a₁a₂ + b₁b₂ = 0

Intersection de deux droites

Pour trouver l'intersection de deux droites, on résout le système formé par leurs équations. Trois cas possibles :

  • Un point unique : les droites sont sécantes (cas général)
  • Aucune solution : les droites sont strictement parallèles
  • Infinité de solutions : les droites sont confondues

Exemple : intersection de y = 2x + 1 et y = -x + 7. On pose 2x + 1 = -x + 7, d'où 3x = 6, x = 2, y = 5. Le point d'intersection est (2, 5).

Le produit scalaire

Le produit scalaire de deux vecteurs u⃗(a, b) et v⃗(c, d) est un nombre réel :

u⃗ · v⃗ = ac + bd = ‖u⃗‖ × ‖v⃗‖ × cos(θ)

où θ est l'angle entre les deux vecteurs.

Applications du produit scalaire :

  • Orthogonalité : u⃗ ⊥ v⃗ ⟺ u⃗ · v⃗ = 0
  • Angle entre vecteurs : cos(θ) = (u⃗ · v⃗) / (‖u⃗‖ × ‖v⃗‖)
  • Norme : ‖u⃗‖² = u⃗ · u⃗ = a² + b²
  • Projection : la projection de u⃗ sur v⃗ a pour norme (u⃗ · v⃗) / ‖v⃗‖

Exemple : u⃗ = (3, 4) et v⃗ = (−4, 3). u⃗ · v⃗ = 3×(−4) + 4×3 = −12 + 12 = 0. Donc u⃗ ⊥ v⃗.

Équation du cercle

Un cercle de centre Ω(a, b) et de rayon r a pour équation :

(x - a)² + (y - b)² = r²

Forme développée : x² + y² - 2ax - 2by + (a² + b² - r²) = 0

Exemple : le cercle de centre (3, -2) et de rayon 5 a pour équation (x - 3)² + (y + 2)² = 25, soit en développant : x² + y² - 6x + 4y - 12 = 0.

Reconnaître un cercle : l'équation x² + y² + Dx + Ey + F = 0 représente un cercle de centre (-D/2, -E/2) et de rayon r = √(D²/4 + E²/4 - F), à condition que D²/4 + E²/4 - F > 0.

Distance d'un point à une droite

La distance du point M(x₀, y₀) à la droite d : ax + by + c = 0 est :

d(M, d) = |ax₀ + by₀ + c| / √(a² + b²)

Exemple : distance de M(1, 3) à la droite 3x + 4y - 2 = 0 : d = |3×1 + 4×3 - 2| / √(9 + 16) = |3 + 12 - 2| / 5 = 13/5 = 2,6.

Pièges courants

  • Oublier le cas de la droite verticale : x = constante n'a pas de pente. La formule y = mx + p ne peut pas la représenter.
  • Se tromper dans le signe : le centre du cercle (x - a)² + (y - b)² = r² est (a, b), pas (-a, -b). Attention aux signes dans la forme développée.
  • Confondre vecteur et point : un vecteur est un déplacement (pas d'origine fixe), un point est une position. AB⃗ = B - A, pas A - B.
  • Produit scalaire et produit vectoriel : le produit scalaire donne un nombre. Le produit vectoriel (en 3D) donne un vecteur. Ne pas les confondre.
  • Perpendicularité : m₁ × m₂ = -1, pas +1. Et penser aux cas où l'une des droites est verticale (pente non définie).

Applications

  • GPS et cartographie : tout système de positionnement repose sur des coordonnées. La géométrie analytique est le langage du GPS, des cartes numériques et des SIG (systèmes d'information géographique).
  • Infographie et CAO : les logiciels de dessin (AutoCAD, Blender) représentent les objets par des équations de courbes et de surfaces dans des repères.
  • Physique : la trajectoire d'un projectile est une parabole y = -gx²/(2v₀²cos²(α)) + x·tan(α), une équation de la géométrie analytique.
  • Robotique : le positionnement et la trajectoire des robots s'expriment en coordonnées cartésiennes ou polaires.

Question 1 / 6

Quelle est la distance entre A(1, 3) et B(4, 7) ?

AB = √((4-1)² + (7-3)²) = √(9 + 16) = √25 = 5. On reconnaît le triplet pythagoricien (3, 4, 5).

Question 2 / 6

Deux droites de pentes 2 et m sont perpendiculaires. Que vaut m ?

Pour des droites perpendiculaires : m₁ × m₂ = -1. Donc 2 × m = -1, soit m = -1/2. Les pentes sont inverses et de signes opposés.

Question 3 / 6

Le milieu de A(-2, 6) et B(4, -2) est :

M = ((-2+4)/2, (6+(-2))/2) = (2/2, 4/2) = (1, 2). Le milieu est la moyenne des coordonnées.

Question 4 / 6

L'équation (x - 2)² + (y + 3)² = 16 représente un cercle de centre :

(x - 2)² + (y - (-3))² = 16. Le centre est (2, -3) et le rayon est √16 = 4. Attention aux signes : (y + 3)² = (y - (-3))².

Question 5 / 6

Les vecteurs u⃗(6, 4) et v⃗(3, 2) sont-ils colinéaires ?

Deux vecteurs (a, b) et (c, d) sont colinéaires si ad - bc = 0. Ici : 6×2 - 4×3 = 12 - 12 = 0. Oui, ils sont colinéaires (v⃗ = u⃗/2).

Question 6 / 6

Le produit scalaire de u⃗(1, 2) et v⃗(3, -1) vaut :

u⃗ · v⃗ = 1×3 + 2×(-1) = 3 - 2 = 1. Le produit scalaire n'est pas nul, donc les vecteurs ne sont pas orthogonaux.

Exercices

Mettez en pratique

Exercice 1 — Alignement de points

Les points A(1, 2), B(3, 5) et C(7, 11) sont-ils alignés ?

Voir la solution

AB⃗ = (3-1, 5-2) = (2, 3). AC⃗ = (7-1, 11-2) = (6, 9). Déterminant : 2×9 - 3×6 = 18 - 18 = 0. Les vecteurs sont colinéaires, donc A, B et C sont alignés. On peut vérifier : AC⃗ = 3 × AB⃗, ce qui signifie que C est sur la droite (AB), au-delà de B.

Exercice 2 — Équation de droite

Déterminer l'équation de la droite passant par A(2, -1) et B(5, 8).

Voir la solution

Pente : m = (8 - (-1))/(5 - 2) = 9/3 = 3. Équation : y - (-1) = 3(x - 2), soit y + 1 = 3x - 6, donc y = 3x - 7. Vérification : pour A(2, -1) : 3×2 - 7 = -1. ✓ Pour B(5, 8) : 3×5 - 7 = 8. ✓

Exercice 3 — Cercle

Déterminer le centre et le rayon du cercle d'équation x² + y² - 4x + 6y - 3 = 0.

Voir la solution

On complète les carrés : (x² - 4x + 4) + (y² + 6y + 9) = 3 + 4 + 9, soit (x - 2)² + (y + 3)² = 16. Le centre est (2, -3) et le rayon est r = √16 = 4.

Exercice 4 — Distance point-droite

Calculer la distance du point P(3, 1) à la droite d : 2x - y + 4 = 0.

Voir la solution

d(P, d) = |2×3 - 1×1 + 4| / √(2² + (-1)²) = |6 - 1 + 4| / √(4 + 1) = |9| / √5 = 9/√5 = 9√5/5 ≈ 4,02.

Exercice 5 — Intersection droite-cercle

Trouver les points d'intersection de la droite y = x + 1 et du cercle x² + y² = 25.

Voir la solution

On substitue y = x + 1 dans x² + y² = 25 : x² + (x + 1)² = 25, soit x² + x² + 2x + 1 = 25, d'où 2x² + 2x - 24 = 0, soit x² + x - 12 = 0. Discriminant : 1 + 48 = 49. x = (-1 + 7)/2 = 3 ou x = (-1 - 7)/2 = -4. Pour x = 3 : y = 4. Pour x = -4 : y = -3. Les points d'intersection sont (3, 4) et (-4, -3). Vérification : 3² + 4² = 25 ✓ et (-4)² + (-3)² = 25 ✓.
