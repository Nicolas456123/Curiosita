Qu'est-ce que Python ?

Résumé et introduction au langage

Python est un langage de programmation interprété, haut niveau et généraliste, créé par Guido van Rossum en 1991. Sa philosophie est simple : un code doit être lisible, explicite et élégant. C'est pourquoi Python utilise l'indentation (les espaces) pour structurer le code plutôt que des accolades ou des mots-clés.

Pourquoi apprendre Python ? C'est le langage n°1 en science des données, IA et automatisation. Il s'apprend rapidement, fonctionne partout, et possède une bibliothèque standard colossale. Des entreprises comme Google, Instagram, Spotify et la NASA l'utilisent quotidiennement.

Python est un langage interprété : le code n'est pas compilé en avance mais exécuté ligne par ligne par un interpréteur. Cela facilite le débogage et l'expérimentation. Il est aussi dynamiquement typé : vous n'avez pas besoin de déclarer le type d'une variable, Python le déduit automatiquement.

L'écosystème Python est l'un des plus riches au monde. PyPI (Python Package Index) recense plus de 450 000 bibliothèques. Parmi les plus célèbres : NumPy pour les calculs numériques, Pandas pour l'analyse de données, Django pour le web, TensorFlow pour le machine learning.

En 2024, Python est classé #1 sur l'index TIOBE, dépassant Java et C. Il est devenu le langage d'entrée dans l'enseignement supérieur dans le monde entier, et la compétence la plus demandée dans les offres d'emploi tech.

03

Concepts fondamentaux

Les briques de base à maîtriser absolument

Variables & Types
Python gère automatiquement les types : int, float, str, bool, list, dict, tuple, set. Une variable se déclare simplement avec =.
Conditions if/elif/else
Structures conditionnelles pour orienter le flux d'exécution selon des conditions booléennes. L'indentation est obligatoire.
Boucles for & while
for itère sur une séquence (liste, chaîne, range…). while s'exécute tant qu'une condition est vraie. break et continue contrôlent le flux.
Fonctions def
Blocs réutilisables de code. Acceptent des paramètres, retournent des valeurs. Les fonctions lambda permettent des fonctions anonymes courtes.
Listes & Compréhensions
Les listes sont mutables et ordonnées. Les list comprehensions [expr for x in liste] permettent de créer des listes de façon élégante et rapide.
Dictionnaires
Structures clé-valeur {clé: valeur}. Très rapides pour chercher des données. Indispensables pour représenter des objets structurés.
Classes & POO
Python supporte la programmation orientée objet. Les classes encapsulent données (attributs) et comportements (méthodes). L'héritage permet la réutilisation.
Modules & Imports
Python s'organise en modules (fichiers .py). import permet d'utiliser du code externe. pip installe les bibliothèques tierces depuis PyPI.
Gestion d'erreurs
try/except pour capturer les exceptions. finally s'exécute toujours. raise permet de lever des erreurs personnalisées. Crucial pour les apps robustes.

Python — Fonctions 
# Fonction simple avec valeur par défaut
def saluer(nom, langue="fr"):
    salutations = {
        "fr": f"Bonjour {nom} !",
        "en": f"Hello {nom}!",
        "es": f"¡Hola {nom}!"
    }
    return salutations.get(langue, "Langue inconnue")

print(saluer("Marie"))        # Bonjour Marie !
print(saluer("Tom", "en"))    # Hello Tom!

# Fonction récursive — Fibonacci
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# Lambda — fonction anonyme courte
doubler = lambda x: x * 2
nombres = [1, 3, 5, 7]
doubles = list(map(doubler, nombres))  # [2, 6, 10, 14]
Python — Classes & POO 
class Animal:
    # Attribut de classe (partagé par toutes les instances)
    espece_count = 0

    def __init__(self, nom, son):
        self.nom = nom
        self.son = son
        Animal.espece_count += 1

    def parler(self):
        return f"{self.nom} dit : {self.son} !"

    def __repr__(self):
        return f"Animal({self.nom})"

# Héritage
class Chien(Animal):
    def __init__(self, nom, race):
        super().__init__(nom, "Woof")
        self.race = race

    def apporter(self):
        return f"{self.nom} apporte la balle !"

rex = Chien("Rex", "Berger")
print(rex.parler())    # Rex dit : Woof !
print(rex.apporter())  # Rex apporte la balle !

[] crée une liste vide. () crée un tuple, {} crée un dictionnaire vide.

"Curiosita" contient exactement 9 caractères : C-u-r-i-o-s-i-t-a. len() compte les caractères d'une chaîne.

Question 4 / 10

En Python, le mot-clé def définit une fonction. La syntaxe complète est def nom(paramètres): suivi du corps indenté.

x = 10, puis x += 5 → x = 15, puis x *= 2 → x = 30. Les opérateurs += et *= modifient la variable en place.

stop

break sort immédiatement de la boucle. continue passe à l'itération suivante sans interrompre la boucle.

Question 7 / 10

Que fait cette list comprehension ?

[x**2 for x in range(5)]
[1, 4, 9, 16, 25]
[0, 1, 4, 9, 16]
[0, 1, 2, 3, 4]
[1, 2, 4, 8, 16]
range(5) génère 0,1,2,3,4. En élevant chacun au carré : 0²=0, 1²=1, 2²=4, 3²=9, 4²=16. Résultat : [0, 1, 4, 9, 16].

Question 8 / 10

Comment accède-t-on à la valeur "Python" dans ce dictionnaire ?

d[0]
d.langage
d["langage"]
d.get[0]
Dans un dictionnaire Python, on accède aux valeurs avec la clé entre crochets : d["langage"]. On peut aussi utiliser d.get("langage") qui retourne None si la clé n'existe pas.

Question 9 / 10

Quel est le rôle de self dans une méthode de classe ?

Un mot-clé réservé qui s'appelle obligatoirement "self"

La classe elle-même
Le module parent
self représente l'instance de la classe. Il permet d'accéder aux attributs et méthodes de l'objet. C'est une convention — on pourrait l'appeler autrement, mais "self" est universel en Python.

Question 10 / 10

Python utilise try/except (et optionnellement else et finally) pour la gestion d'exceptions. C'est l'équivalent de try/catch dans d'autres langages.
