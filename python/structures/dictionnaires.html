01

Introduction

Des données nommées et organisées

Un dictionnaire est une collection de paires clé : valeur. Contrairement aux listes qui utilisent des indices numériques, les dictionnaires utilisent des clés nommées pour accéder aux données. Ils sont extrêmement rapides pour la recherche et omniprésents dans le code Python réel.

Analogie : un dictionnaire Python fonctionne comme un vrai dictionnaire — vous cherchez un mot (la clé) pour trouver sa définition (la valeur). La recherche est instantanée quel que soit le nombre d'entrées.

02

Le cours

Création, accès, méthodes et itération

Créer et accéder aux données

Python
# Créer un dictionnaire
personne = {
    "nom": "Alice",
    "age": 25,
    "ville": "Paris",
    "langages": ["Python", "JavaScript"]
}

# Accéder aux valeurs
print(personne["nom"])     # "Alice"
print(personne.get("age"))  # 25

# .get() avec valeur par défaut — évite les erreurs
print(personne.get("email", "Non renseigné"))

# Modifier / ajouter
personne["age"] = 26
personne["email"] = "alice@mail.com"

Méthodes et itération

Python
stock = {"pommes": 5, "bananes": 3, "oranges": 8}

# Itérer sur les clés
for fruit in stock:
    print(fruit)  # pommes, bananes, oranges

# Itérer sur les paires clé-valeur
for fruit, quantite in stock.items():
    print(f"{fruit}: {quantite}")

# Méthodes utiles
print(stock.keys())    # dict_keys(['pommes', 'bananes', 'oranges'])
print(stock.values())  # dict_values([5, 3, 8])
print("pommes" in stock) # True

# Supprimer une clé
del stock["bananes"]
prix = stock.pop("oranges")  # Supprime et retourne 8

Dictionnaires imbriqués

Python
# Dictionnaires imbriqués — données structurées
etudiants = {
    "alice": {"notes": [15, 18, 12], "classe": "L3"},
    "bob":   {"notes": [10, 14, 16], "classe": "L2"}
}

# Accès imbriqué
print(etudiants["alice"]["notes"][0])  # 15

# Calculer la moyenne
for nom, infos in etudiants.items():
    moy = sum(infos["notes"]) / len(infos["notes"])
    print(f"{nom}: {moy:.1f}/20")

Méthodes avancées

Les dictionnaires proposent des méthodes utiles pour manipuler les données de manière sûre et efficace.

Python
# .setdefault() — ajouter seulement si la clé n'existe pas
compteur = {}
mots = ["pomme", "banane", "pomme", "cerise", "banane", "pomme"]

for mot in mots:
    compteur[mot] = compteur.get(mot, 0) + 1
print(compteur)  # {'pomme': 3, 'banane': 2, 'cerise': 1}

# .update() — fusionner deux dictionnaires
config = {"theme": "sombre", "langue": "fr"}
nouveaux = {"langue": "en", "taille": 14}
config.update(nouveaux)
print(config)
# {'theme': 'sombre', 'langue': 'en', 'taille': 14}

# Fusion avec | (Python 3.9+)
fusion = config | {"debug": True}

# Copie d'un dictionnaire
copie = config.copy()  # copie superficielle

Dict comprehensions et transformations

Python
# Filtrer un dictionnaire
notes = {"Alice": 15, "Bob": 8, "Clara": 18, "David": 6}
reussis = {nom: note for nom, note in notes.items() if note >= 10}
print(reussis)  # {'Alice': 15, 'Clara': 18}

# Transformer les valeurs
sur_cent = {nom: note * 5 for nom, note in notes.items()}
print(sur_cent)  # {'Alice': 75, 'Bob': 40, 'Clara': 90, 'David': 30}

# Créer un dictionnaire à partir de deux listes
cles = ["nom", "age", "ville"]
valeurs = ["Alice", 25, "Paris"]
personne = dict(zip(cles, valeurs))
print(personne)  # {'nom': 'Alice', 'age': 25, 'ville': 'Paris'}

Pièges courants

Python
# PIÈGE 1 : Accès avec [] sur une clé inexistante
d = {"a": 1}
# d["b"]  → KeyError !
print(d.get("b", 0))  # 0 — pas d'erreur ✓

# PIÈGE 2 : Modifier un dict pendant l'itération
# for k in d:
#     del d[k]  # RuntimeError !
# Solution : itérer sur une copie des clés
for k in list(d.keys()):
    del d[k]  # ✓ car on itère sur une copie

# PIÈGE 3 : Les clés doivent être hashables
# {[1, 2]: "valeur"}  → TypeError ! (liste non hashable)
# {(1, 2): "valeur"}  → OK ✓ (tuple hashable)

# PIÈGE 4 : L'ordre est garanti depuis Python 3.7
# Les dictionnaires conservent l'ordre d'insertion

Bonnes pratiques

  • .get() : utilisez toujours d.get(cle, defaut) au lieu de d[cle] quand la clé pourrait ne pas exister.
  • .items() : pour itérer sur clés et valeurs, utilisez for k, v in d.items() et non for k in d: v = d[k].
  • Comptage : pour compter des occurrences, préférez collections.Counter au lieu d'un dictionnaire manuel.
  • Clés descriptives : utilisez des noms de clés clairs : "nom_complet" plutôt que "n".
  • defaultdict : pour des dictionnaires avec valeurs par défaut, utilisez collections.defaultdict.

Quiz

📚 Dictionnaires — QuizScore : 0 / 6

Question 1 / 6

Que retourne {"a": 1, "b": 2}.get("c", 0) ?

.get(clé, défaut) retourne la valeur par défaut si la clé n'existe pas. Ici, "c" n'existe pas, donc il retourne 0.

Question 2 / 6

Comment itérer sur les clés ET les valeurs d'un dictionnaire ?

.items() retourne des paires (clé, valeur). .keys() retourne seulement les clés, .values() seulement les valeurs.

Question 3 / 6

Que se passe-t-il si on fait d["nouvelle_cle"] = 42 sur un dictionnaire ?

Si la clé n'existe pas, elle est automatiquement créée. Si elle existe déjà, sa valeur est mise à jour. Pas d'erreur dans les deux cas.

Question 4 / 6

Peut-on utiliser une liste comme clé de dictionnaire ?

Les clés doivent être hashables (immuables). Les listes sont mutables, donc interdites. Utilisez un tuple à la place : {(1, 2): "valeur"}.

Question 5 / 6

Que fait d1.update(d2) si les deux dictionnaires partagent des clés ?

.update() ajoute les paires de d2 dans d1. Si une clé existe déjà dans d1, sa valeur est remplacée par celle de d2.

Question 6 / 6

Que retourne dict(zip(["a","b"], [1,2])) ?

zip() combine les deux listes en paires : ("a",1), ("b",2). dict() transforme ces paires en un dictionnaire {"a": 1, "b": 2}.

Exercice 1 — Compteur de mots

Écrivez une fonction qui prend une phrase en entrée et retourne un dictionnaire avec le nombre d'occurrences de chaque mot (en minuscules).

Voir la solution 
def compter_mots(phrase):
    """Compte les occurrences de chaque mot."""
    compteur = {}
    for mot in phrase.lower().split():
        compteur[mot] = compteur.get(mot, 0) + 1
    return compteur

texte = "le chat mange le poisson et le chat dort"
print(compter_mots(texte))
# {'le': 3, 'chat': 2, 'mange': 1, 'poisson': 1, 'et': 1, 'dort': 1}

Exercice 2 — Répertoire téléphonique

Créez un programme de répertoire téléphonique avec les commandes : « ajouter » (nom + numéro), « chercher » (par nom), « supprimer » (par nom), « liste » (afficher tout), « quitter ».

Voir la solution 
repertoire = {}

while True:
    cmd = input("\nCommande (ajouter/chercher/supprimer/liste/quitter) : ").strip()

    if cmd == "ajouter":
        nom = input("Nom : ").strip()
        numero = input("Numéro : ").strip()
        repertoire[nom] = numero
        print(f"{nom} ajouté.")

    elif cmd == "chercher":
        nom = input("Nom : ").strip()
        numero = repertoire.get(nom)
        if numero:
            print(f"{nom} : {numero}")
        else:
            print(f"{nom} non trouvé.")

    elif cmd == "supprimer":
        nom = input("Nom : ").strip()
        if nom in repertoire:
            del repertoire[nom]
            print(f"{nom} supprimé.")
        else:
            print(f"{nom} non trouvé.")

    elif cmd == "liste":
        if repertoire:
            for nom, numero in sorted(repertoire.items()):
                print(f"  {nom}: {numero}")
        else:
            print("Répertoire vide.")

    elif cmd == "quitter":
        break

Exercice 3 — Inverser un dictionnaire

Écrivez une fonction qui inverse un dictionnaire (les clés deviennent les valeurs et inversement). Gérez le cas où plusieurs clés ont la même valeur en regroupant dans une liste.

Voir la solution 
def inverser_dict(d):
    """Inverse un dict. Si des valeurs sont dupliquées, les clés sont groupées."""
    inverse = {}
    for cle, valeur in d.items():
        if valeur in inverse:
            inverse[valeur].append(cle)
        else:
            inverse[valeur] = [cle]
    return inverse

langues = {"Python": "backend", "JavaScript": "frontend",
           "Java": "backend", "CSS": "frontend"}
print(inverser_dict(langues))
# {'backend': ['Python', 'Java'], 'frontend': ['JavaScript', 'CSS']}

Exercice 4 — Bulletin de notes

Créez un dictionnaire de dictionnaires représentant les notes de 3 étudiants dans 3 matières. Calculez et affichez la moyenne par étudiant et par matière.

Voir la solution 
bulletin = {
    "Alice": {"Maths": 16, "Français": 14, "Anglais": 18},
    "Bob":   {"Maths": 12, "Français": 15, "Anglais": 10},
    "Clara": {"Maths": 19, "Français": 13, "Anglais": 17},
}

# Moyenne par étudiant
print("=== Moyennes par étudiant ===")
for nom, notes in bulletin.items():
    moy = sum(notes.values()) / len(notes)
    print(f"{nom}: {moy:.1f}/20")

# Moyenne par matière
print("\n=== Moyennes par matière ===")
matieres = list(bulletin[list(bulletin.keys())[0]].keys())
for matiere in matieres:
    notes_matiere = [bulletin[e][matiere] for e in bulletin]
    moy = sum(notes_matiere) / len(notes_matiere)
    print(f"{matiere}: {moy:.1f}/20")
