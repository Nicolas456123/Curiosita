Introduction

Pourquoi utiliser des fonctions

Une fonction est un bloc de code nommé et réutilisable. Au lieu de copier-coller le même code partout, on le met dans une fonction qu'on appelle autant de fois que nécessaire. Les fonctions rendent le code plus lisible, plus court et plus facile à maintenir.

Principe DRY : « Don't Repeat Yourself » — ne vous répétez pas. Si un bloc de code apparaît plus d'une fois, il devrait probablement être dans une fonction.

Le cours

def, paramètres, return et lambda

Définir et appeler une fonction

Python
# Définir une fonction avec def
def saluer(nom):
    """Affiche un message de bienvenue."""
    print(f"Bonjour {nom} !")

# Appeler la fonction
saluer("Alice")   # Bonjour Alice !
saluer("Bob")     # Bonjour Bob !

# Fonction avec return
def carre(x):
    return x ** 2

resultat = carre(5)
print(resultat)  # 25

Paramètres et valeurs par défaut

Python
# Valeurs par défaut
def presenter(nom, age, ville="Paris"):
    return f"{nom}, {age} ans, habite à {ville}"

print(presenter("Alice", 25))
# Alice, 25 ans, habite à Paris

print(presenter("Bob", 30, "Lyon"))
# Bob, 30 ans, habite à Lyon

# Arguments nommés — l'ordre n'importe plus
print(presenter(age=22, ville="Marseille", nom="Clara"))

*args et **kwargs

Python
# *args — nombre variable d'arguments positionnels
def somme(*nombres):
    total = 0
    for n in nombres:
        total += n
    return total

print(somme(1, 2, 3))       # 6
print(somme(10, 20, 30, 40)) # 100

# **kwargs — arguments nommés variables
def fiche(**infos):
    for cle, valeur in infos.items():
        print(f"{cle}: {valeur}")

fiche(nom="Alice", age=25, langue="Python")
# nom: Alice
# age: 25
# langue: Python

Fonctions lambda

Python
# Lambda — fonction anonyme sur une ligne
doubler = lambda x: x * 2
print(doubler(5))  # 10

# Très utile avec sorted(), map(), filter()
eleves = [("Alice", 15), ("Bob", 12), ("Clara", 18)]
par_note = sorted(eleves, key=lambda e: e[1], reverse=True)
print(par_note)
# [('Clara', 18), ('Alice', 15), ('Bob', 12)]

Portée des variables (scope)

Les variables créées à l'intérieur d'une fonction sont locales : elles n'existent pas en dehors. Les variables définies en dehors sont globales.

Python
# Variable locale — existe uniquement dans la fonction
def ma_fonction():
    x = 10  # locale
    print(x)

ma_fonction()  # 10
# print(x)    # NameError : x n'existe pas ici

# Variable globale — accessible partout (lecture)
compteur = 0

def incrementer():
    global compteur  # nécessaire pour modifier une variable globale
    compteur += 1

incrementer()
print(compteur)  # 1

Évitez global : utiliser global rend le code difficile à comprendre et à déboguer. Préférez passer les données en paramètres et utiliser return.

Retours multiples et docstrings

Python
# Retourner plusieurs valeurs (en réalité un tuple)
def statistiques(nombres):
    """Calcule la somme, la moyenne et le max d'une liste.

    Args:
        nombres: liste de nombres

    Returns:
        tuple: (somme, moyenne, maximum)
    """
    total = sum(nombres)
    moy = total / len(nombres)
    maxi = max(nombres)
    return total, moy, maxi

# Unpacking du résultat
s, m, mx = statistiques([4, 7, 2, 9, 1])
print(f"Somme={s}, Moyenne={m:.1f}, Max={mx}")
# Somme=23, Moyenne=4.6, Max=9

Fonctions comme objets

En Python, les fonctions sont des objets de première classe : on peut les stocker dans des variables, les passer en argument, et les retourner depuis d'autres fonctions.

Python
# Stocker une fonction dans une variable
def crier(texte):
    return texte.upper() + " !"

def chuchoter(texte):
    return texte.lower() + "..."

# Choisir la fonction dynamiquement
parler = crier
print(parler("bonjour"))  # BONJOUR !

# Passer une fonction en argument
def appliquer(func, valeur):
    return func(valeur)

print(appliquer(crier, "python"))      # PYTHON !
print(appliquer(chuchoter, "python"))  # python...

Pièges courants

Python
# PIÈGE 1 : Argument mutable par défaut
def ajouter(element, liste=[]):  # DANGER !
    liste.append(element)
    return liste

print(ajouter(1))  # [1]
print(ajouter(2))  # [1, 2] — la même liste est réutilisée !

# Solution : utiliser None comme valeur par défaut
def ajouter_ok(element, liste=None):
    if liste is None:
        liste = []
    liste.append(element)
    return liste

# PIÈGE 2 : Oublier return
def calculer(x):
    resultat = x * 2
    # Oups ! Pas de return → retourne None

# PIÈGE 3 : Confondre print et return
def doubler_print(x):
    print(x * 2)   # Affiche mais ne retourne rien

def doubler_return(x):
    return x * 2  # Retourne la valeur (réutilisable)

Bonnes pratiques

  • Docstrings : ajoutez une docstring ("""Description""") à chaque fonction pour expliquer ce qu'elle fait.
  • Une seule responsabilité : chaque fonction doit faire une seule chose et la faire bien.
  • Taille raisonnable : si une fonction dépasse 20-30 lignes, envisagez de la découper.
  • Noms explicites : calculer_moyenne() est meilleur que calc().
  • Pas de global : passez les données en paramètres et retournez les résultats.
  • Jamais de mutable par défaut : utilisez None au lieu de [] ou {} comme valeur par défaut.

Exercice 1 — Fonctions mathématiques

Créez trois fonctions : perimetre_cercle(rayon), aire_cercle(rayon) et volume_sphere(rayon). Utilisez pi = 3.14159. Testez avec un rayon de 5.

Voir la solution 
import math

def perimetre_cercle(rayon):
    """Calcule le périmètre d'un cercle."""
    return 2 * math.pi * rayon

def aire_cercle(rayon):
    """Calcule l'aire d'un cercle."""
    return math.pi * rayon ** 2

def volume_sphere(rayon):
    """Calcule le volume d'une sphère."""
    return (4 / 3) * math.pi * rayon ** 3

r = 5
print(f"Rayon = {r}")
print(f"Périmètre : {perimetre_cercle(r):.2f}")
print(f"Aire : {aire_cercle(r):.2f}")
print(f"Volume : {volume_sphere(r):.2f}")

Exercice 2 — Palindrome

Écrivez une fonction est_palindrome(mot) qui retourne True si le mot est un palindrome (se lit pareil à l'endroit et à l'envers), en ignorant les majuscules et les espaces.

Voir la solution 
def est_palindrome(mot):
    """Vérifie si un mot est un palindrome."""
    mot_clean = mot.lower().replace(" ", "")
    return mot_clean == mot_clean[::-1]

# Tests
print(est_palindrome("kayak"))        # True
print(est_palindrome("Radar"))        # True
print(est_palindrome("elle"))         # True
print(est_palindrome("python"))       # False

Exercice 3 — Fonction avec *args

Créez une fonction moyenne(*notes) qui accepte un nombre variable de notes et retourne la moyenne. Si aucune note n'est fournie, retournez 0. Ajoutez un paramètre optionnel arrondi=2 pour le nombre de décimales.

Voir la solution 
def moyenne(*notes, arrondi=2):
    """Calcule la moyenne de notes avec arrondi configurable."""
    if not notes:
        return 0
    return round(sum(notes) / len(notes), arrondi)

print(moyenne(15, 12, 18, 14))        # 14.75
print(moyenne(15, 12, 18, arrondi=0)) # 15.0
print(moyenne())                       # 0

Exercice 4 — Mini-système de commandes

Créez un dictionnaire qui associe des noms de commandes à des fonctions. L'utilisateur tape une commande et le programme exécute la fonction correspondante. Commandes : « bonjour », « heure », « aide », « quitter ».

Voir la solution 
from datetime import datetime

def cmd_bonjour():
    print("Bonjour ! Comment allez-vous ?")

def cmd_heure():
    print(f"Il est {datetime.now().strftime('%H:%M:%S')}")

def cmd_aide():
    print("Commandes : bonjour, heure, aide, quitter")

commandes = {
    "bonjour": cmd_bonjour,
    "heure": cmd_heure,
    "aide": cmd_aide,
}

while True:
    saisie = input("> ").strip().lower()
    if saisie == "quitter":
        print("Au revoir !")
        break
    elif saisie in commandes:
        commandes[saisie]()  # Appeler la fonction
    else:
        print(f"Commande inconnue : {saisie}. Tapez 'aide'.")
