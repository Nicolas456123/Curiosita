Introduction

Prendre des décisions dans le code

Les structures conditionnelles permettent au programme de choisir quel code exécuter selon une condition. Sans elles, un programme exécuterait toujours les mêmes instructions dans le même ordre — aucune intelligence, aucune adaptation.

L'indentation est obligatoire : en Python, le bloc de code à exécuter sous une condition est défini par l'indentation (4 espaces). C'est ce qui remplace les accolades {} utilisées dans d'autres langages.

02

Le cours

if, elif, else et expressions conditionnelles

La structure if

Le mot-clé if exécute un bloc de code uniquement si la condition est True.

Python
# Structure if simple
age = 20

if age >= 18:
    print("Vous êtes majeur")
    print("Vous pouvez voter")

# Ce code s'exécute dans tous les cas (hors du if)
print("Fin du programme")

if / else

Le bloc else s'exécute quand la condition du if est False.

Python
# if / else
temperature = 15

if temperature > 25:
    print("Il fait chaud")
else:
    print("Il ne fait pas chaud")

if / elif / else

Le mot-clé elif (contraction de « else if ») permet de tester plusieurs conditions en cascade.

Python
# if / elif / else — plusieurs conditions
note = 15

if note >= 16:
    mention = "Très bien"
elif note >= 14:
    mention = "Bien"
elif note >= 12:
    mention = "Assez bien"
elif note >= 10:
    mention = "Passable"
else:
    mention = "Insuffisant"

print(f"Note : {note}/20 — Mention : {mention}")
# Note : 15/20 — Mention : Bien

Conditions imbriquées et combinées

On peut imbriquer des conditions ou les combiner avec les opérateurs logiques and, or et not.

Python
# Conditions combinées avec and / or
age = 25
permis = True

if age >= 18 and permis:
    print("Vous pouvez conduire")

# Expression conditionnelle (ternaire)
statut = "majeur" if age >= 18 else "mineur"
print(statut)  # "majeur"

# Test d'appartenance avec in
fruit = "pomme"
if fruit in ["pomme", "poire", "cerise"]:
    print(f"{fruit} est un fruit connu")

Valeurs truthy et falsy

En Python, toute valeur peut être utilisée comme condition. Certaines valeurs sont considérées comme « fausses » (falsy), toutes les autres sont « vraies » (truthy).

Python
# Valeurs falsy : 0, 0.0, "", [], {}, (), None, False
# Tout le reste est truthy

# Utilisation pratique — vérifier si une liste est vide
panier = []
if not panier:
    print("Le panier est vide")

# Vérifier si une chaîne n'est pas vide
nom = input("Votre nom : ")
if nom:
    print(f"Bonjour {nom}")
else:
    print("Vous n'avez rien saisi")

# Vérifier si un résultat existe
resultat = None
if resultat is not None:
    print(f"Résultat : {resultat}")

Astuce : plutôt que if len(liste) > 0, écrivez simplement if liste. C'est plus pythonique et plus lisible.

Le match/case (Python 3.10+)

Depuis Python 3.10, le pattern matching permet de comparer une valeur à plusieurs motifs, similaire au switch d'autres langages.

Python
# match/case — pattern matching (Python 3.10+)
commande = input("Action : ")

match commande:
    case "quitter":
        print("Au revoir !")
    case "aide":
        print("Commandes : quitter, aide, version")
    case "version":
        print("Version 1.0")
    case _:
        print(f"Commande inconnue : {commande}")

# Le _ est le cas par défaut (wildcard)

Pièges courants

Python
# PIÈGE 1 : Oublier les deux-points après if/elif/else
# if x > 5       → SyntaxError !
if x > 5:        # ✓ n'oubliez pas le ':'
    print("ok")

# PIÈGE 2 : Utiliser = au lieu de ==
# if x = 5:      → SyntaxError !
if x == 5:       # ✓ comparaison avec ==
    print("x vaut 5")

# PIÈGE 3 : Indentation incohérente
# Mélanger tabs et espaces cause des erreurs
# Utilisez TOUJOURS 4 espaces

# PIÈGE 4 : Conditions trop complexes
# Mauvais :
# if not (age < 18 or (revenu <= 2000 and not garant)):
# Mieux : décomposer avec des variables
est_majeur = age >= 18
a_revenu_suffisant = revenu > 2000
if est_majeur and (a_revenu_suffisant or garant):
    print("Éligible")

Bonnes pratiques

  • Simplicité : préférez les conditions simples. Si une condition est trop longue, décomposez-la en variables booléennes nommées.
  • Évitez les if inutiles : au lieu de if condition: return True; else: return False, écrivez simplement return condition.
  • Valeurs truthy : utilisez if liste au lieu de if len(liste) > 0 et if nom au lieu de if nom != "".
  • Guard clauses : traitez les cas d'erreur en premier avec des return anticipés pour réduire l'imbrication.

Exercice 1 — Catégorie d'âge

Demandez l'âge de l'utilisateur et affichez sa catégorie : « bébé » (0-2), « enfant » (3-11), « adolescent » (12-17), « adulte » (18-64), « senior » (65+).

Voir la solution 
age = int(input("Votre âge : "))

if age <= 2:
    categorie = "bébé"
elif age <= 11:
    categorie = "enfant"
elif age <= 17:
    categorie = "adolescent"
elif age <= 64:
    categorie = "adulte"
else:
    categorie = "senior"

print(f"Vous êtes un(e) {categorie}.")

Exercice 2 — Calculatrice avec choix

Demandez deux nombres et une opération (+, -, *, /) à l'utilisateur. Effectuez le calcul et affichez le résultat. Gérez le cas de la division par zéro et d'un opérateur inconnu.

Voir la solution 
a = float(input("Premier nombre : "))
b = float(input("Deuxième nombre : "))
op = input("Opération (+, -, *, /) : ")

if op == "+":
    print(f"{a} + {b} = {a + b}")
elif op == "-":
    print(f"{a} - {b} = {a - b}")
elif op == "*":
    print(f"{a} * {b} = {a * b}")
elif op == "/":
    if b == 0:
        print("Erreur : division par zéro !")
    else:
        print(f"{a} / {b} = {a / b}")
else:
    print(f"Opérateur inconnu : {op}")

Exercice 3 — Année bissextile

Demandez une année et déterminez si elle est bissextile. Une année est bissextile si elle est divisible par 4, sauf si elle est divisible par 100, sauf si elle est divisible par 400.

Voir la solution 
annee = int(input("Année : "))

if (annee % 4 == 0 and annee % 100 != 0) or (annee % 400 == 0):
    print(f"{annee} est bissextile")
else:
    print(f"{annee} n'est pas bissextile")

# Tests : 2024 → bissextile, 1900 → non, 2000 → bissextile

Exercice 4 — Tarif de cinéma

Calculez le tarif d'un billet de cinéma selon les règles : moins de 12 ans = 6 euros, étudiant = 8 euros, senior (65+) = 7 euros, tarif normal = 12 euros, séance 3D = +3 euros. Utilisez des variables booléennes.

Voir la solution 
age = int(input("Âge : "))
est_etudiant = input("Étudiant ? (oui/non) : ") == "oui"
est_3d = input("Séance 3D ? (oui/non) : ") == "oui"

if age < 12:
    tarif = 6
elif est_etudiant:
    tarif = 8
elif age >= 65:
    tarif = 7
else:
    tarif = 12

if est_3d:
    tarif += 3

print(f"Tarif : {tarif} euros")
