Introduction

Les outils de calcul et de comparaison

Les opérateurs sont des symboles qui effectuent des opérations sur des valeurs. Python propose plusieurs catégories d'opérateurs : arithmétiques pour les calculs, de comparaison pour tester des conditions, logiques pour combiner des booléens, et d'affectation pour modifier des variables.

Priorité des opérations : comme en mathématiques, Python respecte un ordre de priorité. La multiplication passe avant l'addition, et les parenthèses permettent de forcer l'ordre d'évaluation.

Tous les opérateurs Python

Opérateurs arithmétiques

Les opérateurs de base pour effectuer des calculs mathématiques.

Python
# Opérateurs arithmétiques
a = 17
b = 5

print(a + b)   # Addition : 22
print(a - b)   # Soustraction : 12
print(a * b)   # Multiplication : 85
print(a / b)   # Division : 3.4 (toujours un float)
print(a // b)  # Division entière : 3
print(a % b)   # Modulo (reste) : 2
print(a ** b)  # Puissance : 1419857

# Priorité : ** > * / // % > + -
resultat = 2 + 3 * 4    # 14 (pas 20)
force = (2 + 3) * 4    # 20 (parenthèses forcent)

Opérateurs de comparaison

Ils comparent deux valeurs et retournent un booléen (True ou False).

Python
# Opérateurs de comparaison
x = 10
y = 20

print(x == y)   # Égal à : False
print(x != y)   # Différent de : True
print(x > y)    # Supérieur : False
print(x < y)    # Inférieur : True
print(x >= 10)  # Supérieur ou égal : True
print(x <= 5)   # Inférieur ou égal : False

# Chaînage de comparaisons (unique à Python)
age = 25
print(18 <= age < 65)  # True — élégant !

Opérateurs logiques

Ils combinent des expressions booléennes : and, or et not.

Python
# Opérateurs logiques
age = 25
revenu = 35000

# and — les DEUX conditions doivent être vraies
eligible = age >= 18 and revenu > 30000
print(eligible)  # True

# or — AU MOINS UNE condition doit être vraie
acces = age < 12 or age >= 65
print(acces)  # False (25 n'est ni <12 ni >=65)

# not — inverse le booléen
connecte = True
print(not connecte)  # False

Opérateurs d'affectation composée

Raccourcis pour modifier une variable en combinant une opération et une affectation.

Python
# Affectation composée
x = 10
x += 5    # x = x + 5  → 15
x -= 3    # x = x - 3  → 12
x *= 2    # x = x * 2  → 24
x //= 5   # x = x // 5 → 4
x **= 3   # x = x ** 3 → 64

# Fonctionne aussi avec les chaînes
texte = "Bonjour"
texte += " le monde"  # "Bonjour le monde"

Opérateurs d'identité et d'appartenance

Python propose aussi des opérateurs pour tester l'identité d'un objet (is) et l'appartenance à une collection (in).

Python
# Opérateur 'in' — test d'appartenance
fruits = ["pomme", "banane", "cerise"]
print("pomme" in fruits)      # True
print("mangue" not in fruits) # True

# Fonctionne aussi avec les chaînes
print("Py" in "Python")  # True

# Opérateur 'is' — identité d'objet
a = [1, 2, 3]
b = [1, 2, 3]
c = a

print(a == b)   # True (mêmes valeurs)
print(a is b)   # False (objets différents)
print(a is c)   # True (même objet en mémoire)

is vs == : == compare les valeurs, is compare les identités (même objet en mémoire). Utilisez is uniquement pour None, True et False.

Priorité des opérateurs

Python suit un ordre de priorité strict. Du plus prioritaire au moins prioritaire :

Python
# Ordre de priorité (du plus au moins prioritaire)
# 1. **        (puissance)
# 2. -, +      (unaires : -x, +x)
# 3. *, /, //, %
# 4. +, -      (addition, soustraction)
# 5. ==, !=, >, <, >=, <=, is, in
# 6. not
# 7. and
# 8. or

# Exemples concrets
print(2 ** 3 ** 2)     # 512 (** est associatif à droite : 2 ** 9)
print(-2 ** 2)         # -4 (** avant le - unaire)
print((-2) ** 2)       # 4 (parenthèses forcent l'ordre)

# En cas de doute, utilisez des parenthèses
resultat = (5 + 3) * (10 - 2) / (4 ** 2)  # 4.0

Pièges courants

Python
# PIÈGE 1 : Division entière vs décimale
print(7 / 2)    # 3.5 (division classique → float)
print(7 // 2)   # 3   (division entière → int)

# PIÈGE 2 : Division entière avec négatifs
print(-7 // 2)  # -4 (pas -3 ! arrondi vers le bas)

# PIÈGE 3 : Évaluation court-circuit
# Python arrête d'évaluer dès que le résultat est connu
x = 0
# x != 0 est False, donc le 'and' n'évalue pas 10/x
print(x != 0 and 10 / x > 2)  # False (pas d'erreur)

# PIÈGE 4 : Confondre 'and'/'or' avec '&'/'|'
# 'and'/'or' → logique booléenne
# '&'/'|'    → opérations bit à bit (différent !)

Bonnes pratiques

  • Parenthèses : en cas de doute sur la priorité, ajoutez des parenthèses. Elles rendent le code plus lisible.
  • Chaînage : profitez du chaînage de comparaisons : 0 < x < 100 est plus lisible que x > 0 and x < 100.
  • Court-circuit : utilisez and/or pour des gardes : if x != 0 and 10/x > 2 évite la division par zéro.
  • Espaces : mettez des espaces autour des opérateurs : x = 5 + 3 et non x=5+3.

Exercice 1 — Calculatrice simple

Demandez deux nombres à l'utilisateur et affichez le résultat de chaque opération arithmétique : addition, soustraction, multiplication, division, division entière, modulo et puissance.

Voir la solution 
a = float(input("Premier nombre : "))
b = float(input("Deuxième nombre : "))

print(f"{a} + {b} = {a + b}")
print(f"{a} - {b} = {a - b}")
print(f"{a} * {b} = {a * b}")
print(f"{a} / {b} = {a / b}")
print(f"{a} // {b} = {a // b}")
print(f"{a} % {b} = {a % b}")
print(f"{a} ** {b} = {a ** b}")

Exercice 2 — Pair ou impair

Demandez un nombre entier à l'utilisateur et affichez si ce nombre est pair ou impair en utilisant l'opérateur modulo %.

Voir la solution 
nombre = int(input("Entrez un nombre entier : "))

if nombre % 2 == 0:
    print(f"{nombre} est pair")
else:
    print(f"{nombre} est impair")

Exercice 3 — Décomposition en heures, minutes, secondes

Demandez un nombre de secondes total (ex: 3725) et affichez la décomposition en heures, minutes et secondes en utilisant // et %.

Voir la solution 
total_secondes = int(input("Nombre de secondes : "))

heures = total_secondes // 3600
minutes = (total_secondes % 3600) // 60
secondes = total_secondes % 60

print(f"{total_secondes}s = {heures}h {minutes}min {secondes}s")
# 3725s = 1h 2min 5s

Exercice 4 — Vérification d'éligibilité

Écrivez un programme qui vérifie si une personne est éligible à un prêt. Les conditions sont : avoir au moins 18 ans ET avoir un revenu annuel supérieur à 25000 euros OU avoir un garant. Utilisez les opérateurs logiques.

Voir la solution 
age = int(input("Âge : "))
revenu = float(input("Revenu annuel : "))
garant = input("Avez-vous un garant ? (oui/non) : ") == "oui"

eligible = age >= 18 and (revenu > 25000 or garant)

if eligible:
    print("Vous êtes éligible au prêt.")
else:
    print("Vous n'êtes pas éligible au prêt.")
