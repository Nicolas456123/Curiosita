Introduction

Pourquoi la typographie est essentielle

La typographie est l'art de choisir et d'agencer les caractères pour rendre un texte lisible, compréhensible et visuellement harmonieux. Elle représente environ 95 % de la surface d'un design web ou éditorial : maîtriser la typographie, c'est maîtriser l'essentiel de la communication visuelle.

L'idée fondamentale : la typographie n'est pas décorative — elle est fonctionnelle. Un bon choix typographique guide le regard, établit la hiérarchie de l'information, exprime le ton du message et rend la lecture confortable. Une mauvaise typographie, même avec un excellent contenu, fait fuir le lecteur.

De Gutenberg à la révolution numérique, la typographie a évolué de l'artisanat du plomb fondu aux polices variables et au responsive design. Mais les principes fondamentaux — lisibilité, contraste, rythme — restent inchangés depuis des siècles.

02

Le cours

Les fondamentaux de la typographie

Anatomie d'un caractère

Chaque lettre possède une anatomie précise. Les termes essentiels sont :

  • Fût : trait vertical principal (le trait droit du « d »)
  • Empattement (serif) : petit trait terminal aux extrémités des fûts
  • Hauteur d'x : hauteur des lettres minuscules sans ascendante ni descendante
  • Ascendante : partie d'une lettre qui dépasse la hauteur d'x (b, d, f, h, k, l)
  • Descendante : partie sous la ligne de base (g, j, p, q, y)
  • Oeil : partie fermée ou semi-fermée d'une lettre (le creux du « e »)
  • Graisse : épaisseur du trait (light, regular, bold, black)

Les grandes familles typographiques

La classification Vox-ATypI distingue plusieurs familles, dont les quatre principales :

  • Serif (avec empattements) : Times New Roman, Garamond, Georgia. Associées à la tradition, l'élégance et la lecture longue (livres, journaux).
  • Sans-serif (sans empattements) : Helvetica, Arial, Futura, DM Sans. Modernes, épurées, privilégiées pour les écrans et l'interface.
  • Script : polices imitant l'écriture manuscrite. Décoratives, à utiliser avec parcimonie.
  • Monospace : chaque caractère occupe la même largeur (Courier, JetBrains Mono). Utilisées pour le code informatique.

Règle d'or : ne jamais utiliser plus de 2 à 3 polices dans un même projet. Une pour les titres, une pour le corps de texte, éventuellement une troisième pour les accents ou le code. La cohérence typographique est la marque d'un design professionnel.

Taille, interlignage et mesure

La taille du texte courant se situe généralement entre 16 et 20 px pour le web, et 10-12 pt pour l'impression. L'interlignage (espace entre les lignes de base) doit être de 120 à 150 % de la taille du corps pour assurer une lecture confortable.

La longueur de ligne (mesure) idéale est de 45 à 75 caractères par ligne. Trop court, le texte est saccadé ; trop long, l'oeil peine à revenir au début de la ligne suivante.

Hiérarchie typographique

La hiérarchie guide le lecteur à travers le contenu. Elle s'établit par :

  • La taille : les titres sont plus grands que le corps de texte
  • La graisse : le gras attire l'attention sur les éléments importants
  • La couleur : un accent coloré distingue les liens ou les mots-clés
  • L'espacement : plus d'espace autour d'un titre le sépare visuellement du texte
  • Le style : l'italique souligne une citation ou un terme étranger

Crénage, approche et chasse

Le crénage (kerning) ajuste l'espace entre deux caractères spécifiques pour corriger les déséquilibres optiques (par exemple, entre « A » et « V »). L'approche (tracking) modifie uniformément l'espace entre tous les caractères d'un mot ou d'un bloc. La chasse est la largeur qu'occupe chaque caractère.

Typographie pour le web

Le web a ses propres contraintes typographiques. Les web fonts (Google Fonts, Adobe Fonts) permettent d'utiliser des milliers de polices sur le web. Les polices variables (variable fonts) regroupent toutes les graisses et styles en un seul fichier, optimisant les performances. Le responsive type adapte la taille et l'interlignage à la largeur de l'écran via des unités relatives (rem, em, vw, clamp).

Question 1 / 6

Qu'est-ce qu'un empattement (serif) ?

L'empattement (serif) est le petit trait décoratif aux extrémités des fûts des lettres. Les polices serif (Times, Garamond) en possèdent ; les sans-serif (Helvetica, Arial) n'en ont pas.

Question 2 / 6

Quel est l'interlignage recommandé pour un texte courant ?

Un interlignage de 120 à 150 % du corps assure une lecture confortable. Trop serré, les lignes se chevauchent visuellement ; trop lâche, le texte perd sa cohésion.

Question 3 / 6

Combien de polices maximum recommande-t-on par projet ?

La règle d'or est de ne pas dépasser 2 à 3 polices par projet : une pour les titres, une pour le texte courant, éventuellement une troisième pour les accents. La cohérence est la marque d'un design professionnel.

Question 4 / 6

Quelle est la longueur de ligne idéale pour un texte courant ?

Une ligne de 45 à 75 caractères offre le meilleur confort de lecture. En dessous, le texte est haché ; au-dessus, l'oeil a du mal à retrouver le début de la ligne suivante.

Question 5 / 6

Qu'est-ce que le crénage (kerning) ?

Le crénage ajuste l'espace entre deux caractères précis pour corriger les déséquilibres optiques (par exemple AV, To, Ly). Il se distingue de l'approche (tracking) qui modifie uniformément l'espacement de tous les caractères.

Question 6 / 6

Quel type de police est privilégié pour le code informatique ?

Les polices monospace (Courier, JetBrains Mono, Fira Code) donnent la même largeur à chaque caractère, ce qui aligne verticalement le code et facilite sa lecture.

Exercices

Mettez en pratique

Exercice 1 — Classification

Classez ces polices dans leur famille : Garamond, Helvetica, Courier New, Brush Script, Futura, Georgia.

Voir la solution

Serif : Garamond, Georgia. Sans-serif : Helvetica, Futura. Monospace : Courier New. Script : Brush Script.

Exercice 2 — Hiérarchie

Vous devez hiérarchiser ces éléments pour un article de blog : titre principal, sous-titre, corps de texte, citation, légende d'image. Proposez une échelle typographique cohérente.

Voir la solution

Exemple d'échelle : Titre principal : 32px, bold, sans-serif. Sous-titre : 24px, semibold, sans-serif. Corps : 18px, regular, serif, interlignage 1.6. Citation : 20px, italic, serif, avec bordure gauche colorée. Légende : 14px, regular, sans-serif, couleur atténuée. Le ratio entre les niveaux doit être d'environ 1.25 à 1.5 (échelle typographique).

Exercice 3 — Appariement

Proposez un couple de polices (titre + corps) pour : (a) un magazine de mode, (b) un site de documentation technique, (c) une boulangerie artisanale.

Voir la solution

(a) Magazine de mode : Playfair Display (titre) + Lato (corps) — élégance et contraste serif/sans-serif. (b) Documentation technique : Inter (titre, bold) + Inter (corps, regular) — clarté, neutralité, une seule famille avec plusieurs graisses. (c) Boulangerie : Lora (titre) + DM Sans (corps) — warmth et lisibilité, serif traditionnel + sans-serif moderne.

Exercice 4 — Diagnostic

Un texte web a ces paramètres : police 12px, interlignage 1.0, ligne de 120 caractères, 5 polices différentes. Identifiez les problèmes et proposez des corrections.

Voir la solution

Problèmes : (1) 12px trop petit pour le web → passer à 16-18px. (2) Interlignage 1.0 trop serré → passer à 1.5 (150 %). (3) 120 caractères par ligne trop long → réduire à 60-75 caractères (max-width sur le conteneur). (4) 5 polices = incohérence visuelle → réduire à 2-3 polices maximum. Chacun de ces paramètres nuit individuellement à la lisibilité ; combinés, ils rendent le texte très difficile à lire.
