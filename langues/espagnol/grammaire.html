Introduction

La grammaire espagnole pour les francophones

La grammaire espagnole partage beaucoup de points communs avec le français : les deux langues ont des genres grammaticaux (masculin/féminin), des articles définis et indéfinis, et une conjugaison riche. Cependant, certaines différences fondamentales existent.

La grande différence : L'espagnol possède deux verbes « être » — ser et estar. Cette distinction, absente du français, est souvent la plus grande source d'erreurs pour les francophones.

Ce cours couvre les règles grammaticales fondamentales : les articles, le genre et le nombre, la distinction ser/estar, les prépositions et la structure des phrases.

02

Le cours

Les fondamentaux de la grammaire espagnole

Les articles

L'espagnol a quatre articles définis (contre deux en français) :

  • el (masculin singulier) : el libro → le livre
  • la (féminin singulier) : la casa → la maison
  • los (masculin pluriel) : los libros → les livres
  • las (féminin pluriel) : las casas → les maisons

Les articles indéfinis : un, una, unos, unas.

Attention : Le genre des noms ne correspond pas toujours entre le français et l'espagnol. Exemples : la leche (le lait), el color (la couleur), la sangre (le sang).

Ser vs Estar : les deux « être »

C'est la distinction la plus importante de la grammaire espagnole :

SER — pour les caractéristiques permanentes, l'identité, l'origine :

  • Soy francés. → Je suis français. (identité)
  • Es médico. → Il est médecin. (profession)
  • La mesa es de madera. → La table est en bois. (matière)
  • Son las tres. → Il est trois heures. (heure)

ESTAR — pour les états temporaires, la localisation, les émotions passagères :

  • Estoy cansado. → Je suis fatigué. (état temporaire)
  • Madrid está en España. → Madrid est en Espagne. (localisation)
  • Estamos contentos. → Nous sommes contents. (émotion)
  • La puerta está abierta. → La porte est ouverte. (résultat)

Astuce mnémotechnique : SER = ce que quelque chose EST par nature. ESTAR = ce que quelque chose EST en ce moment. Un même adjectif peut changer de sens : Ser listo = être intelligent. Estar listo = être prêt.

Les prépositions

Les principales prépositions et leurs différences avec le français :

  • a — direction, complément d'objet animé : Voy a Madrid. (Je vais à Madrid.) / Veo a María. (Je vois María.)
  • de — origine, possession : Soy de Francia. (Je suis de France.)
  • en — lieu (intérieur), moyen : Estoy en casa. (Je suis à la maison.)
  • por / para — deux mots pour « pour » : por (cause, échange) vs para (but, destination)

La phrase négative et interrogative

La négation est simple : on place no devant le verbe :

  • No hablo italiano. → Je ne parle pas italien.
  • No tengo hambre. → Je n'ai pas faim.

L'interrogation utilise des signes inversés au début et à la fin :

  • ¿Hablas español? → Tu parles espagnol ?
  • ¿Dónde vives? → Où habites-tu ?

Por vs Para : les deux « pour »

C'est l'une des distinctions les plus délicates pour les francophones :

POR — cause, échange, durée, moyen :

  • Gracias por tu ayuda. → Merci pour ton aide (cause)
  • Compré el libro por diez euros. → J'ai acheté le livre pour dix euros (échange)
  • Estudié por dos horas. → J'ai étudié pendant deux heures (durée)
  • Viajamos por avión. → Nous voyageons par avion (moyen)

PARA — but, destination, délai, destinataire :

  • Estudio para aprender. → J'étudie pour apprendre (but)
  • Salgo para Madrid. → Je pars pour Madrid (destination)
  • Es para mañana. → C'est pour demain (délai)
  • Este regalo es para ti. → Ce cadeau est pour toi (destinataire)

Astuce mnémotechnique : POR regarde vers le passé (la cause, l'échange déjà fait). PARA regarde vers le futur (le but, la destination à venir).

Les pronoms personnels

L'espagnol omet souvent le pronom sujet car la conjugaison suffit à identifier la personne :

  • Hablo español. → Je parle espagnol (pas besoin de « yo »)
  • ¿Hablas francés? → Tu parles français ?

On utilise le pronom sujet pour insister ou lever une ambiguïté :

  • Yo hablo español, pero él habla inglés. → Moi je parle espagnol, mais lui parle anglais.

Pièges courants pour les francophones

Les erreurs les plus fréquentes :

  • Le « a » personnel : Devant un COD humain, on ajoute a. On dit Veo a María (pas *Veo María). Cette règle n'existe pas en français.
  • Confondre ser et estar : Es aburrido (il est ennuyeux / c'est une personne ennuyeuse) vs Está aburrido (il s'ennuie / il est ennuyé en ce moment).
  • Le genre des noms : El problema (masculin !), el mapa (masculin !), la mano (féminin !). Ces exceptions piègent les francophones.
  • L'accent écrit : L'accent change le sens : si (si) vs sí (oui), el (le) vs él (lui), tu (ton) vs tú (tu).

Exercices

Mettez en pratique

Exercice 1 — Ser ou Estar ?

Complétez avec la forme correcte de ser ou estar :

1. Madrid ___ en España.
2. Mi hermana ___ médica.
3. Hoy yo ___ muy cansado.
4. La mesa ___ de madera.
5. ¿Dónde ___ el baño?
6. Nosotros ___ franceses.

Voir la solution

1. Madrid está en España. → localisation = estar
2. Mi hermana es médica. → profession = ser
3. Hoy yo estoy muy cansado. → état temporaire = estar
4. La mesa es de madera. → matière = ser
5. ¿Dónde está el baño? → localisation = estar
6. Nosotros somos franceses. → nationalité = ser

Exercice 2 — Por ou Para ?

Complétez avec por ou para :

1. Este regalo es ___ mi madre.
2. Gracias ___ venir.
3. Estudio ___ aprobar el examen.
4. Pagué veinte euros ___ el libro.
5. Salimos ___ Barcelona mañana.

Voir la solution

1. Este regalo es para mi madre. → destinataire
2. Gracias por venir. → cause
3. Estudio para aprobar el examen. → but
4. Pagué veinte euros por el libro. → échange
5. Salimos para Barcelona mañana. → destination

Exercice 3 — Traduisez en espagnol

1. Je vois Marie dans le parc.
2. Le problème est difficile.
3. Où est la gare ?
4. Il est médecin, mais aujourd'hui il est malade.

Voir la solution

1. Veo a María en el parque. → N'oubliez pas le « a » personnel !
2. El problema es difícil. → « Problema » est masculin.
3. ¿Dónde está la estación? → Localisation = estar.
4. Es médico, pero hoy está enfermo. → Profession = ser, état temporaire = estar.
