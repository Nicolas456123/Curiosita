01

Introduction

Modéliser le hasard discret

Une loi de probabilité discrète décrit le comportement d'une variable aléatoire qui ne peut prendre qu'un nombre fini ou dénombrable de valeurs (souvent des entiers). Le nombre de succès dans une série d'essais, le nombre de clients arrivant dans une file d'attente, le nombre de lancers avant d'obtenir un résultat : toutes ces situations se modélisent par des lois discrètes.

Variable aléatoire discrète : Une variable aléatoire X est discrète si son ensemble de valeurs possibles X(Ω) = {x₁, x₂, ...} est fini ou dénombrable. Sa loi est entièrement déterminée par les probabilités P(X = xᵢ) pour chaque valeur xᵢ.

Ce cours présente les quatre lois discrètes les plus importantes : la loi de Bernoulli (une seule épreuve), la loi binomiale (répétition d'épreuves de Bernoulli), la loi de Poisson (événements rares) et la loi géométrique (temps d'attente du premier succès). Chacune possède des formules d'espérance et de variance qui permettent de caractériser rapidement le comportement de la variable aléatoire.

Le cours

Bernoulli, binomiale, Poisson et géométrique

Loi de Bernoulli

La loi de Bernoulli de paramètre p modélise une expérience à deux issues : succès (1) avec probabilité p, et échec (0) avec probabilité 1-p = q.

X ~ Bernoulli(p) : P(X=1) = p, P(X=0) = 1-p

E(X) = p, V(X) = p(1-p) = pq

C'est la brique élémentaire de toutes les lois discrètes. Un lancer de pièce (pile/face), un tirage oui/non, un contrôle qualité (conforme/défectueux) sont des épreuves de Bernoulli.

Exemple : Un composant a une probabilité p = 0,05 d'être défectueux. Pour un seul composant, X ~ Bernoulli(0,05). E(X) = 0,05, V(X) = 0,05 × 0,95 = 0,0475.

Loi binomiale

La loi binomiale de paramètres n et p modélise le nombre de succès dans n épreuves de Bernoulli indépendantes, chacune de probabilité de succès p.

X ~ B(n, p) : P(X = k) = C(n,k) × pᵏ × (1-p)ⁿ⁻ᵏ pour k = 0, 1, ..., n

E(X) = np, V(X) = np(1-p), σ(X) = √(np(1-p))

Conditions d'application :

  • Nombre fixe n d'épreuves
  • Chaque épreuve a deux issues (succès/échec)
  • Les épreuves sont indépendantes
  • La probabilité de succès p est constante

Exemple : On lance 10 fois une pièce équilibrée. X = nombre de « pile ». X ~ B(10, 0,5).

  • P(X = 3) = C(10,3) × 0,5³ × 0,5⁷ = 120 × (1/1024) = 120/1024 ≈ 0,117
  • E(X) = 10 × 0,5 = 5
  • V(X) = 10 × 0,5 × 0,5 = 2,5, σ(X) ≈ 1,58

Propriété : La somme de deux variables binomiales indépendantes B(n₁, p) et B(n₂, p) (même p) suit une loi B(n₁+n₂, p).

Loi de Poisson

La loi de Poisson de paramètre λ > 0 modélise le nombre d'événements survenant dans un intervalle de temps (ou d'espace) fixé, lorsque ces événements sont rares et indépendants.

X ~ P(λ) : P(X = k) = e⁻λ × λᵏ / k! pour k = 0, 1, 2, ...

E(X) = λ, V(X) = λ

La particularité de la loi de Poisson est que l'espérance et la variance sont égales (toutes deux valent λ).

Exemples classiques :

  • Nombre d'appels reçus par un standard en une heure
  • Nombre de fautes de frappe dans un texte de 100 pages
  • Nombre d'accidents sur une autoroute par mois
  • Nombre de désintégrations radioactives par seconde

Exemple numérique : Un standard reçoit en moyenne 4 appels par heure. X ~ P(4).

  • P(X = 0) = e⁻⁴ × 4⁰/0! = e⁻⁴ ≈ 0,0183
  • P(X = 2) = e⁻⁴ × 4²/2! = e⁻⁴ × 8 ≈ 0,1465
  • P(X ≥ 1) = 1 - P(X=0) ≈ 1 - 0,0183 = 0,9817

Approximation de Poisson : Quand n est grand, p est petit, et λ = np est modéré (typiquement n ≥ 30, p ≤ 0,1), la loi B(n,p) est bien approchée par la loi P(np). C'est la « loi des événements rares ».

Loi géométrique

La loi géométrique de paramètre p modélise le rang du premier succès dans une suite d'épreuves de Bernoulli indépendantes. C'est la « loi du temps d'attente ».

X ~ Géom(p) : P(X = k) = (1-p)ᵏ⁻¹ × p pour k = 1, 2, 3, ...

E(X) = 1/p, V(X) = (1-p)/p²

La loi géométrique est la seule loi discrète possédant la propriété d'absence de mémoire : P(X > s+t | X > s) = P(X > t). Le passé n'influence pas le futur.

Exemple : On lance un dé jusqu'à obtenir un 6. X = nombre de lancers nécessaires. X ~ Géom(1/6).

  • E(X) = 1/(1/6) = 6 lancers en moyenne
  • P(X = 1) = 1/6 ≈ 0,167 (succès au 1er lancer)
  • P(X = 4) = (5/6)³ × (1/6) ≈ 0,0965
  • P(X > 10) = (5/6)¹⁰ ≈ 0,162

Tableau récapitulatif

Voici un résumé des quatre lois discrètes avec leurs caractéristiques :

  • Bernoulli(p) : X ∈ {0,1}, E = p, V = pq. Usage : une seule épreuve.
  • B(n,p) : X ∈ {0,...,n}, E = np, V = npq. Usage : n épreuves, compter les succès.
  • P(λ) : X ∈ ℕ, E = λ, V = λ. Usage : événements rares dans un intervalle.
  • Géom(p) : X ∈ ℕ*, E = 1/p, V = q/p². Usage : attente du premier succès.

Comment reconnaître la bonne loi ?

Pour identifier la loi appropriée, posez-vous ces questions :

  • Une seule épreuve oui/non ? → Bernoulli
  • n épreuves identiques et indépendantes, on compte les succès ? → Binomiale
  • Événements rares dans un intervalle continu ? → Poisson
  • On attend le premier succès ? → Géométrique
  • n grand, p petit, np modéré ? → Binomiale approchée par Poisson

Attention : La loi binomiale suppose des épreuves indépendantes avec probabilité constante. Si les tirages se font sans remise dans une population finie, c'est la loi hypergéométrique qui s'applique (la binomiale reste une bonne approximation si la population est grande par rapport à l'échantillon).

Propriétés de stabilité

Certaines lois sont stables par sommation de variables indépendantes :

  • Binomiale : B(n₁,p) + B(n₂,p) = B(n₁+n₂, p) (même p obligatoire)
  • Poisson : P(λ₁) + P(λ₂) = P(λ₁+λ₂)

Ces propriétés sont très utiles en pratique. Par exemple, si deux sources indépendantes génèrent des événements selon des lois de Poisson de paramètres λ₁ et λ₂, le total suit une loi de Poisson de paramètre λ₁+λ₂.

Exercices

Mettez en pratique

Exercice 1 — Loi binomiale

Un QCM comporte 8 questions à 4 choix chacune. Un étudiant répond au hasard. Quelle est la probabilité qu'il ait exactement 3 bonnes réponses ?

Voir la solution

X ~ B(8, 1/4). P(X=3) = C(8,3) × (1/4)³ × (3/4)⁵ = 56 × (1/64) × (243/1024) = 56 × 243/65536 = 13608/65536 ≈ 0,2076 ≈ 20,8%.

Exercice 2 — Loi de Poisson

Un serveur web reçoit en moyenne 5 requêtes par seconde. Quelle est la probabilité de recevoir exactement 8 requêtes en une seconde ?

Voir la solution

X ~ P(5). P(X=8) = e⁻⁵ × 5⁸/8! = e⁻⁵ × 390625/40320 = 0,006738 × 9,6899 ≈ 0,0653 ≈ 6,5%.

Exercice 3 — Loi géométrique

La probabilité de gagner à un jeu est 0,1 à chaque partie. Quelle est la probabilité de gagner pour la première fois à la 5e partie ? Quelle est la probabilité de devoir jouer plus de 10 parties avant de gagner ?

Voir la solution

X ~ Géom(0,1). P(X=5) = 0,9⁴ × 0,1 = 0,6561 × 0,1 = 0,0656. P(X>10) = (1-0,1)¹⁰ = 0,9¹⁰ ≈ 0,3487 ≈ 34,9%. Il y a donc environ 35% de chances de devoir jouer plus de 10 parties.

Exercice 4 — Approximation de Poisson

Un lot de 1000 composants a un taux de défaut de 0,2%. En utilisant l'approximation de Poisson, calculer la probabilité d'avoir au moins 3 composants défectueux.

Voir la solution

X ~ B(1000, 0,002) ≈ P(λ=2). P(X≥3) = 1 - P(X≤2) = 1 - [P(X=0)+P(X=1)+P(X=2)]. P(X=0)=e⁻²≈0,1353. P(X=1)=2e⁻²≈0,2707. P(X=2)=2e⁻²≈0,2707. P(X≤2)≈0,6767. Donc P(X≥3) ≈ 1 - 0,6767 = 0,3233 ≈ 32,3%.

Exercice 5 — Problème combiné

Un contrôleur vérifie 15 pièces par lot. Le lot est accepté si au plus 1 pièce est défectueuse. Si le taux de défaut réel est de 10%, quelle est la probabilité que le lot soit accepté ?

Voir la solution

X ~ B(15, 0,1). P(X≤1) = P(X=0) + P(X=1). P(X=0) = C(15,0)×0,1⁰×0,9¹⁵ = 0,9¹⁵ ≈ 0,2059. P(X=1) = C(15,1)×0,1¹×0,9¹⁴ = 15×0,1×0,2288 ≈ 0,3432. P(X≤1) ≈ 0,2059 + 0,3432 = 0,5491 ≈ 54,9%. Le lot est accepté environ une fois sur deux, ce qui montre les limites de ce contrôle.
