Introduction

Bouger, tourner, refléter, agrandir

Les transformations géométriques sont des opérations qui associent à chaque point du plan un autre point, selon une règle précise. Elles permettent de décrire les mouvements (déplacements rigides) et les changements d'échelle des figures. On les retrouve partout : dans les pavages (carrelage, motifs de papier peint), dans la nature (symétrie des cristaux, des feuilles, des flocons de neige) et dans l'art (frises, rosaces, motifs islamiques).

On distingue deux grandes familles : les isométries, qui conservent les distances (translation, rotation, symétrie), et les similitudes, qui conservent les angles mais peuvent modifier les distances (homothétie). Ces transformations forment des structures algébriques (groupes) qui sont au cœur de la géométrie moderne.

Principe fondamental : une transformation géométrique est entièrement déterminée par l'image de quelques points. Par exemple, une isométrie du plan est déterminée par l'image de trois points non alignés.

Ce cours présente les quatre transformations classiques du plan : la translation, la rotation, la symétrie (axiale et centrale) et l'homothétie. Pour chacune, nous étudierons la définition, les propriétés conservées, et les coordonnées de l'image d'un point.

Le cours

Les quatre transformations fondamentales

La translation

La translation de vecteur u⃗ = (a, b) associe à chaque point M(x, y) le point M'(x + a, y + b). C'est un « glissement » de toute la figure dans une direction donnée, sans rotation ni déformation.

Formule : M(x, y) → M'(x + a, y + b)

La translation conserve les distances, les angles, les aires, le parallélisme et l'orientation.

Propriétés :

  • Tout segment [AB] est envoyé sur un segment [A'B'] parallèle et de même longueur
  • La composée de deux translations est une translation : tu⃗ ∘ tv⃗ = tu⃗+v⃗
  • La translation inverse est t-u⃗
  • Une translation n'a pas de point fixe (sauf si le vecteur est nul)

Exemple : la translation de vecteur (3, -2) envoie le point A(1, 4) sur A'(4, 2) et le point B(5, 1) sur B'(8, -1).

La rotation

La rotation de centre O et d'angle θ fait tourner chaque point M autour de O d'un angle θ (positif dans le sens anti-horaire). La distance OM est conservée.

Formule : pour une rotation de centre O(0,0) et d'angle θ :

x' = x·cos(θ) - y·sin(θ)

y' = x·sin(θ) + y·cos(θ)

Propriétés :

  • Le centre O est le seul point fixe (sauf si θ = 0)
  • La rotation conserve les distances, les angles et les aires
  • La composée de deux rotations de même centre est une rotation : l'angle est la somme des angles
  • La rotation inverse est la rotation d'angle -θ
  • Une rotation de 180° est une symétrie centrale

Exemple : la rotation de centre O(0,0) et d'angle 90° envoie A(3, 1) sur A'(-1, 3), car x' = 3×cos(90°) - 1×sin(90°) = 0 - 1 = -1 et y' = 3×sin(90°) + 1×cos(90°) = 3 + 0 = 3.

La symétrie axiale (réflexion)

La symétrie axiale d'axe (d) associe à chaque point M le point M' tel que (d) est la médiatrice de [MM']. C'est un « miroir » par rapport à la droite (d).

Cas particuliers :

Symétrie par rapport à l'axe Ox : M(x, y) → M'(x, -y)

Symétrie par rapport à l'axe Oy : M(x, y) → M'(-x, y)

Symétrie par rapport à y = x : M(x, y) → M'(y, x)

Propriétés :

  • Chaque point de l'axe est un point fixe
  • La symétrie conserve les distances, les angles et les aires
  • La symétrie inverse l'orientation : un triangle ABC orienté dans le sens direct a une image A'B'C' orientée dans le sens indirect
  • La composée d'une symétrie avec elle-même est l'identité (s ∘ s = Id)

Exemple : le symétrique du point A(3, 2) par rapport à l'axe Ox est A'(3, -2). Par rapport à l'axe Oy, c'est A'(-3, 2).

La symétrie centrale

La symétrie centrale de centre O associe à chaque point M le point M' tel que O est le milieu de [MM']. Si O = (a, b), alors M(x, y) → M'(2a - x, 2b - y).

Formule : M(x, y) → M'(2a - x, 2b - y) (centre O(a, b))

La symétrie centrale est une rotation de 180° autour de O.

Propriétés :

  • Le centre O est le seul point fixe
  • Elle conserve distances, angles, aires et parallélisme
  • Contrairement à la symétrie axiale, elle conserve l'orientation
  • s ∘ s = Id (la composée avec elle-même donne l'identité)

Exemple : le symétrique de A(5, 3) par rapport au centre O(2, 1) est A'(2×2 - 5, 2×1 - 3) = A'(-1, -1).

L'homothétie

L'homothétie de centre O et de rapport k (k ≠ 0) associe à chaque point M le point M' tel que OM' = k × OM (en tant que vecteurs). C'est un agrandissement (|k| > 1) ou une réduction (|k| < 1).

Formule : M(x, y) → M'(a + k(x - a), b + k(y - b)) (centre O(a, b))

Si O est l'origine : M(x, y) → M'(kx, ky)

Propriétés :

  • Le centre O est le seul point fixe
  • L'homothétie conserve les angles et le parallélisme
  • Les distances sont multipliées par |k| : A'B' = |k| × AB
  • Les aires sont multipliées par k² : A' = k² × A
  • Les volumes sont multipliés par |k|³
  • Si k > 0, l'orientation est conservée. Si k < 0, elle est inversée
  • L'homothétie de rapport k = -1 est la symétrie centrale

Exemple : l'homothétie de centre O(0,0) et de rapport 2 envoie le triangle de sommets (1,1), (3,1), (2,3) sur le triangle (2,2), (6,2), (4,6). Les côtés sont doublés, l'aire est quadruplée.

Composition de transformations

La composition de deux transformations f et g, notée g ∘ f, applique d'abord f puis g. Résultats importants :

  • Deux translations : la composée est une translation (les vecteurs s'additionnent)
  • Deux rotations de même centre : une rotation (les angles s'additionnent)
  • Deux symétries axiales d'axes parallèles : une translation (de vecteur perpendiculaire aux axes, de norme double de la distance entre les axes)
  • Deux symétries axiales d'axes sécants : une rotation (de centre le point d'intersection, d'angle double de l'angle entre les axes)
  • Deux homothéties de même centre : une homothétie (les rapports se multiplient)

Isométries et classification

Les isométries sont les transformations qui conservent toutes les distances. En dimension 2, il n'en existe que quatre types :

  • L'identité (cas trivial)
  • Les translations (pas de point fixe)
  • Les rotations (un point fixe : le centre)
  • Les symétries axiales (une droite de points fixes)

Les isométries qui conservent l'orientation (translations et rotations) sont appelées déplacements. Celles qui inversent l'orientation (symétries axiales et symétries glissées) sont appelées antidéplacements.

Pièges courants

  • Sens de rotation : par convention, un angle positif correspond à une rotation dans le sens anti-horaire (trigonométrique). Un angle négatif tourne dans le sens horaire.
  • Centre de rotation ≠ origine : si le centre n'est pas l'origine, il faut d'abord translater, puis appliquer la rotation, puis translater en sens inverse.
  • Rapport d'homothétie négatif : k < 0 signifie que l'image est de l'autre côté du centre par rapport à l'original. k = -1 donne une symétrie centrale.
  • Symétrie axiale et orientation : la symétrie axiale est la seule isométrie simple qui inverse l'orientation. Le triangle ABC et son image A'B'C' sont « en miroir ».
  • Aire et homothétie : le rapport des aires est k², pas k. Un agrandissement de rapport 3 multiplie les aires par 9, pas par 3.

Applications

  • Pavages et frises : les motifs qui recouvrent le plan sans trou ni chevauchement sont engendrés par des compositions de translations, rotations et symétries. Il existe exactement 17 types de pavages périodiques du plan.
  • Cristallographie : les cristaux possèdent des symétries de rotation et de réflexion qui déterminent leurs propriétés physiques.
  • Infographie : les transformations (translation, rotation, mise à l'échelle) sont à la base de toutes les animations et modélisations 3D.
  • Cartographie : les projections cartographiques sont des transformations qui mettent en correspondance la sphère terrestre et un plan.

Question 1 / 6

L'image du point A(2, 5) par la translation de vecteur (3, -1) est :

Par la translation de vecteur (3, -1) : x' = 2 + 3 = 5, y' = 5 + (-1) = 4. Donc A' = (5, 4).

Question 2 / 6

Quelle transformation inverse l'orientation d'une figure ?

La symétrie axiale (réflexion) est la seule isométrie simple qui inverse l'orientation. La symétrie centrale conserve l'orientation car c'est une rotation de 180°.

Question 3 / 6

L'image de (4, 2) par la rotation de centre O(0,0) et d'angle 90° est :

x' = 4×cos(90°) - 2×sin(90°) = 0 - 2 = -2. y' = 4×sin(90°) + 2×cos(90°) = 4 + 0 = 4. Donc l'image est (-2, 4).

Question 4 / 6

Une homothétie de rapport k = 3 multiplie les aires par :

Les distances sont multipliées par |k| = 3, les aires par k² = 9, et les volumes par |k|³ = 27. L'aire dépend de deux dimensions, d'où le carré.

Question 5 / 6

La composée de deux symétries axiales dont les axes sont parallèles est :

La composée de deux symétries axiales d'axes parallèles distants de d est une translation de norme 2d, perpendiculaire aux axes. Si les axes sont sécants, c'est une rotation.

Question 6 / 6

Le symétrique du point M(3, -1) par rapport au centre I(1, 2) est :

M'(2×1 - 3, 2×2 - (-1)) = M'(2 - 3, 4 + 1) = M'(-1, 5). I est bien le milieu de [MM'] : ((3-1)/2, (-1+5)/2) = (1, 2) = I. ✓

Exercices

Mettez en pratique

Exercice 1 — Translation

Un triangle a pour sommets A(1, 2), B(4, 2) et C(2, 5). On applique la translation de vecteur u⃗ = (-3, 4). Déterminer les coordonnées de A', B' et C'.

Voir la solution

A' = (1 + (-3), 2 + 4) = (-2, 6). B' = (4 - 3, 2 + 4) = (1, 6). C' = (2 - 3, 5 + 4) = (-1, 9). On peut vérifier que A'B' = AB = 3, B'C' = BC = √13, et A'C' = AC = √12 (les distances sont conservées).

Exercice 2 — Rotation

Déterminer l'image du point A(2, 0) par la rotation de centre O(0, 0) et d'angle 60°.

Voir la solution

x' = 2×cos(60°) - 0×sin(60°) = 2×(1/2) - 0 = 1. y' = 2×sin(60°) + 0×cos(60°) = 2×(√3/2) + 0 = √3. L'image est A'(1, √3) ≈ (1 ; 1,73). Vérification : OA' = √(1 + 3) = 2 = OA. ✓

Exercice 3 — Symétrie axiale

Trouver le symétrique du point P(5, 3) par rapport à la droite y = x.

Voir la solution

La symétrie par rapport à la droite y = x échange les coordonnées : P(x, y) → P'(y, x). Donc P'(3, 5). Vérification : le milieu de [PP'] est ((5+3)/2, (3+5)/2) = (4, 4), qui est bien sur la droite y = x. ✓ Le segment [PP'] a pour pente (5-3)/(3-5) = -1, qui est bien perpendiculaire à y = x (pente 1). ✓

Exercice 4 — Homothétie

On applique l'homothétie de centre A(1, 1) et de rapport k = -2 au point B(3, 4). Trouver B'. Que vaut le rapport des aires si on applique cette homothétie à un carré de côté 5 ?

Voir la solution

B' = (1 + (-2)(3-1), 1 + (-2)(4-1)) = (1 - 4, 1 - 6) = (-3, -5). Le rapport k est négatif, donc B' est de l'autre côté de A par rapport à B. Le rapport des aires est k² = (-2)² = 4. Le carré de côté 5 (aire 25) devient un carré de côté 10 (aire 100 = 4 × 25). ✓

Exercice 5 — Composition

On effectue d'abord une symétrie par rapport à l'axe Oy, puis une rotation de 90° autour de l'origine. Quelle est l'image du point M(2, 3) ?

Voir la solution

Étape 1 — Symétrie par rapport à Oy : M(2, 3) → M₁(-2, 3).
Étape 2 — Rotation de 90° autour de O : x' = -2×cos(90°) - 3×sin(90°) = 0 - 3 = -3. y' = -2×sin(90°) + 3×cos(90°) = -2 + 0 = -2. Donc M' = (-3, -2). On peut vérifier que OM' = √(9 + 4) = √13 = OM, ce qui est cohérent avec la conservation des distances par les isométries.
