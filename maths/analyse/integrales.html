01

Introduction

Accumuler l'infiniment petit

L'intégration est le processus qui permet de calculer des accumulations : aire sous une courbe, distance parcourue, volume d'un solide, travail d'une force. Là où la dérivée mesure le changement instantané, l'intégrale additionne une infinité de contributions infinitésimales pour obtenir un total.

Le théorème fondamental du calcul établit que dérivation et intégration sont des opérations inverses. Si F est une primitive de f (c'est-à-dire F' = f), alors l'intégrale de f entre a et b vaut F(b) − F(a). Ce lien profond est le coeur de toute l'analyse.

Historiquement, Archimède calculait déjà des aires par la méthode d'exhaustion (approximation par des polygones). Au XVIIe siècle, Newton et Leibniz ont formalisé le calcul intégral. Au XIXe siècle, Riemann a donné une définition rigoureuse de l'intégrale en termes de sommes de rectangles, puis Lebesgue a généralisé cette notion pour traiter des fonctions plus complexes.

Le cours

Primitives, intégrales et méthodes de calcul

Primitives

Une primitive de f sur un intervalle I est une fonction F telle que F'(x) = f(x) pour tout x dans I. Si F est une primitive de f, alors toutes les primitives de f sont de la forme F(x) + C, où C est une constante réelle.

Primitives des fonctions usuelles :

  • ∫ xn dx = xn+1/(n+1) + C (pour n ≠ −1)
  • ∫ 1/x dx = ln|x| + C
  • ∫ ex dx = ex + C
  • ∫ cos(x) dx = sin(x) + C
  • ∫ sin(x) dx = −cos(x) + C
  • ∫ 1/cos²(x) dx = tan(x) + C
  • ∫ 1/(1+x²) dx = arctan(x) + C
  • ∫ 1/√(1−x²) dx = arcsin(x) + C

L'intégrale définie

L'intégrale définie de f entre a et b, notée ∫ab f(x) dx, représente l'aire algébrique entre la courbe y = f(x), l'axe des abscisses et les droites verticales x = a et x = b.

Théorème fondamental du calcul :

Si f est continue sur [a, b] et F est une primitive de f, alors :

∫ab f(x) dx = F(b) − F(a) = [F(x)]ab

Exemple : ∫02 x² dx = [x³/3]02 = 8/3 − 0 = 8/3.

Propriétés fondamentales de l'intégrale :

  • Linéarité : ∫ab (αf + βg) dx = α∫ab f dx + β∫ab g dx
  • Relation de Chasles : ∫ab f dx + ∫bc f dx = ∫ac f dx
  • Inversion des bornes : ∫ab f dx = −∫ba f dx
  • Positivité : si f ≥ 0 sur [a, b], alors ∫ab f dx ≥ 0

Intégration par substitution (changement de variable)

Cette méthode est l'analogue de la règle de la chaîne pour les dérivées. Si u = g(x), alors :

∫ f(g(x)) × g'(x) dx = ∫ f(u) du

Exemple : calculons ∫ 2x × ex² dx. On pose u = x², donc du = 2x dx. L'intégrale devient ∫ eu du = eu + C = ex² + C.

Exemple avec bornes : ∫01 2x(x²+1)³ dx. On pose u = x²+1, du = 2x dx. Quand x = 0, u = 1 ; quand x = 1, u = 2. Donc ∫12 u³ du = [u⁴/4]12 = 16/4 − 1/4 = 15/4.

Intégration par parties

Dérivée de la formule du produit, l'intégration par parties (IPP) permet de transférer la dérivation d'un facteur à l'autre :

∫ab u(x)v'(x) dx = [u(x)v(x)]ab − ∫ab u'(x)v(x) dx

Astuce mnémotechnique LIATE pour choisir u (dans l'ordre de priorité) : Logarithme, Inverse trigonométrique, Algébrique (polynôme), Trigonométrique, Exponentielle.

Exemple : calculons ∫ x·ex dx. On pose u = x (algébrique) et v' = ex, donc u' = 1 et v = ex. IPP : ∫ x·ex dx = x·ex − ∫ ex dx = x·ex − ex + C = ex(x − 1) + C.

Décomposition en éléments simples

Pour intégrer une fraction rationnelle P(x)/Q(x) (quotient de polynômes), on la décompose en somme de fractions plus simples :

  • Si Q a une racine simple r : on obtient un terme A/(x − r), dont la primitive est A·ln|x − r|
  • Si Q a une racine double r : on obtient A/(x − r) + B/(x − r)²
  • Si Q a un facteur irréductible x² + px + q : on obtient (Ax + B)/(x² + px + q), qui se ramène à des arctan et des logarithmes

Exemple : ∫ 1/(x²−1) dx. On décompose : 1/(x²−1) = 1/((x−1)(x+1)) = A/(x−1) + B/(x+1). En identifiant : A = 1/2, B = −1/2. Donc ∫ = (1/2)ln|x−1| − (1/2)ln|x+1| + C = (1/2)ln|(x−1)/(x+1)| + C.

Intégrales impropres (généralisées)

Lorsqu'une borne est infinie ou que la fonction n'est pas bornée, on parle d'intégrale impropre. On la définit comme une limite :

  • Borne infinie : ∫a+∞ f(x) dx = limb→+∞ ∫ab f(x) dx
  • Singularité : si f n'est pas bornée en a, ∫ab f(x) dx = limε→0⁺ ∫a+εb f(x) dx

Si la limite existe et est finie, l'intégrale converge ; sinon, elle diverge.

Exemple : ∫1+∞ 1/x² dx = limb→+∞ [−1/x]1b = limb→+∞ (−1/b + 1) = 1. L'intégrale converge vers 1.

En revanche, ∫1+∞ 1/x dx = limb→+∞ [ln(x)]1b = +∞. L'intégrale diverge.

Pièges courants

  • Oublier la constante d'intégration : ∫ f(x) dx = F(x) + C. La constante C disparaît dans les intégrales définies mais est indispensable pour les primitives
  • Confondre aire et intégrale : l'intégrale est une aire algébrique (signée). Si f < 0 sur un intervalle, l'intégrale y est négative. L'aire géométrique est ∫|f(x)| dx
  • Ne pas changer les bornes lors d'une substitution : si on pose u = g(x), les bornes deviennent g(a) et g(b)
  • Intégrer ∫ 1/x dx = ln(x) au lieu de ln|x| : le logarithme n'est défini que pour les réels positifs, la valeur absolue est nécessaire
  • Ignorer les singularités : avant de calculer ∫ab f(x) dx, vérifier que f est continue sur [a, b]. Si ce n'est pas le cas, il faut traiter l'intégrale comme impropre

Exercices

Mettez en pratique

Exercice 1 — Intégrale définie simple

Calculer ∫13 (2x + 1) dx.

Voir la solution

Une primitive de 2x + 1 est x² + x. Donc ∫13 (2x + 1) dx = [x² + x]13 = (9 + 3) − (1 + 1) = 12 − 2 = 10.

Exercice 2 — Intégration par substitution

Calculer ∫ cos(3x) dx.

Voir la solution

On pose u = 3x, donc du = 3 dx, soit dx = du/3. L'intégrale devient ∫ cos(u) × (du/3) = (1/3) sin(u) + C = (1/3) sin(3x) + C. Vérification : [(1/3) sin(3x)]' = (1/3) × 3 cos(3x) = cos(3x).

Exercice 3 — Intégration par parties

Calculer ∫ x·ln(x) dx.

Voir la solution

Par la règle LIATE, on choisit u = ln(x) (logarithme en priorité) et v' = x. Donc u' = 1/x et v = x²/2. IPP : ∫ x·ln(x) dx = (x²/2)·ln(x) − ∫ (x²/2)·(1/x) dx = (x²/2)·ln(x) − ∫ x/2 dx = (x²/2)·ln(x) − x²/4 + C = (x²/4)(2·ln(x) − 1) + C.

Exercice 4 — Aire entre deux courbes

Calculer l'aire comprise entre les courbes y = x² et y = x sur [0, 1].

Voir la solution

Sur [0, 1], on a x ≥ x² (car x(1−x) ≥ 0). L'aire est donc ∫01 (x − x²) dx = [x²/2 − x³/3]01 = 1/2 − 1/3 = 1/6.

Exercice 5 — Intégrale impropre

Déterminer si ∫1+∞ 1/x³ dx converge et, si oui, calculer sa valeur.

Voir la solution

∫1+∞ x−3 dx = limb→+∞ [x−2/(−2)]1b = limb→+∞ (−1/(2b²) + 1/2) = 0 + 1/2 = 1/2. L'intégrale converge. Cela illustre la règle de Riemann : ∫1+∞ 1/xα dx converge si et seulement si α > 1.
