Introduction

Pourquoi étudier les polynômes

Un polynôme est une expression de la forme P(x) = aₙxⁿ + aₙ₋₁xⁿ⁻¹ + ... + a₁x + a₀, où les aᵢ sont des coefficients et n est le degré du polynôme. Les polynômes sont les objets les plus fondamentaux de l'algèbre : ils généralisent les équations du premier et du second degré et apparaissent dans presque tous les domaines des mathématiques.

Théorème fondamental de l'algèbre : tout polynôme de degré n ≥ 1 à coefficients complexes admet exactement n racines (comptées avec multiplicité) dans les nombres complexes. Ce résultat, prouvé par Gauss en 1799, unifie toute la théorie.

La factorisation est l'opération clé : décomposer un polynôme en produit de facteurs plus simples. Elle permet de trouver les racines, de simplifier des expressions et de résoudre des équations. Ce cours couvre les méthodes essentielles : identités remarquables, division euclidienne, et racines évidentes.

02

Le cours

Polynômes, racines et factorisation

Opérations sur les polynômes

On peut additionner, soustraire et multiplier des polynômes en combinant les termes de même degré. Le degré du produit est la somme des degrés : deg(P × Q) = deg(P) + deg(Q).

Exemple : (2x² + 3x - 1)(x + 2) = 2x³ + 4x² + 3x² + 6x - x - 2 = 2x³ + 7x² + 5x - 2.

Division euclidienne des polynômes

Comme pour les entiers, on peut effectuer la division euclidienne d'un polynôme A par un polynôme B (de degré non nul) :

A = B × Q + R où deg(R) < deg(B). Q est le quotient et R le reste.

Exemple : Diviser P(x) = x³ - 2x² + 3x - 1 par D(x) = x - 1.

  • x³ ÷ x = x², puis x²(x-1) = x³ - x². Reste : -x² + 3x - 1
  • -x² ÷ x = -x, puis -x(x-1) = -x² + x. Reste : 2x - 1
  • 2x ÷ x = 2, puis 2(x-1) = 2x - 2. Reste : 1
  • Donc x³ - 2x² + 3x - 1 = (x - 1)(x² - x + 2) + 1

Racines et factorisation

Un nombre a est une racine de P si P(a) = 0. Si a est racine de P, alors (x - a) divise P (le reste de la division est nul). Réciproquement, si (x - a) divise P, alors a est racine de P.

Pour trouver les racines rationnelles d'un polynôme à coefficients entiers, on teste les diviseurs du terme constant divisés par les diviseurs du coefficient dominant.

Exemple : Factoriser P(x) = x³ - 6x² + 11x - 6.

  • On teste x = 1 : P(1) = 1 - 6 + 11 - 6 = 0. Donc (x - 1) est facteur.
  • Division : P(x) = (x - 1)(x² - 5x + 6)
  • On factorise x² - 5x + 6 = (x - 2)(x - 3)
  • Résultat : P(x) = (x - 1)(x - 2)(x - 3)

Identités remarquables (degré 3)

Au-delà des identités du degré 2, les identités de degré 3 sont très utiles :

  • (a + b)³ = a³ + 3a²b + 3ab² + b³
  • (a - b)³ = a³ - 3a²b + 3ab² - b³
  • a³ - b³ = (a - b)(a² + ab + b²)
  • a³ + b³ = (a + b)(a² - ab + b²)

Multiplicité des racines

Une racine a est de multiplicité k si (x - a)ᵏ divise P mais (x - a)ᵏ⁺¹ ne divise pas P. Par exemple, dans P(x) = (x - 1)²(x + 3), la racine 1 est de multiplicité 2 (racine double) et la racine -3 est de multiplicité 1 (racine simple).

Conséquence graphique : à une racine simple, la courbe traverse l'axe des x. À une racine double, elle touche l'axe sans le traverser (tangente). À une racine triple, elle traverse avec un point d'inflexion.

Schéma de Horner

Le schéma de Horner est une méthode efficace pour évaluer un polynôme et effectuer une division par (x - a). Le principe est de réécrire le polynôme sous forme emboîtée :

P(x) = aₙxⁿ + aₙ₋₁xⁿ⁻¹ + ... + a₁x + a₀ = (...((aₙx + aₙ₋₁)x + aₙ₋₂)x + ...)x + a₀

Exemple : Évaluer P(x) = 2x³ - 3x² + 4x - 5 en x = 3 par Horner :

  • On part de 2 (coefficient dominant)
  • 2 × 3 + (-3) = 3
  • 3 × 3 + 4 = 13
  • 13 × 3 + (-5) = 34
  • Donc P(3) = 34. Les coefficients intermédiaires (2, 3, 13) donnent le quotient : 2x² + 3x + 13 avec reste 34.

Polynômes irréductibles

Un polynôme est irréductible sur les réels s'il ne peut pas se factoriser en produit de polynômes de degré inférieur à coefficients réels. Sur les réels, les irréductibles sont les polynômes de degré 1 et les trinômes de degré 2 à discriminant strictement négatif.

Décomposition complète sur les réels : tout polynôme réel se décompose en produit de facteurs de degré 1 et de facteurs de degré 2 irréductibles (Δ < 0). Par exemple : x⁴ + 1 = (x² + √2·x + 1)(x² - √2·x + 1).

Pièges courants

Erreurs fréquentes à éviter avec les polynômes :

  • Confondre degré et nombre de termes : x⁵ + 1 est de degré 5 bien qu'il n'ait que 2 termes.
  • Oublier le coefficient dominant : dans 3x² - x + 2, le coefficient dominant est 3, pas 1. Cela affecte le discriminant et les formules de Viète.
  • Tester des diviseurs incomplets : pour les racines rationnelles, il faut tester ±(diviseurs du terme constant)/(diviseurs du coefficient dominant), pas seulement les diviseurs du terme constant.
  • Division euclidienne mal posée : ne pas oublier les termes de coefficient 0. Dans x³ + 1, il faut écrire x³ + 0x² + 0x + 1 pour la division.
  • Multiplicité oubliée : dans (x-1)²(x+2) = 0, la racine 1 compte double. Le polynôme a 3 racines comptées avec multiplicité.

Applications

Les polynômes sont utilisés dans de nombreux domaines :

  • Interpolation : le polynôme de Lagrange permet de trouver le polynôme de degré minimal passant par n points donnés. Utilisé en calcul numérique et en infographie.
  • Codes correcteurs : les codes de Reed-Solomon, utilisés dans les CD, DVD et QR codes, reposent sur les propriétés des polynômes sur les corps finis.
  • Approximation : les polynômes de Taylor approchent les fonctions compliquées par des polynômes : sin(x) ≈ x - x³/6 + x⁵/120 pour x proche de 0.

Exercices

Mettez en pratique

Exercice 1 — Division euclidienne

Effectuer la division euclidienne de P(x) = 2x³ + 5x² - 3x + 1 par D(x) = x + 3.

Voir la solution

2x³ ÷ x = 2x². 2x²(x+3) = 2x³ + 6x². Reste : -x² - 3x + 1. -x² ÷ x = -x. -x(x+3) = -x² - 3x. Reste : 0x + 1 = 1. 0 ÷ x... reste 1. P(x) = (x + 3)(2x² - x) + 1. Vérification : P(-3) = 2(-27) + 5(9) + 9 + 1 = -54 + 45 + 10 = 1 (= le reste).

Exercice 2 — Factorisation complète

Factoriser P(x) = x³ - 2x² - 5x + 6.

Voir la solution

On cherche les racines parmi les diviseurs de 6 : ±1, ±2, ±3, ±6. P(1) = 1 - 2 - 5 + 6 = 0. Donc (x - 1) divise P. Par Horner ou division : P(x) = (x - 1)(x² - x - 6). On factorise x² - x - 6 : Δ = 1 + 24 = 25, racines 3 et -2. P(x) = (x - 1)(x - 3)(x + 2).

Exercice 3 — Identité remarquable de degré 3

Factoriser 27a³ + 8b³.

Voir la solution

On reconnaît la somme de cubes : 27a³ = (3a)³ et 8b³ = (2b)³. En appliquant a³ + b³ = (a + b)(a² - ab + b²) avec a = 3a et b = 2b : 27a³ + 8b³ = (3a + 2b)(9a² - 6ab + 4b²).

Exercice 4 — Polynôme avec paramètre

Déterminer la valeur de k pour que (x - 2) divise P(x) = x³ + kx² - 5x + 2.

Voir la solution

(x - 2) divise P si et seulement si P(2) = 0. P(2) = 8 + 4k - 10 + 2 = 4k. Donc 4k = 0, soit k = 0. Vérification : P(x) = x³ - 5x + 2 et P(2) = 8 - 10 + 2 = 0.

Exercice 5 — Évaluation par Horner

Utiliser le schéma de Horner pour évaluer P(x) = 3x⁴ - 2x³ + x² - 4x + 7 en x = 2.

Voir la solution

Coefficients : 3, -2, 1, -4, 7. En x = 2 : 3 → 3×2 + (-2) = 4 → 4×2 + 1 = 9 → 9×2 + (-4) = 14 → 14×2 + 7 = 35. Vérification : 3(16) - 2(8) + 4 - 8 + 7 = 48 - 16 + 4 - 8 + 7 = 35.
