Introduction

Pourquoi les matrices sont essentielles

Une matrice est un tableau rectangulaire de nombres disposés en lignes et colonnes. Loin d'être un simple tableau, les matrices forment un outil mathématique d'une puissance extraordinaire : elles permettent de représenter des systèmes d'équations linéaires, des transformations géométriques, des graphes, des réseaux et bien plus encore.

Applications concrètes : Google utilise des matrices pour classer les pages web (PageRank). Les jeux vidéo utilisent des matrices 4×4 pour positionner chaque objet 3D à l'écran. Le machine learning repose entièrement sur des multiplications matricielles.

Ce cours couvre les opérations fondamentales sur les matrices : addition, multiplication, transposition, calcul du déterminant et de l'inverse. Ces outils sont le socle de l'algèbre linéaire, indispensable en physique, informatique, data science et ingénierie.

02

Le cours

Matrices, déterminants et inverse

Définitions et notations

Une matrice de taille m × n a m lignes et n colonnes. On note A = (aᵢⱼ) où aᵢⱼ est l'élément en ligne i et colonne j. Une matrice carrée a autant de lignes que de colonnes. La matrice identité Iₙ est la matrice carrée avec des 1 sur la diagonale et des 0 ailleurs.

Matrices particulières :

  • Matrice nulle : tous les éléments sont 0
  • Matrice diagonale : non nuls seulement sur la diagonale
  • Matrice symétrique : A = Aᵀ (symétrique par rapport à la diagonale)
  • Matrice triangulaire : nulle au-dessus ou au-dessous de la diagonale

Opérations sur les matrices

L'addition se fait terme à terme : (A + B)ᵢⱼ = aᵢⱼ + bᵢⱼ (les matrices doivent avoir la même taille). La multiplication par un scalaire : (λA)ᵢⱼ = λ·aᵢⱼ.

La multiplication matricielle est plus complexe. Pour A (m×n) et B (n×p), le produit AB est une matrice m×p dont l'élément (i,j) est :

(AB)ᵢⱼ = Σₖ aᵢₖ · bₖⱼ (somme des produits ligne i de A par colonne j de B)

Attention : la multiplication n'est pas commutative : AB ≠ BA en général. C'est une propriété fondamentale qui distingue les matrices des nombres ordinaires.

La transposée Aᵀ s'obtient en échangeant lignes et colonnes : (Aᵀ)ᵢⱼ = aⱼᵢ.

Déterminant

Le déterminant est un nombre associé à toute matrice carrée. Pour une matrice 2×2 :

det([[a,b],[c,d]]) = ad - bc

Pour une matrice 3×3, on développe selon la première ligne (règle de Sarrus ou cofacteurs). Le déterminant a plusieurs interprétations :

  • Géométrique : |det(A)| est le facteur de dilatation des aires (2D) ou des volumes (3D)
  • Inversibilité : A est inversible si et seulement si det(A) ≠ 0
  • Propriétés : det(AB) = det(A)·det(B), det(Aᵀ) = det(A), det(λA) = λⁿ·det(A)

Matrice inverse

La matrice inverse A⁻¹ vérifie A·A⁻¹ = A⁻¹·A = I. Elle n'existe que si det(A) ≠ 0. Pour une matrice 2×2 :

A⁻¹ = (1/det(A)) · [[d, -b], [-c, a]]

Pour les matrices plus grandes, on utilise la méthode de Gauss-Jordan : on juxtapose A et I, puis on applique le pivot de Gauss jusqu'à obtenir I à gauche. La matrice à droite est alors A⁻¹.

Application aux systèmes : le système AX = B se résout par X = A⁻¹B quand A est inversible.

Trace et propriétés

La trace d'une matrice carrée est la somme de ses éléments diagonaux : tr(A) = Σ aᵢᵢ. Propriétés utiles : tr(A + B) = tr(A) + tr(B) et tr(AB) = tr(BA).

Calcul de l'inverse d'une matrice 2×2

Exemple détaillé : Calculer l'inverse de A = [[2, 1], [5, 3]].

  • det(A) = 2×3 - 1×5 = 6 - 5 = 1
  • A⁻¹ = (1/1) × [[3, -1], [-5, 2]] = [[3, -1], [-5, 2]]
  • Vérification : A×A⁻¹ = [[2×3+1×(-5), 2×(-1)+1×2], [5×3+3×(-5), 5×(-1)+3×2]] = [[1, 0], [0, 1]] = I

Puissances de matrices

Pour calculer Aⁿ, on peut chercher une régularité ou diagonaliser la matrice. Si A = PDP⁻¹ avec D diagonale, alors Aⁿ = PDⁿP⁻¹, ce qui simplifie considérablement le calcul.

Exemple : Pour A = [[1, 1], [0, 1]], on vérifie par récurrence que Aⁿ = [[1, n], [0, 1]]. En effet, A² = [[1, 2], [0, 1]], A³ = [[1, 3], [0, 1]], etc.

Rang d'une matrice

Le rang d'une matrice est le nombre maximal de lignes (ou colonnes) linéairement indépendantes. On le calcule en échelonnant la matrice par le pivot de Gauss : le rang est le nombre de pivots non nuls.

Théorème du rang : pour une matrice m × n, rang(A) ≤ min(m, n). Le système AX = B a une solution unique si et seulement si rang(A) = n (nombre d'inconnues).

Pièges courants

  • Multiplier terme à terme : le produit matriciel n'est PAS aᵢⱼ × bᵢⱼ. Il faut faire les sommes produits ligne-colonne.
  • Supposer la commutativité : AB ≠ BA en général. L'ordre des facteurs compte.
  • AB = 0 n'implique pas A = 0 ou B = 0 : contrairement aux nombres, le produit de deux matrices non nulles peut être la matrice nulle.
  • Inverse d'un produit : (AB)⁻¹ = B⁻¹A⁻¹ (attention à l'ordre inversé), et non A⁻¹B⁻¹.
  • Erreur de taille : A (m×n) × B (n×p) = C (m×p). Le nombre de colonnes de A doit égaler le nombre de lignes de B.

Applications

  • Transformations géométriques : rotation d'angle θ : [[cos θ, -sin θ], [sin θ, cos θ]]. Réflexion, homothétie et projection sont aussi des matrices.
  • Chaînes de Markov : les matrices stochastiques modélisent les probabilités de transition entre états (météo, navigation web, génétique).
  • Compression d'images : la décomposition en valeurs singulières (SVD) permet de compresser une image en ne gardant que les composantes matricielles principales.
  • Machine learning : les réseaux de neurones sont fondamentalement des suites de multiplications matricielles suivies de fonctions d'activation.

Exercices

Mettez en pratique

Exercice 1 — Produit matriciel

Calculer le produit AB avec A = [[1, 2], [3, 4]] et B = [[5, 6], [7, 8]].

Voir la solution

(AB)₁₁ = 1×5 + 2×7 = 19. (AB)₁₂ = 1×6 + 2×8 = 22. (AB)₂₁ = 3×5 + 4×7 = 43. (AB)₂₂ = 3×6 + 4×8 = 50. AB = [[19, 22], [43, 50]]. Note : BA = [[23, 34], [31, 46]] ≠ AB (non commutatif).

Exercice 2 — Inverse d'une matrice

Calculer l'inverse de A = [[4, 7], [2, 6]].

Voir la solution

det(A) = 4×6 - 7×2 = 24 - 14 = 10. A⁻¹ = (1/10) × [[6, -7], [-2, 4]] = [[3/5, -7/10], [-1/5, 2/5]]. Vérification : AA⁻¹ = [[4(3/5)+7(-1/5), ...], [...]] = [[1, 0], [0, 1]].

Exercice 3 — Déterminant 3×3

Calculer le déterminant de A = [[1, 2, 3], [0, 1, 4], [5, 6, 0]].

Voir la solution

Développement selon la 1re ligne : det = 1×(1×0 - 4×6) - 2×(0×0 - 4×5) + 3×(0×6 - 1×5) = 1×(-24) - 2×(-20) + 3×(-5) = -24 + 40 - 15 = 1. Comme det ≠ 0, la matrice est inversible.

Exercice 4 — Système matriciel

Résoudre le système AX = B avec A = [[2, 1], [5, 3]] et B = [[4], [11]] en utilisant l'inverse de A.

Voir la solution

det(A) = 6 - 5 = 1. A⁻¹ = [[3, -1], [-5, 2]]. X = A⁻¹B = [[3×4 + (-1)×11], [(-5)×4 + 2×11]] = [[12-11], [-20+22]] = [[1], [2]]. Donc x = 1, y = 2. Vérification : 2(1) + 2 = 4 et 5(1) + 3(2) = 11.

Exercice 5 — Puissance de matrice

Calculer A³ pour A = [[1, 1], [0, 1]].

Voir la solution

A² = [[1, 1], [0, 1]] × [[1, 1], [0, 1]] = [[1, 2], [0, 1]]. A³ = A² × A = [[1, 2], [0, 1]] × [[1, 1], [0, 1]] = [[1, 3], [0, 1]]. On observe le motif : Aⁿ = [[1, n], [0, 1]] pour tout n ≥ 0.
