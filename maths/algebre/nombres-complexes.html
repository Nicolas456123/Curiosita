Introduction

Pourquoi inventer des nombres « imaginaires »

Les nombres complexes sont nés d'une nécessité mathématique : résoudre des équations comme x² + 1 = 0, qui n'a aucune solution réelle. En introduisant le nombre i tel que i² = -1, on ouvre un monde nouveau où toute équation polynomiale admet des solutions. Ce qui semblait un artifice de calcul au XVIe siècle s'est révélé être l'un des outils les plus puissants et les plus beaux des mathématiques.

La formule d'Euler : e^(iθ) = cos(θ) + i·sin(θ). Ce lien entre l'exponentielle, le cosinus et le sinus est considéré comme la plus belle formule des mathématiques. Pour θ = π : e^(iπ) + 1 = 0.

Les nombres complexes ne sont pas qu'une curiosité théorique. Ils sont indispensables en physique (mécanique quantique, électromagnétisme), en ingénierie (traitement du signal, circuits électriques) et en informatique (fractales, transformées de Fourier). Ce cours présente les trois formes des complexes et les opérations fondamentales.

02

Le cours

Les nombres complexes sous toutes leurs formes

Forme algébrique

Un nombre complexe s'écrit z = a + bi, où a est la partie réelle (Re(z)) et b la partie imaginaire (Im(z)). Deux complexes sont égaux si et seulement si leurs parties réelles et imaginaires sont respectivement égales.

Opérations :

  • Addition : (a + bi) + (c + di) = (a+c) + (b+d)i
  • Multiplication : (a + bi)(c + di) = (ac - bd) + (ad + bc)i
  • Conjugué : z̄ = a - bi (on change le signe de la partie imaginaire)

Module et conjugué

Le module de z = a + bi est |z| = √(a² + b²). C'est la distance du point z à l'origine dans le plan complexe.

Propriétés fondamentales : z·z̄ = |z|², |z₁·z₂| = |z₁|·|z₂|, et z⁻¹ = z̄/|z|² (pour z ≠ 0).

Exemple : Pour z = 3 + 4i : |z| = √(9 + 16) = 5, z̄ = 3 - 4i, z·z̄ = 25, z⁻¹ = (3 - 4i)/25.

Forme trigonométrique

Tout nombre complexe non nul s'écrit z = r(cos θ + i sin θ), où r = |z| est le module et θ = arg(z) est l'argument (l'angle avec l'axe réel). Cette forme est idéale pour les multiplications et les puissances :

  • Multiplication : on multiplie les modules et on additionne les arguments
  • Division : on divise les modules et on soustrait les arguments
  • Puissance (De Moivre) : zⁿ = rⁿ(cos(nθ) + i sin(nθ))

Forme exponentielle

Grâce à la formule d'Euler, on écrit z = r·e^(iθ). C'est la forme la plus compacte et la plus élégante. Les opérations deviennent naturelles :

z₁·z₂ = r₁r₂·e^(i(θ₁+θ₂)) — multiplier les modules, additionner les angles.

Racines n-ièmes de l'unité

Les solutions de zⁿ = 1 sont les n racines n-ièmes de l'unité : zₖ = e^(2ikπ/n) pour k = 0, 1, ..., n-1. Elles forment un polygone régulier à n côtés inscrit dans le cercle unité.

Exemple : Les racines cubiques de l'unité (z³ = 1) sont : z₀ = 1, z₁ = e^(2iπ/3) = -1/2 + i√3/2, z₂ = e^(4iπ/3) = -1/2 - i√3/2. Elles forment un triangle équilatéral.

Le plan complexe

Le plan complexe (ou plan d'Argand-Gauss) représente chaque nombre complexe z = a + bi comme le point de coordonnées (a, b). L'axe horizontal est l'axe réel, l'axe vertical est l'axe imaginaire. Cette représentation géométrique transforme les opérations algébriques en transformations géométriques : l'addition est une translation, la multiplication par e^(iθ) est une rotation d'angle θ.

Division de nombres complexes

Pour diviser z₁ par z₂, on multiplie numérateur et dénominateur par le conjugué de z₂ :

z₁/z₂ = (z₁ × z̄₂) / |z₂|²

Exemple : Calculer (3 + 2i) / (1 - i).

  • On multiplie par le conjugué : (3 + 2i)(1 + i) / ((1 - i)(1 + i))
  • Numérateur : 3 + 3i + 2i + 2i² = 3 + 5i - 2 = 1 + 5i
  • Dénominateur : 1 - i² = 1 + 1 = 2
  • Résultat : (1 + 5i)/2 = 1/2 + 5i/2

Équations du second degré à discriminant négatif

Quand Δ < 0, l'équation ax² + bx + c = 0 a deux solutions complexes conjuguées :

z₁ = (-b + i√|Δ|) / (2a) et z₂ = (-b - i√|Δ|) / (2a) = z̄₁

Exemple : Résoudre z² + 2z + 5 = 0.

  • Δ = 4 - 20 = -16 < 0
  • z₁ = (-2 + i√16) / 2 = (-2 + 4i) / 2 = -1 + 2i
  • z₂ = -1 - 2i (conjugué de z₁)
  • Vérification : z₁ + z₂ = -2 = -b/a et z₁ × z₂ = (-1)² + (2)² = 5 = c/a

Formule de De Moivre et linéarisation

La formule de De Moivre permet de calculer cos(nθ) et sin(nθ) en fonction de cos θ et sin θ. Par exemple, pour n = 2 :

(cos θ + i sin θ)² = cos 2θ + i sin 2θ

En développant le membre de gauche : cos²θ - sin²θ + 2i cos θ sin θ.

En identifiant parties réelle et imaginaire : cos 2θ = cos²θ - sin²θ et sin 2θ = 2 cos θ sin θ.

Pièges courants

  • Confondre i et -i : le conjugué de a + bi est a - bi, pas -a - bi. Seul le signe de la partie imaginaire change.
  • Oublier que i² = -1 : dans les calculs, chaque fois que i² apparaît, il faut le remplacer par -1. C'est la source d'erreurs la plus fréquente.
  • Module au carré vs module : |z|² = a² + b² (pas de racine carrée), tandis que |z| = √(a² + b²).
  • Argument mal défini : l'argument de z = a + bi n'est pas simplement arctan(b/a). Il faut tenir compte du quadrant : si a < 0, ajouter π (ou -π selon le signe de b).
  • Racine carrée d'un complexe : √(z) n'est pas aussi simple que pour les réels. Pour résoudre w² = z, on pose w = x + iy et on résout le système x² - y² = a et 2xy = b.

Applications

  • Électricité : l'impédance d'un circuit RLC en courant alternatif s'écrit Z = R + i(Lω - 1/(Cω)). Les complexes simplifient considérablement les calculs de circuits.
  • Traitement du signal : la transformée de Fourier décompose un signal en somme d'exponentielles complexes e^(iωt), permettant l'analyse fréquentielle.
  • Fractales : l'ensemble de Mandelbrot est défini par l'itération zₙ₊₁ = zₙ² + c dans le plan complexe. Les points c pour lesquels la suite reste bornée forment la fractale.
  • Mécanique quantique : la fonction d'onde ψ est à valeurs complexes, et le carré de son module |ψ|² donne la probabilité de présence d'une particule.

Question 1 / 6

Quel est le module de z = 3 + 4i ?

|z| = √(3² + 4²) = √(9 + 16) = √25 = 5. C'est le triplet pythagoricien (3, 4, 5).

Question 2 / 6

Que vaut i⁴ ?

i¹ = i, i² = -1, i³ = -i, i⁴ = 1. Les puissances de i sont cycliques de période 4.

Question 3 / 6

La formule d'Euler e^(iπ) est égale à...

e^(iπ) = cos(π) + i·sin(π) = -1 + 0·i = -1. D'où la célèbre identité d'Euler : e^(iπ) + 1 = 0.

Question 4 / 6

Quel est le conjugué de z = 2 - 3i ?

Le conjugué s'obtient en changeant le signe de la partie imaginaire uniquement : z̄ = 2 + 3i. La partie réelle reste inchangée.

Question 5 / 6

Que vaut (1 + i)² ?

(1 + i)² = 1 + 2i + i² = 1 + 2i - 1 = 2i. On peut aussi voir que |1+i| = √2, arg(1+i) = π/4, donc (1+i)² a pour module 2 et argument π/2, c'est-à-dire 2i.

Question 6 / 6

Combien de racines cubiques de l'unité existe-t-il ?

L'équation z³ = 1 a exactement 3 solutions : z₀ = 1, z₁ = e^(2iπ/3) = -1/2 + i√3/2, z₂ = e^(4iπ/3) = -1/2 - i√3/2. Elles forment un triangle équilatéral inscrit dans le cercle unité.

Exercices

Mettez en pratique

Exercice 1 — Opérations en forme algébrique

Calculer z₁ × z₂ et z₁/z₂ avec z₁ = 2 + 3i et z₂ = 1 - 2i.

Voir la solution

Produit : z₁z₂ = (2+3i)(1-2i) = 2 - 4i + 3i - 6i² = 2 - i + 6 = 8 - i. Division : z₁/z₂ = (2+3i)(1+2i)/((1-2i)(1+2i)) = (2+4i+3i+6i²)/(1+4) = (2+7i-6)/5 = -4/5 + 7i/5.

Exercice 2 — Module et argument

Déterminer le module et l'argument de z = -1 + i√3, puis écrire z sous forme exponentielle.

Voir la solution

|z| = √(1 + 3) = √4 = 2. cos θ = -1/2 et sin θ = √3/2, donc θ = 2π/3 (2e quadrant). Forme exponentielle : z = 2e^(i·2π/3).

Exercice 3 — Équation à discriminant négatif

Résoudre dans les complexes : z² - 4z + 13 = 0.

Voir la solution

Δ = 16 - 52 = -36 < 0. √|Δ| = 6. z₁ = (4 + 6i)/2 = 2 + 3i. z₂ = (4 - 6i)/2 = 2 - 3i. Vérification : z₁ + z₂ = 4 = -b/a et z₁z₂ = 4 + 9 = 13 = c/a.

Exercice 4 — Puissance par De Moivre

Calculer (1 + i)⁸.

Voir la solution

|1 + i| = √2, arg(1+i) = π/4. Forme expo : 1 + i = √2 · e^(iπ/4). Par De Moivre : (1+i)⁸ = (√2)⁸ · e^(i·8π/4) = 2⁴ · e^(i·2π) = 16 · 1 = 16. Le résultat est un réel positif.

Exercice 5 — Racines n-ièmes

Trouver les racines carrées de z = -4 (c'est-à-dire résoudre w² = -4).

Voir la solution

-4 = 4·e^(iπ). Les racines carrées sont √4 · e^(iπ/2) = 2i et √4 · e^(i(π/2 + π)) = 2·e^(i·3π/2) = -2i. Donc les racines carrées de -4 sont w = 2i et w = -2i. Vérification : (2i)² = 4i² = -4 et (-2i)² = 4i² = -4.
